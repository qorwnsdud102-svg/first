# 구글 애즈 디스커버리 기능 한번에 이해하기 : 2년간 06억써본 인사이트 5탄

안녕하세요 구글 애즈

관련 벌써 5탄입니다!!

구글애즈 에서 가장 중요한

운영 캠페인 목표라고 하면

디스커버리라고 할 수 있는데요.

디스커버리 캠페인이 중요한 이유는

아무래도 유투브 라는 지면에서 가장

잘 보이는 지면이기도 합니다.

디스커버리지면의 기능을 하나하나

살펴보고 어떻게 운영하면 좋은지

간략하게 알아보는 시간을 가져보겠습니다.

우선 디스커버리 캠페인을 만들어봅니다.

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727183597278.png)

캠페인 목표는 판매로

그리고 전환 목표 는

저는 처음에 이전에 말했듯이

구매, 휘원가입, 결제시작, 장바구니

구매 퍼널 모두를 넣어둡니다.

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727184402153.png)

디스커버리를 선택하면 이렇게

메뉴가 활성화 됩니다

최근에 또 제품 피드가 있는 디스커버리 캠페인이

생성이 되어서 활용 할 수 있게 되었습니다.

 이전에는 표준 디스커버리

캠페인만 운영이 가능했습니다.

위 보이시는 웹사이트 링크는

내가 홍보할 사이트 링크 넣어주시면 됩니다

어차피 광고 세팅시 마지막에

다시 도착 URL 설정을 할거라서

메인 URL 넣어주시면 됩니다.

그리고 캠페인 이름은

구분하기 편하게 구성을 해주시면 됩니다.

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727191320958.png)

이렇게 넣어주시면 됩니다.

이름 만드는건 이전 시간에도

설명드렸고 구분하기 편하게

설정하시면 됩니다!

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727192665792.png)

그렇게 설정하면

4 개 전환 목표 값을 같이 못쓰고

1개의 목표를 설정하라고 뜹니다.

당연히 저희느 매출이 목표이니

구매 목표로 많이 잡습니다.

혹여나 사이트 특성상

다른 목표가 매출에 더 견인역활을 한다면

예를들어서

회원가입이 많이 되면

좋은 사이트라면

회원가입을 잡아도 됩니다

매출이 목표로 잡는 사이트는

대부분이 온라인 쇼핑몰 특성일 것 입니다.

일단 저는 "구매"를 선택합니다.

아참 그리고

디스커버리 광고의 특성이 있습니다.

디스커버리 광고는

최적화 광고 라고 생각 하시면 됩니다

무슨말이냐.

완전 학습 된 픽셀

여러가지 광고가 돌아갔던

계정에는 방문 이나 장바구니 결제시작

같은 모수들이 스크립트에 쌓여 있겟죠?

픽셀 모수에 말이죠

그런 모수들이 없는

깨끗한 계정에서는 머신러닝이

좀 더 오래 걸리는 경향이 있습니다.

즉 첫 광고 모수를 잘 쌓고

그 모수를 활용해서 디스커버리를

활용하는게 좀 더 좋다 라고 생각하시면

좋을듯 합니다.

다만 절대적으로 첫모수를 안쌓으면

디스커버리 광고 못하나요?

라고 하시면 당연히 할 수 있습니다.

첫 모수 없이 디스커버리 광고를

진행해도 상관은 없습니다.

저는 경험상 디스커버리 광고를

어느정도 모수가 쌓이면 하면 좋았고

초반에 운영시에도 타겟이나

컨텐츠를 잘잡아서 학습이 잘되면

점차 개선되면서 성과가 잘 따라오는

경향이 있었습니다

최고 장점은

디스커버리 광고는

CPC 가 매우 저렴합니다.

이미지형이고 노출되는

구좌들이 많아서 그런지

CTR 형성도 좋은 편이고

평균적으로 안착하고

성공한 캠페인의 경우

CPC 50~100원대를 보입니다.

아래 캡쳐 이미지 처럼 말이죠

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727204043425.png)

CPC 가 저렴해서 좋다 라기 보다는

전환 성과도 우수하면서

유입 비용도 좋다 라고 판단해야 할 것 같습니다,

이 유입 전환 성과가 좋게 하려면

간단하게 아까 말했던

기존의 모수들이 잘 쌓여있으면

충분히 더 좋은 데이터 결과를

만들 수 있을 것으로 보입니다.

그래서~ 이어서 말씀드리면

이제 전환 목표를 설정하는데

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727204666399.png)

저는 입찰 할 때

보통 그냥 전환으로 많이 설정합니다.

초반에는 말이죠

이제 광고를 1달 2달 정도하면

그 캠페인에 데이터가 어느정도

쌓이고 CPA나 로아스 기준이

잡히기 시작합니다.

그때 입찰 목표를 바꿔줍니다.

전환가치를 할수 있는데

전환가치는 ROAS 나 CPA 형태로

설정이 가능합니다.

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727205288482.png)

CPA의 경우

구매 전환 액션당 얼마 비용을

지불을 할 것인가 설정하는 기능인데

지금 예를들어서

CPA 비용이 1만원 정도 평균으로

나오는데 내가 원하는 CPA 목표는 6000원 이라고

가정을하면 CPA를 바로 6000원으로

설정을 하면 절대 안됩니다.

이렇게 6000원 목표라서 낮춰버리면

바로 광고가 멈춰버리는 현상이 생깁니다

구글은 머신러닝 기반이다 보니

어떤 특정값을 계속 학습을 시키면서

광고가 점차 개선되는 형태를 보입니다

1만원 나오던 결과값을 6000원으로

변경하게되면 그만큼 데이터 차이가

커서 광고 효율을 낼수 가 없기 때문에

라고 판단하여 머신러닝이 광고비

소진을 멈춰버리는 현상이 생기고

광고비가 하루 30만원 설정인데

하루에 1만원도 소진이 안되느 그런

상황에 봉착 할 것 입니다.

그래서 보통은 CPA 설정할때도

1만원이라면 9000원으로 줄여서

일주일 학습 ~ 또는 원하는 결과만큼

떨어지는지 데이터 확인하고

평균 데이터 값을 확인하면서 조절하면

가장좋습니다. 광고비 증액처럼

10~% 대 정도로 CPA값을 내려주면서

실제 소진이 잘되는지 그리고

우리가 원하는 설정한 CPA 값에

근접하게 성과가 나오는지 봐야합니다.

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727208612432.png)

전환 가치도 마찬가지입니다.

전환 가치도 ROAS 기준인데

이것도 지금 현재 7일기준

평균로아스가 얼마인지 보고

거기에 맞춰서 점진적으로

올려나가고 소진이 잘되는지

맞춰봐야합니다.

그래서 광고 라이브 부터

이 타겟 CPA 나 ROAS 기능을

쓰면 아에 소진을 못하거나

캠페인이 바보가 되어서

아에 먹통이 되기도 하니

라이브하면서 학습이 되고

데이터가 쌓인 캠페인에서

사용해보시길 권장드리고

이 해당 기능을 잘쓰면

 굉장히 성과가 잘나오기도 합니다

특히 모수가 많이 쌓인 계정은

진짜 구매를 잘할 거 같은 사람을

쏙쏙 뽑아서 대려와서

CPC 도 싸게 전환 비용도 저렴하게

만들어주기도합니다 신기할 정도로

구글의 머신러닝도 똑똑합니다.

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_1672721028103.png)

설정 더 보기 누르면

이렇게 나오는 형태를 볼 수 있습니다.

이전에 보셨던 기능들이죠?

패스하겟습니다 설명은 이하 생략

다른 기능편 보시면 다 나와있으니

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202301/67ed29aec76b20bbe86e68722dc91943_16727214847327.png)

많습니다. 다만 세팅이되고 광고가
