# AI 활용 성숙도 자가진단 체크리스트 자가진단 해보세요!

**채점법:** (a)=0점, (b)=1점, (c)=2점,
(d)=3점 → 합산 후 마지막 표로 레벨 판정

**A.
환경 / 셋업 (기초)**

**Q1. 평소 AI를 어떤 인터페이스로 쓰나요?**

**☐  (a)** 카톡 ChatGPT 봇이나

**☐  (c)** 데스크탑 앱 + 웹 + 가끔
IDE 익스텐션   [2점]

**☐  (d)** CLI 창에서
Claude Code / Codex / Gemini를 직접 호출해서 쓴다   [3점]

**Q2. 새 컴퓨터를 받으면 AI 환경 세팅에 얼마나 걸리나요?**

**☐  (b)** 반나절~하루 걸려서 겨우
세팅   [1점]

**☐  (c)** 1~2시간 안에
chatgpt/claude 로그인까지 가능   [2점]

**☐  (d)** 30분 안에 npm i
-g → 3개 CLI 로그인 + 인증/스킬 동기화 완료   [3점]

둠   [2점]

**☐  (d)** OAuth/구독 인증 우선
(codex login / claude login / gemini OAuth) — API 키는 fallback   [3점]

**Q4. AI 도구 비용을 어떻게 통제하나요?**

**☐  (a)** 모름 / 회사 카드로 불명확하게

**Q5. 한글 인코딩 / 콘솔 깨짐 문제를 만나면?**
