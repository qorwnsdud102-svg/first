# [월10억매출 만든 컨텐츠 흐름] SNS 컨텐츠로 성공하는 업체들 공통흐름 4가지

![](https://cdn.ibos.kr/design/upload_file/__HTMLEDITOR__/202003/1130470c7b365f858990f7779e099747_15840838017255.jpg)

저희 회사 SNS 대행 운영 흐름입니다.

사람들은 그럼 어떻게 보통 운영이 되냐 라고 물어보십니다.

A 단계 : 보시는 것처럼 녹색그래프가 엄청 높게 올라갑니다.

컨텐츠 실험단계라 광고비 그래프 분홍색 도 매우 많이 올라갑니다.

보통 1단계 실험 첫 테스트 컨텐츠 100개정도 실험합니다.

1개당 2만원 x 100개 = 200만원 이렇게 3~4회 반복해서

2~5% 정도 살아남는 베스트 컨텐츠에 예산을 분배 증액 합니다.

B단계 : 베스트컨텐츠가 나오면서 광고비대비 수익율이 조금씩좋아집니다.

이단계에서 중요한건 제대로된 디벨롭 혹은 새로운 소구 실험입니다.

강화를 하면서 내부 컨텐츠도 톤을 비슷하게 수정해주는 작업을합니다.

C단계 :  폭팔적 구간전 소구정리 및 대량 양산 그리고 증액

컨텐츠 단계에서 먹히는 소구점 발견된 컨텐츠를 더욱 SNS 채널에 맞게 강화합니다.

여기서 증액을 엄청나게 공격적으로 진행합니다.

Z단계 : 이구간에서 들어서면 굉장히 수익율이 좋아집니다.

컨텐츠 양산도 많이 할필요없이 포인트잡아서 주마다 50개씩

보충하면서 성과가 무너지지않도록 잡아줍니다.
