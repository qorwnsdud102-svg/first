# 상징재

## 한 줄 정의

제품의 [[상징성과 당위성]]을 외부에서 빌려오는 도구. [[엠타트업]]의 고유 용어이자 **[[재 분류]] 4가지 중 하나**.

## [[재 분류]] 안에서의 위치

상징재는 [[재 분류]]의 4가지 중 하나 (기능재 · 경험재 · **상징재** · 경제재).
**진입 순서로는 기능재 → 경험재 → 상징재**. 처음부터 상징재로 가지 않는다.

## 온라인의 상징재 — 상위노출 = 상징

오프라인의 명품·연예인 같은 상징재가 온라인에서는 다음으로 변환된다:

- 상위노출 / 자주 보이는 것 / 광고 많이 하는 것
- 키워드 1위·대상 받음
- 많이 팔린 것 ("유명하고 많이 팔린 거 사면 손해는 안 보겠지" 심리)
- 구매로 사회적 인정·과시 가능한 것

→ DA 광고의 7도구가 "상징성을 입히는 방법"이라면, **검색·노출 자체도 또 하나의 상징재 입히기**.

## 7가지 도구

### 1. 국가 사대주의 (신뢰성 ↑)

선진국 이름을 제품에 붙이면 신뢰성 자동 상승. 개발도상국은 X.

- 햄버거 → 유러피안 햄버거
- 베이글 → 런던 베이글

### 2. 국가적 특장점 (당위성 ↑)

각 나라의 클리셰를 제품 효능과 연결.

| 국가 | 클리셰 |
|---|---|
| 독일 | 기술력 |
| 핀란드 | 치아 (자일리톨) |
| 네덜란드 | 키 |
| 일본 | 고령화 → 역노화 |
| 북유럽 | 대자연·청정 |

> 예: "전세계 고령화율 1위 일본 도쿄연구소에서 개발한 역노화 주름앰플"
> "북유럽 대자연의 침엽수로 만든 피톤치드 스프레이"
> "독일 맥주공장 노동자들이 탈모에 안 걸리는 이유는?"

### 3. 외형 따라하기

프리미엄 제품의 외형만 닮아도 그 상징성이 일부 묻어옴.

- 메디큐브 PDRN ↔ 메라블 PDRN
- 쏜리서치 ↔ 뉴벨라
- 다이슨 ↔ 차이슨
- 오쏘몰 ↔ 오쏘몰 짭

**관건**: **프리미엄 + 특징이 확실한** 외형의 제품을 따라할 것. 아이허브 같은 저렴이는 신뢰는 있어도 가져올 상징성이 약함. "다이슨 짭" 같은 프레임 씌워지지 않게. 디자인·상표권 주의.

### 4. UI/UX 합치기

뻔한 두 제품을 합치면 새로워 보임.

> "새로운 건 없다, 새로워 보이는 것만 있다."

- 네리티아 **세럼파데** — 세럼 + 파운데이션. "생얼 같지만 발랐다"는 욕구.
- 그라인더 + 레이저 → 무좀 시장에서 차별화.

### 5. 인물/전문가 상징재

연예인은 기본적으로 신뢰도 ↑. 제품 이미지에 맞는 연예인을 쓰면 효과 배가.

| 연예인 | 매칭 제품 | 매칭 이유 |
|---|---|---|
| 쯔양 | 다이어트·소화제 | 잘 먹고 소화 잘 시키는 이미지 |
| 이덕화 | 가발 | 본인 이슈 |
| 남보라 (5남매) | 생활용품 | 대가족 = 알뜰·다용도 |
| 성시경 | 주방 | 요리·깐깐 이미지 |
| 김종국 | 헬스 | 운동 이미지 |
| 노라조 | 파티 플랫폼 | 흥·분위기 |
| 하하 와이프 | 밀키즈 (별의 엄마) | 양육 |
| 음향감독 | 이명 | 직업 매칭 |
| 스튜어디스 | 다리 붓기·편한 구두 | 직업 매칭 |

### 6. 트렌드 차용

이미 떠오르는 다른 트렌드의 상징성을 내 제품에 입히기.

- NMN(노화 영양제 트렌드) → 'NMN 주름 앰플'
- 올레샷(올리브+레몬, 다이어트 식단) → '바르는 올레놀샷' 주름앰플
- 독도토너, 누룩밤 — 같은 패턴
- 오쏘몰의 상징성을 아이크림에 차용

### 7. 의학 상징성

피부과·병원 시술의 인식을 제품에 입힘.

- 메라블 **유쎄라블 리프팅샷 900 아이크림** = "바르는 울쎄라"
- 약국·병원 입점 표기 자체가 상징재. (실제 신뢰성보다 '있어 보임' 효과가 큼)

## 적용 체크리스트 — 좋은 상징재의 조건

자료의 '바르는 울쎄라' 분석에서 추출:

- [ ] **결핍이 명확한가** (스킨부스터처럼 효능이 분산되면 약함, 울쎄라처럼 "리프팅"으로 명확하면 강함)
- [ ] **제품과의 결 일치도** (60% 이상 권장. 100% 못 맞춰도 됨)
- [ ] **직관적으로 무슨 효과인지 1초에 인지 가능**
- [ ] **제품명에 상징재가 연상되게 박혀 있는가** ('유쎄라블', '900샷')
- [ ] **창의적 수습** (미세열감처럼 화장품에서 본 적 없는 디테일로 시술 느낌 부여)

## 여행지 맛집 패턴

> 중요도 낮고 고민도 좀 높은 곳에서 상징재가 결정적 — 후기 수·1위·인기 표시가 강하게 작동.

## 주의

너무 일찍 "법적으로 안 됨", "소송 들어올 수 있음" 생각하면 후킹이 안 나온다.
**먼저 창의로 최대치 후킹을 짜고, 현실 반영은 그 다음.**
