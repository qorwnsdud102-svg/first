# 상징성과 당위성

## 한 줄 정의

1초 광고에서 "이 제품은 특별하다(**상징성**)"와 "그래서 효과 있다(**당위성**)" 두 인식을 동시에 심어야 한다는 프레임.

## 핵심 속성

- **상징성**: 제품이 특별해 보이게 만드는 외부 권위·이미지.
- **당위성**: 설명 없이도 논리적으로 그럴듯해 보이게 만드는 인과 고리.
- 둘 다 **외부에서 빌려온다** — 자체 생성이 아니라 이미 사람들 머릿속에 있는 인식을 끌어다 쓴다.
- 둘은 한 광고 안에서 **함께** 발사돼야 한다. 상징성만 있으면 "있어 보이는데 왜 사야 하지?", 당위성만 있으면 "맞는 말인데 굳이 이 브랜드?"가 된다.

## 사례 — 밀키즈 어린이 키성장 영양제

> "네덜란드는 전세계 키 1위, 비결은 네덜란드 유제품. 우리는 네덜란드산 유제품을 넣어 만들었다."

- 상징성: 네덜란드(선진국 사대주의) + 전세계 키 1위
- 당위성: 유제품 → 키 성장이라는 직관적 인과
- 결과: 1초 안에 "이 제품은 특별하고 실제 효과 있을 것" 인식 형성.

## [[그로스짐]]의 4분할 — 4가지 사회적 증거

[[그로스짐]] 강의가 사용하는 더 추상화된 분류. **상징성·당위성과 결 일치**:

| 그로스짐 분할 | 정의 | 본 페이지 매핑 |
|---|---|---|
| **사회적 통념** | 보편 상식·규범 ("초등학생도 이해하는 키워드") | **당위성** (논리 인과 X 직관 인과) |
| **사회적 근거** | 많은 사람이 쓴다는 증거 (몇 명 샀다·줄 서 있다·리뷰 개수) | 상징성 — 군중 권위 |
| **사회적 권위** | 명사·전문가·연예인 (강남 피부과·인플루언서·박사) | 상징성 — 명사 권위 ([[상징재]] 6번 인물·전문가) |
| **숫자적 증거** | 객관 (00% 임상·N위·만 개 판매) | 상징성 — 데이터 권위 |

→ 7도구 ([[상징재]])는 **상징성을 가져오는 운반 수단**, 그로스짐 4분할은 **상징성·당위성을 객관화하는 분류 축**.

> "결국에 모든 e-commerce 콘텐츠는 다 동일하다. 이거의 반복."

### 통념 디테일 (1주차 다시보기 3강)

- **미국 경찰 사례**: 90% 통제 따름 = 권위 통념의 극단.
- **미역국·도화살** = 한국 학습 통념.
- **피부과 원장실 우회 키워드** = 직접 표기 X, 연상 가능한 배경·이미지로 통념 환기.
- 출처 책 2권: **로버트 치알디니 《설득의 심리학》** · **조지라드** (자동차 판매왕 1만3100대).

### 회색지대 채우기 — 합법 숫자 만들기 (1주차)

규제·임상 자료 부족 시 패턴:

- **서베이 20명 만족도 95%**
- **4년 = 1400일**
- **3개 런칭 = 3개 완판**
- 어머니 200ml 주름크림 (런칭 직후 월 3억 — 강사 본인 사례)

→ 숫자는 만들기 나름. 단, 거짓 X — **사실 데이터를 임팩트 있게 가공**하는 기술.

### 정력 인식 키워드 — 통념의 실증 (2주차 우선피드백 2)

> "**바이킹 X / 장어 O.**"

같은 정력 카테고리에도 통념의 무게가 다름. **직관 정력 = 장어**, 바이킹은 통념 동력 약함. 강사 데이터 기반.
