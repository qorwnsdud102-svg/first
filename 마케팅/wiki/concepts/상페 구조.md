# 상페 구조 (오메클 + 자사몰 5단계 + 그로스짐 3축)

## 한 줄 정의

상세페이지·자사몰의 판매 플로우. **세 가지 모델 공존**:

- **[[엠타트업]] 모델** — 오프닝-메리트-클로징 (오메클). "오프닝이 전부다 90%."
- **[[환사장]] 모델** — 자사몰 5단계 플로우 (인지·고려·탐색·평가·재구매). 자사몰 + 메타광고 단일 라인.
- **[[그로스짐]] 모델** — USP + Pain Point + 손실·손해 3축 + **콘텐츠 3종 분리** (광고·랜딩·바이럴).

세 모델은 상충하지 않고 **다른 채널·관여도·시점에 적용**.

## [[환사장]] 자사몰 5단계 플로우

> 우리가 광고를 보고 제품을 사는 과정 = 자사몰 판매 플로우

| 단계 | 행동 | 핵심 도구 |
|---|---|---|
| **1. 인지** | 광고로 사이트 OR 검색 유입 | 메타 광고 ([[환사장]] 4 캠페인) |
| **2. 고려** | 상페·리뷰 본다 | 상위 5개 리뷰 + 비포·에프터 / 저관여(5만원↓) 충동구매 vs 고관여(추가 검증) |
| **3. 탐색** | 타사 비교 | 상페 차별점 + 권위 |
| **4. 평가/확인** | **블로그·카페·유튜브 내돈내산** 검색 | 물 아래 작업 (블로그·기사·지식인 깔기) |
| **5. 구매·재구매** | CRM (카카오톡 채널) | 신상·기획전 안내 |

### 물 아래 vs 물 위

| 구분 | 작업 | 초반 중요도 |
|---|---|---|
| **물 아래** (깔기) | 리뷰 500-1000개 (상위 5개+포토·영상이 핵심) | 상 |
| 물 아래 | 블로그·카페 체험단·기사·지식인 | 하 |
| **물 위** (광고) | 메타 광고 (초반 집중) | 상 |
| 물 위 | 협력광고 (인플루언서) | 중 |

→ **물 아래 먼저 깔린 후 광고 시작**. 광고로 유입돼도 상페·리뷰 약하면 전환 X.

## [[엠타트업]] 오메클 (기존)

## 3단 구조

| 단계 | 목적 | 도구 |
|---|---|---|
| **오프닝** | 경계심 해소 + 신뢰도 = 공감 + 집중 | 첫 5초 (첫인사·분위기·농담·말투·표정·색·폰트·이미지) |
| **메리트** | 오프닝에서 받은 인상 강화 | [[가치더하기]] · [[상징재]] |
| **클로징** | 결정 강요 | [[선택제어]] (소거법·더블바인드) · [[매몰비용]] |

## 핵심 명제 — 오프닝이 전부

> "**오프닝이 전부다.** 첫 5초를 잡아라. 1초면 결정된다.
> **이게 내 상페의 전부다. 90% 이상이다.**
> 메리트·클로징은 오프닝에서 느낀 감정을 강화할 뿐이다."

> "**제품보다 중요한 건 분위기.**"

## 오프닝의 골든타임

- **첫 1-2분의 긍정 경험이 쌓이면 이후 정보는 긍정 필터로 알아서 걸러진다.**
- 분량: 첫 스크롤 한 번.
- 후기 수가 매우 중요 (개수). 쿠팡에서 못 채울 바엔 자사몰로.

## 오프닝 카피 — 사례

- 재떨이를 비흡연자에게 팔 때 → "**당신은 매너 있는 사람입니다.**"
- 마인드맵의 [[매몰비용]] 카피 패턴:
  - "기존의 전기방석들 ***인 거 알고 계셨나요?" (제로 — 결핍 낮추기)
  - "안전하지 않은 건 ***가 아닙니다" (제로 — 결핍 채우기)
  - "그럼 안전하기만 하면 다 좋은 걸까?" (가치더하기 진입)

## 마인드맵의 3단계와의 매핑

마인드맵에서는 같은 구조를 다른 명명:

| 본강의 (오메클) | 마인드맵 |
|---|---|
| 오프닝 | 1단계 — 이벤트·신뢰 (프로모션·100% 환불·키워드 1위·사은품) |
| 오프닝 끝 ~ 메리트 시작 | 2단계 — 제로마케팅 (첫 카피의 중요성) |
| 메리트 | (가치더하기 단계 통합) |
| 클로징 | 3단계 — 배제설득 (소거법) |

→ 같은 구조를 본강의에서는 "오프닝 90%"로 강조점을 옮긴 표현.

## 오프닝 후 흐름

> "**오프닝에서 받은 인상 + 그 후로는 그걸 증명하기 위한 근거들.**"

오프닝이 약하면 메리트·클로징 아무리 잘 만들어도 무용. 메리트·클로징은 오프닝에서 형성된 감정을 강화하는 보조.

## [[그로스짐]] 모델 — USP + Pain + 손실 3축

> "광고 콘텐츠와 상페·랜딩의 본질은 같다. 화자만 바꿔서 무한 변주."

### 요약 포인트 3축 (가장 위)

1. **USP** — 우리만의 강점
2. **Pain Point** — 고객의 고통
3. **고객의 손실·손해** — 안 사면 잃는 것

### 콘텐츠 5요소 흐름 (광고 콘텐츠)

| 요소 | 정의 |
|---|---|
| 후킹 | 콘텐츠를 볼 이유 (썸네일·첫 1초) |
| 스토리텔링 | 고객 공감 |
| 설득 | 4가지 사회적 증거 ([[상징성과 당위성]]) |
| 프로모션 / 포모 | 랜딩에서만 (네이티브 광고 보호) |
| 행동요구 (CTA) | 셀링 광고 강하게, 네이티브 약하게 |

### 화자 4종 (상페·광고 공통)

- **고객** (리뷰)
- **전문가** (임상·근거)
- **판매자** (셀링 영업톤)
- **제3자** (연예인·인플루언서·외부 전문가)

→ 같은 메시지를 화자만 바꿔 다양한 섹션에 변주. 연예인 위 vs 리뷰 위 = 광고 데이터로 자주 갱신.

### 3 flow (구성 순서)

> 문제 제기 → 문제 해결 → 근거 제시.

순서 가변 — 근거 제시 → 문제 해결 → 문제 도출도 가능. 3 flow가 본질, 순서는 데이터로.

### 콘텐츠 3종 분리

| 종류 | 프로모션·포모·CTA |
|---|---|
| **광고 콘텐츠** (SNS·DA) | 약하게 — 후킹·스토리 우선 |
| **랜딩 페이지** | 강하게 OK — 한정 수량·임박감·CTA |
| **바이럴 콘텐츠** (블로그·카페·지식인·유튜브·인스타) | 자연스러운 톤 — 환사장의 "물 아래"와 같은 결 |

### 디테일

- **GIF는 USP 핵심 1-2곳에만**. 남발 시 사이트 느려짐 → 3초 이탈.
- 문장 짧고 간결, 쉬운 단어. 컬러·톤은 2~3개로 제한.
