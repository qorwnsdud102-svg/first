# (부록 2) 퍼포캔 기반 광고 콘텐츠 실험방법

## 한 줄 요지

퍼포먼스 캔버스 작성 후 **광고 실험 운영 프레임**. 5대 지표(한계비용·목표비용·CPA/CPC·BEP·직접 전환) + USP 방향성 전략 + CBO/ABO + 매체별 특성 + 메시지 삼위일체.

## 핵심 1 — 실험 설계 5대 지표

| 지표 | 정의 | 활용 |
|---|---|---|
| **1. 한계비용** | 콘텐츠당 얼마까지 써보고 포기할지 | CPC·CPM 평균 + 객단가로 계산 (감 X, 과학 O) |
| **2. 목표비용** | 도달하면 과감히 끄거나 증액할 기준선 | 로하스 200%면 CPA = 객단가÷2 |
| **3. 목표 CPA/CPC** | 수학식 역산 | 100명 중 2% 전환 = CPA 4만원 → CPC 800원 |
| **4. BEP 로하스** | 손익분기 로하스 | 맞추면 유통(쿠팡·스마트스토어) 낙수 효과로 자사몰 이익 보장 |
| **5. 직접 전환 vs 간접 전환** | 의사결정 기준 분리 | ROAS는 간접(1/7일) 포함, 증감액 의사결정은 **직접** 기준 |

> 직접 전환만 보는 광고주 vs 간접까지 보는 광고주의 성과 격차 > 90% 상위권.

## 핵심 2 — USP 방향성 전략 (3개 룰)

> "방향성은 최대 3개만. 본질 동일·표현만 변화."

**올바른 USP 조합** (예: 청소기 흡입력):
- 1번·2번·3번이 **다 같은 본질** (예: 흡입력 우수)
- 화자·카피 표현만 다름 (고객 리뷰 vs 임상 자료 vs 판매자 설명)
- = 무한 변주 가능, 비용 낮음

**틀린 USP 조합**:
- 1번 흡입력 / 2번 수명 / 3번 소음 = 방향성 3개 (본질 다름)
- 테스트 비용 3배 증가, 대부분 실패. 2·3번은 부수기능 = 마진만 깎음.

**데이터 기반 재정렬 사례 — 바디워시**:
- 원래 "보습 USP" → 고객 피드백에서 "향기" 반응 우수
- 랜딩 A(보습) vs B(향기) 히든 AB 테스트
- B 전환율 우수 → "향기 중심" USP로 모든 매체 집중 → **월 매출 10억 달성**
- 본질 우선순위: 객관적 근거 자료 가까운 내용 → 먼 내용 순.

## 핵심 3 — CBO vs ABO (캠페인 예산 최적화)

| 모드 | 정의 | 사용 시점 |
|---|---|---|
| **CBO** (Campaign Budget Optimization) | 캠페인 1예산 → 그룹 자동 분배. 머신러닝 자동. | 방향성 확정 + 확장(Scale) 단계 |
| **ABO** (Adset Budget Optimization) | 그룹별 수동 동등 배분. AB 가설 동일 조건 검증. | 초기 실험·새 방향성·타겟팅 가설 |

**예산별 운용**:
- 소예산(10-30만원): 그룹 1개, 소재 2-3개 (CBO 또는 ABO 단순)
- 중예산(100만원): ABO로 먼저 테스트, 승자 확정 후 CBO 스케일
- 대예산(500만원+): ABO + CBO 병행 (고성과 → CBO 몰음, 신소재 → ABO 실험)

**원칙**: 의도 없는 설정 = 낭비. 각 그룹·소재마다 "왜 이 타겟팅인가" 가설 명시 필수.

## 핵심 4 — 메타·구글·GFA·DCO 매체별 특성

| 매체 | 특징 | 실험 전략 |
|---|---|---|
| **메타** (페북·인스타) | 픽셀 머신러닝 우수, 모수 관리형 | 세팅 후 자동화 맡김. 빈도 평균 2-2.5회. **리타겟팅 180일 장기 X** (구매 안 할 고객 CPM 낭비) |
| **구글** (검색·GDN·유튜브·디스커버리) | 모수 매우 넓음(특히 유튜브), 머신러닝 천천움 | 타겟팅 세밀화 + **리타겟팅 우선**. 픽셀/전환 데이터 쌓아야 머신러닝 가속. 검색+GDN+유튜브+디스커버리 모두 운영 (누적 노출 = 빈도수 = 구매결정↑) |
| **GFA** (네이버) | 모수 넓음(30대+ 강세), 머신러닝 반쪽짜리 | 타겟팅 뾰족하게 (연령·관심사 정교화). CPC 기준 낮춰 직접 전환 높은 콘텐츠 증액. **40-60대가 가장 우수** |
| **DCO** (Dynamic Creative Optimization) | 동적 조합 자동 | 방향성 미결정·이미지 100장 초기 테스트. 10장씩 묶어 조합. 메타에선 너무 많은 조합은 재활용 어려움 주의 |

## 핵심 5 — 메시지 삼위일체 (광고·상페·외부)

> 광고·상세페이지·외부(블로그·유튜브·리뷰)의 메시지가 **동일 본질** + **화자만 다름**.

**올바른 예** (향기 바디워시):
- 광고: "향기 좋음" (공격적)
- 상페: "향기 좋음 + 무자극 이유" (상세)
- 고객 리뷰/블로그: "향기도 좋고 피부 자극 없네요" (경험 기반)
- → 본질 동일, 화자만 다름 = 설득력↑ + 자연스러움

**잘못된 예**: 광고 "향기" / 상페 "무자극" / 블로그 "둘 다" → 메시지 산재 = 확률 낮음.

운영: 외부 리뷰(쿠팡·스마트스토어) 자주 모니터링. 별점 급락 시 즉시 분석 + 상페 수정.
