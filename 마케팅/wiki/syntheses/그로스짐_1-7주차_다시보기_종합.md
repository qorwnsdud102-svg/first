## 핵심 1 — 리뷰 작업 신규 운영 디테일

### 리뷰 전환율 격차

- **좋은 리뷰 상단 배치 = 10% 전환율** vs 부정 리뷰 상단 = 2.5% — **4배 격차**
- 리뷰 최신성 = 밴드웨건 효과 (줄 서 있는 맛집)
- 1-2점 리뷰도 작업 (제품 만족 + 배송 불만 같은 혼합형 — 순수해 보이며 신뢰도↑)

### 플랫폼별 운영

- **스마트스토어**: 매달 2-3개씩 주간 단위. 한꺼번에 수백 개 X (스토어 정지). **디지터/솔리샵** (에이베러 7년 운영) 같은 검증된 체험단 추천
- **쿠팡**: 로켓 배송 = 전환율 3배 (윙 3% → 판매자 로켓 9-10%). 3일~1주일 주기로 최신순/추천순 밀어내기
- **자사몰**: DB 주체 = 판매자 본인. 가격은 최저가 (네이버페이 클릭 바로 결제). **회원가입 강제 X** (전환율 급락)

### 자사몰 vs 쿠팡 ROAS

> 쿠팡 광고 100만 → 자사몰 150% ROAS. **+ 브랜드 키워드 검색 → 쿠팡 로켓 70% 전환율 추가** → 전체 200% 달성. 쿠팡과 자사몰을 적대 X, 시너지 O.

## 핵심 2 — 네이버 쇼핑 SEO 디테일

### 키워드별 섹션 순서 확인

> 네이버 쇼핑 섹션(이미지·블로그·카페·지식·쇼핑) 순서를 키워드별로 먼저 확인 후 역배치.

- **"기미"** = 의료 정보 위주 → 자연 노출 기대 X (블로그 작업 무의미)
- **"기미 크림"** = 쇼핑 섹션 상단 → 수익성 높음 (쇼핑 광고 + 자연 노출 병행)

### 상품명 최적화

- "필터링된 키워드"만 — 중복 성분(에센스 범주에 이미 포함된 '병풀 추출물')은 상품명 낭비
- AI(Claude/Gemini)에 네이버 가이드 PDF + 자사 상품명 입력 → 자동 정제
- "약알칼리성" 같은 중간 허브 키워드 함정 — 네이티브해 보이나 경쟁사 침입 가능 (상표권 X)

### 블로그 랜딩 추적 한계

- 블로그 랜딩 → 전환율↑ But **광고 추적 불가** (메타·구글 광고)
- 네이버 검색광고·GFA는 네이버페이 클릭 시 추적되기도 함
- 네이버 블로그 > 티스토리·워드프레스 (중년층 신뢰도)
- 초반: 자사몰 랜딩으로 성과 확인 → 필요시 블로그 전환

## 핵심 3 — 상세페이지 미세 튜닝

### 모바일 최적화 폰트·색상

- 폰트 **2종**, 색상 **2-3종**만 (과다 시 가독성↓)
- 메인 카피 **8-15자** / 설명 **15-20자** (모바일 14픽셀 이상)
- 컨셉 컬러 반복 노출 → 무의식 인식 (광고비 10억+ 쓰면 더 효과)

### 인트로 테스트 유연성 (고정 X)

- 정형: Pain Point → 공감 → 기대감 설정
- **But** 경쟁사 1등은 연예인 협찬 / 제품 리뉴얼 안내를 먼저 배치
- 최상단 요소 일주일 단위 A/B 테스트 → 2개월이면 최적값
- 도입부 에너지 집중: Pain Point + 숫자 + 호기심 필수

### 손실 심리(FOMO) 극대화

- "지금 놓치면 못 산다" 메시지 = 전환율 **0.23%p 상승** (e-commerce 평균 2% 대비 **10% 증가** 효과)
- 한정 수량 = 실제 한정 여부와 무관 (누룩밤도 "몇 명 함께 보는 중" 표시)
- 당장 구매 버튼 위에 한정 수량·한정 세트 + 환불 보장
- "무조건 환불" X → **"제품 효과에 자신 → 환불 보장"** 근거 추가

### 이미지·영상 역할 분담

- **GIF 남발 X** (사이트 속도↓ → 3초 이탈) — USP 핵심 1-2곳만
- **영상 = 텍스트로 설명 불가능한 부분만** (조작 과정·성분 작용)
- **Before/After = AI 생성(제미나이) 허용** — 실제 촬영 불필요. 다양한 모델 무제한 제작

## 핵심 4 — AI 자동화 신규 워크플로우

### 리뷰 생성 (50개 → 20개)

- 경쟁사 리뷰 50개 다운로드
- 예시 2개 + 퍼포먼스 캔버스 정보 → AI에 입력
- 각색 리뷰 20개 추출
- BNA 이미지 50장 생성 분포: **20% DNA · 20% 바르는 장면 · 10% 애프터 예쁜 여성 · 50% 케이스/체험단**

### 경쟁사 분석 (자동)

- 경쟁사 상세페이지 크롤링 + Claude/Gemini 패턴 분석 (폰트·자간·글자 수)
- 스마트스토어·쿠팡 리뷰 다운 → 패턴화 (반복되는 불만) → 상세페이지 선제 해결 메시지 삽입

### Claude MCP > ChatGPT

- Claude MCP: 폴더 자동 정리 가능
- ChatGPT: 폴더 자동 정리 불가
- 제미나이: 이미지 생성 + 힉스필드 카메라 스펙·조명 미세조정

## 핵심 5 — 우회적 욕구 (욕구 3분할)

> 기존: 표면적 vs 원초적. **추가: 우회적 욕구.**

| 깊이 | 정의 | 예시 (탈모 제품) |
|---|---|---|
| **표면** | 직접적 명시 | "탈모 관리" |
| **원초적** | 매슬로 1-2단계 | "외모 욕구·성적 욕구" |
| **우회** | 간접 연결 | 성욕 → 남편 사랑받기 → 아침형 남편 설거지 (간접 욕구 건드리기) |

## 핵심 6 — 메시지 삼위일체 (광고·상페·바이럴)

> 사전 VOD·부록의 "삼위일체"가 다시보기에서 더 구체화됨.

같은 메시지(예: "피부과 전문의 93% 추천")를 3개 채널 화자로 반복:
- 광고: 판매자 메시지 (강한 후킹)
- 상페: 판매자 + 전문가 (상세 근거)
- 바이럴: 블로거·유튜버 (경험 화자)

→ 본질 동일, 화자만 다름 = 자연스러움 + 설득력↑.

## 핵심 7 — 고객 의심 심리 4가지 (선제 해결)

> 모든 페이지가 답해야 할 고객 의심:

1. **"정말 효과적?"** — 임상·리뷰·전문가 권위
2. **"합리적 가격?"** — 비교 가격·할인·세트
3. **"나도 맞나?"** — Persona 구체 사례·비포/애프터
4. **"지금 안 사면?"** — FOMO·한정 수량·환불 보장

→ 경쟁사·커뮤니티 리뷰 스캔으로 사전 발굴 후 페이지에 선제 답변.
