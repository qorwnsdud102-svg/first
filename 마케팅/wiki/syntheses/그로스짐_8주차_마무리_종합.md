# 8주차 다시보기 3강 — 마무리 종합 + AI 자동화 + 사장님 사례

## 한 줄 요지

8주 코칭 마무리. **퍼포먼스 캔버스 = 나침반** 재강조 + 상세페이지 3 flow 무한 변주 + AI 자동화 워크플로우 (Claude Code + Gemini + Photoshop) + 권위 재정의 + 사장님 사례 (아이크림 유세라블).

## 핵심 1 — 퍼포먼스 캔버스 = 나침반

> "건망증 많은 인간이 제품 세부 내용을 다 기억할 수 없으므로, 캔버스가 그 기억을 대신함."

- 제품 정체성 = 강점 + 타겟 누구인지에 대한 방향성
- 정체성 맞는 것만이 고객들이 실제로 구매하게 만드는 콘텐츠 생산 가능
- 캔버스 없이는 매주 광고·콘텐츠 갱신 시 일관성 깨짐

## 핵심 2 — 상세페이지 3 flow 무한 변주

> 반복 구조: **문제 제기 → 문제 해결 → 근거 제시** (순서 가변, 화자·이미지만 바꿈)

- 광고에서 제시한 무의식·의식이 그대로 상세페이지에 투영
- 리뷰 개수·주문 수량 = 사회적 근거 + 숫자적 증거 (오프라인 "줄 서 있는" 장면의 온라인 대체)
- 리뷰 후기는 전문가/판매자가 아닌 "고객" 화자로 신뢰도 극대화

## 핵심 3 — AI 자동화 워크플로우 (특전 공개)

### Claude Code Terminal Skill — 경쟁사 자동 분석

새 도구 (8주차에 강사가 특전으로 공개):
- 입력: 카테고리 + 급상승 조건 (검색량 0→상승)
- 출력: 경쟁사 이름·브랜드 링크·Meta 광고 라이브러리 스크랩·자사몰 상세페이지 스크랩
- 활용: 수동으로 매주 경쟁사 찾던 것 → 자동 실행 (매주/매달 반복)

### 상세페이지 완전 자동화 (3단계 통합)

**기존**: Claude가 텍스트만
**확장**: Gemini로 이미지 생성 → Upscaling AI로 고화질화 → Photoshop API로 텍스트·강조색 삽입

**프로세스**:
1. 경쟁사 레퍼런스 + 우리 퍼포먼스 캔버스 → Claude가 기획서 작성
2. 기획서의 각 섹션(도입·문제·해결·근거·이미지 배경)을 Gemini에 개별 입력 → 이미지 생성
3. 각 이미지를 "새 채팅창"에서 단계적 수정 (배경·인물·색감 분리) → 일관성 유지
4. 최종 이미지 + 텍스트 → 디자이너에게 넘기면 "70-80% 완성 상태" → 외주 비용 절감

→ **한 장 생성 = 30분 → 비용 효율 3배 증대**.

## 핵심 4 — 권위 재정의 (연예인 외 모든 권위)

> "돈 있으면 좋지만, 없으면 **뉘앙스·무의식**으로도 권위 만들 수 있다"

| 권위 유형 | 비용 | 사례 |
|---|---|---|
| **연예인 (A급)** | 비쌈, 운 의존적 | IU 5억 투입 → 8배 성장 (운 80%) |
| **B급 연예인** | 2-3천만원 + 6개월 계약 | 촬영팀 + 스튜디오 직접 (퍼포먼스 필름) |
| **일반인 권위** | 0원 | 엄마·아빠·본인 스토리 |
| **스토리텔링 권위** | 0원 | "10년 고민 끝에 2만 6700시간 투입해 개발" |
| **제조사 권위** | 자료비 | 논문·박사·임상자료 (법적 테두리 먼저 설정) |

→ [[상징재]] · [[상징성과 당위성]] 권위 4분할 보강 자료.

## 핵심 5 — 사장님 사업 사례 — 아이크림 유세라블

> **8주차 1강에서 사장님 본인이 운영하는 사업 정체 일부 노출.**

- **시장 지위**: 온라인에서 **AHC와 어깨를 나란히 함** (AHC 오프라인·홈쇼핑 3천억 vs 자사 온라인 메인)
- **성장 동기**: 경쟁사 신고 다수 → 콘텐츠 "덜어내고 빼면서" 노멀화 → 그럼에도 판매 유지 이유 = **콘텐츠·홍보량 누적**
- **원가 절감 + 유통 명함**: 판매 규모 커지니 원가 싸지고, 유통처(올리브영 등)가 우리를 인정
- **확장 효과**: 홈쇼핑 진출 이유 = 매출이 아니라 **"노출" 목적** (검색량·인식도 상승)

→ 교훈: "매출만 보지 말고, 노출의 브랜드 시그널 효과를 함께 계산하라"

→ 사장님 SKU 중 하나 = **아이크림 (유세라블 카테)**. 자사몰 + 메타·GFA·구글 운영 추정. 향후 wiki 보강 시 사장님 사업 단서로 활용.

## 핵심 6 — 다른 수강생 사례 — 아기세제 카테고리 확장

- 문제: 아기세제 카테고리 급상승 브랜드 찾기 어려움
- 해결: **"같은 카테고리가 아니라 기전이 비슷한 류를 찾아라"** (예: 염색약·다이어트 식품)
- 실행: 30대 엄마들 문제 해결 구조 = 다른 카테고리도 동일 가능
- AB 테스트: 자사몰에서만 페이지 2개 (리뷰 많은 버전 vs 썸네일만 다른 버전) 비교

## 핵심 7 — 8주 종합 체크리스트

1. 퍼포먼스 캔버스 작성 완료 (정체성·페르소나·욕구·USP·근거)
2. 경쟁사 3개 분석 완료 (도입부~CTA 흐름 정리)
3. 자사 상세페이지 레이아웃 기획서 작성 (섹션별 목적 명시)
4. 권위 섭외 계획 (연예인 아니어도 OK — 엄마·아빠·본인 스토리도 권위)
5. AI 활용 workflow 숙달 (Claude·Gemini 단계별 생성 프롬프팅)
