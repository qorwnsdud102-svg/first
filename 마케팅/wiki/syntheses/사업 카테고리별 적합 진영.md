# 사업 카테고리별 적합 진영

## 종합 질문

사장님 사업의 **어떤 카테고리·SKU**가 4대 1차 출처 중 어느 라인에 가장 적합한가?
- [[엠타트업]] (네이버바이럴 dominant)
- [[환사장]] (자사몰+메타 단일)
- [[근육돌이]] (GFA·구글·메타 다채널 퍼포먼스, i-boss 168글)
- [[그로스짐]] (= 근육돌이 회사 [[마케팅모먼트]]의 8주 코칭, **퍼포먼스 캔버스** 메소드)

> 사장님 사업 정의 (`핵심-맥락.md` §2): "특정 분야 고정 X. 돈 되는 건 다 판다 — 기회·마진 중심으로 카테고리를 횡단."
> → 진영 선택 = **카테고리·SKU 단위 결정**. 같은 사장님이 SKU별로 다른 진영 라인 동시 운영 가능.

## 3 진영의 dominant 카테고리 (각 출처 사례 기반)

### [[엠타트업]] (네이버바이럴 dominant)

> "RCS·NRCS·자관인간·가치더하기·프레임 분리"가 가장 잘 먹히는 곳.

**적합 카테고리**:
- **저관여 5만원 이하 충동구매** — 자관인간으로 욕구 재배치, 가치더하기로 프리미엄
- **소비재 일반** (생활용품·소형 가전·잡화) — 다채널 분산 (쿠팡·네이버·카페·지식인)
- **이미 만들어진 시장에서 틈새 진입** — [[프레임 분리]]로 식약처 인증 연고 같은 기존 카테고리 분리
- **빠른 회전 + 빠른 폐기** — RCS·NRCS로 R 검증 → 안 되면 다음 R
- **사례**: 손톱 영양제 → 치료제 / 화장품 → 식약처 인증 연고 / 미세풍기 / 손가락 운동기구

**부적합**:
- 자사몰 단일 브랜드 빌딩 (장기 자산 축적 의도) — 채널 분산이 약점
- 패션·뷰티 감성 빌딩 (감성 비주얼 부족)

### [[환사장]] (퍼포먼스 dominant — 자사몰+메타 단일)

> "원가율 30% · 자사몰 5단계 · 메타 4캠페인"이 가장 잘 먹히는 곳.

**적합 카테고리**:
- **패션** — 감성형 단일 사진/영상으로 ROAS 1000% 가능
- **뷰티·코스메틱** — 감성 + B급 4형식 잘 작동
- **의외성 제품** — "OO인 줄 알았는데 OO" (데일밴드 → 관절패치)
- **트렌드 결합 제품** — 공복 올리브 → 알약, 뽕브라 → 감성 연출
- **충동구매 40-50대 타겟** — 다이어트·주름·탈모·건강식품 (환사장 명제 6번)
- **원가율 30% 이하 가능 제품** — 판매가 3만 → 9천 이하 소싱 가능해야 메타광고 ROAS 성립
- **자체 브랜드 빌딩 의도가 있는 SKU** — 자사몰 = 장기 자산

**부적합**:
- 원가율 30%+ 제품 — 메타광고 마진 안 남음
- 다채널 분산 의도 (자사몰 단일 라인)
- 신뢰·전문성 요구 카테고리 (의료기기·전문가용)

### [[그로스짐]] (퍼포먼스 dominant — 8주 코칭 + 퍼포먼스 캔버스)

> "퍼포먼스 캔버스 1장 + 콘텐츠 5요소 + 사회적 증거 4축"이 가장 잘 먹히는 곳.
> 사장님이 **본인 사업 SKU에 대해 직접 캔버스를 작성·실험**하는 단계.

**적합 카테고리**:
- **광고 콘텐츠가 핵심인 모든 카테고리** (= 거의 전부)
- **컨셉·한끗 차이로 시장 포지셔닝 새로 짜야 하는 SKU** — 말기주름·반달크림·약국 용기형 화장품 등 사례 활용
- **표면적 욕구 vs 원초적 욕구가 분리되는 카테고리** — 다이어트(자아실현 표면 + 식욕)·향수(섹시 표면 + 성)·뷰티 일반
- **AI 활용으로 시장조사·리뷰 분석 자동화 가능한 SKU** — Claude MCP·크롬 자동화 적용 가능
- **사장님이 8주간 1:1 코칭 받는 환경에 있는 모든 SKU** — 그로스짐 자체가 사장님 본인 사업 코칭

**부적합**:
- 컨셉·USP가 이미 확정되어 더 이상 캔버스 작성 의미 적은 SKU
- 광고 콘텐츠가 부차적이고 채널·운영이 주가 되는 SKU (그건 환사장·근육돌이)

### [[근육돌이]] (퍼포먼스 dominant — GFA + 구글 + 메타 + 페북 종합)

> "GFA 800억 · 구글 60억 · CTR 10배 카피 · 정육점 재구매율 86%"가 가장 잘 먹히는 곳.

**적합 카테고리**:
- **뷰티·건기식** — GFA CPC 300원 가능 (M1 그룹)
- **패션** — GFA 패션 쇼핑몰 소재 1만 개 등록 사례
- **다채널 GFA + 구글 + 메타 동시 운영** — 단일 라인 X, 매체 믹스
- **CRM·재구매 핵심 카테고리** — 식품·일상소비재 (정육점 사례 응용)
- **오프라인 매장 + 온라인 결합** — 정육점 12개 사례 (M10 그룹). 사장님이 오프라인까지 확장한다면 직결
- **컨텐츠 강한 카테고리** — "광고에서 컨텐츠 10배 중요" — 컨텐츠 자산 누적 가능한 카테고리

**부적합**:
- 빠른 회전·소량 테스트 (다채널 운영 부하)
- 자본·인력 부족 (광고비 큰 단위 운영)

| **자본 큼 + 매체 믹스 + 컨텐츠 자산** | [[근육돌이]] | GFA 800억 노하우 + 컨텐츠 시리즈 |

## 미해결 — 사장님 직접 결정 필요

- 사장님 현재 운영 SKU가 어느 카테고리·자본·운영 단계인지 (위키 안에 명시 안 됨)
- 한 SKU에 한 진영만 적용할지, 자관인간(사고)+환사장(실행)+근육돌이(매체 믹스) 직렬 통합으로 갈지 → [[자관인간과 메타광고의 통합]] 참조
- 오프라인 매장(정육점 사례 응용) 확장 의향
- 단일 자사몰 vs 다채널 동시 (운영 부하 결정)
