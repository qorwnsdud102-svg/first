# 자관인간 + 메타 4캠페인 — 한 자사몰에 어떻게 통합되나

## 종합 질문

[[엠타트업]]의 [[통찰이론 4단계]] (자관인간 — 네이버바이럴 진영)과 [[환사장]]의 메타 4캠페인 (퍼포먼스 진영)은 어떻게 **한 자사몰 운영에 직렬 연결**되나? 둘은 충돌하는가, 보완하는가?

## 한 줄 답

**보완. 직렬 연결.** 자관인간은 광고 **소재·메시지를 만드는 사고 도구**이고, 메타 4캠페인은 그 소재를 **돌리는 운영 도구**다. 환사장 본인 명제 "**소재가 꽃, 광고 관리는 물 주기**"가 곧 두 진영의 직렬 통합 선언.

## 4 진영의 정확한 위치 (그로스짐 합류 후 갱신)

| 도구 | 출처 | 다루는 것 | 입력 | 출력 |
|---|---|---|---|---|
| **자관인간 4단계** | [[엠타트업]] (네이버바이럴) | R(리소스/제품) 사고 분석 | 막연한 제품 | 통찰(타겟이 나를 어떻게 보는지를 내가 아는 상태) |
| **퍼포먼스 캔버스** | [[그로스짐]] (퍼포먼스) | 통찰의 워크시트화 | 통찰 + 시장 데이터 | 1장 캔버스 (존재이유·USP·Pain·페르소나·사회적 증거 4축) |
| **광고 콘텐츠 5요소** | [[그로스짐]] (퍼포먼스) | 캔버스 → 콘텐츠 변환 | 캔버스 | 후킹·스토리·설득·포모·CTA 흐름 |
| **광고 소재 4형식** | [[환사장]] (퍼포먼스) | 콘텐츠의 시각화 | 콘텐츠 + 트렌드 | 감성/B급/시나리오 소재 |
| **메타 4캠페인** | [[환사장]] (퍼포먼스) | 소재 분배 | 소재 + 픽셀 | 매출 + 머신러닝 데이터 |
| **자사몰 5단계 플로우** | [[환사장]] (퍼포먼스) | 고객 여정 운영 | 광고 노출 | 인지→고려→탐색→평가→구매 |
| **GFA·구글 매체 운영** | [[근육돌이]] (퍼포먼스) | 매체 믹스 | 소재 + 매체별 | GFA 800억·구글 60억 노하우 |

→ **자관인간 = 0번 단계 (사고)**. 퍼포먼스 캔버스 = **0.5번 (통찰을 1장 워크시트로 정리)**. 환사장 운영 매뉴얼이 시작되기 전. R이 통찰되지 않은 채 메타 캠페인을 돌리면 "머신러닝 깨졌어요 ㅠㅠ"의 함정.

## 직렬 통합 7단계 (4 진영 통합 가설)

1. **자관인간 1-4단계로 제품 통찰 완성** — [[엠타트업]]
   - 자각: 내 제품의 진짜 정체 (예: "뻔한 선풍기")
   - 관찰: 타겟의 욕구 ([[NRCS]] 분리, [[욕구와 결핍]])
   - 인지: 타겟이 내 제품을 어떻게 보는지 (1점 후기·카페 반응)
   - 간파: 그 인식을 내가 아는 상태에서 [[프레임 분리]]·[[가치더하기]]
2. **퍼포먼스 캔버스로 1장 정리** — [[그로스짐]]
   - 본질적 존재이유 → 문제·해결 → 매슬로 욕구 (표면 vs 원초적) → USP
   - Pain Point + 고통지수 1~10 → 페르소나 5요소
   - 4가지 사회적 증거 (통념·근거·권위·숫자) 채우기
3. **콘텐츠 5요소로 광고 흐름 짜기** — [[그로스짐]]
   - 후킹 (경쟁사 100개 캡처 → 공통 키워드)
   - 스토리텔링 ([[스토리텔링]])
   - 설득 (4가지 사회적 증거)
   - 프로모션·포모 (광고 X, 랜딩에서만 — 네이티브 보호)
   - CTA (셀링 광고 강하게 / 네이티브 약하게)
4. **환사장 광고 소재 4형식으로 시각화** — [[환사장]]
   - "뻔한 선풍기 → 미세풍기" 같은 [[가치더하기]] 카피 → 감성형 또는 B급
   - 환사장 명제 1·2 ("대중은 똑똑하지 않음 + 그럴싸해보이면 구매") 적용
5. **자사몰 5단계 플로우 + 물 아래 셋업** — [[환사장]]
   - 물 아래: 리뷰 500-1000개, 블로그·카페·바이럴 콘텐츠 깔기
   - 상페: [[상페 구조]] 3 모델 중 선택 (오메클·자사몰 5단계·USP+Pain+손실 3축)
6. **매체별 캠페인 돌리기** — [[환사장]] + [[근육돌이]]
   - 자사몰 단일 라인이면 환사장 메타 4캠페인
   - 다채널이면 [[근육돌이]] 168글의 GFA·구글·메타 매체 믹스
   - END ROAS 미만 OFF / CTR 1% 미만 후킹 부족 → 1단계 자관인간으로 회귀
7. **재구매 사이클 + CRM** — [[근육돌이]] M10 정육점 86% / [[환사장]] 카카오톡 CRM
   - ROAS 단순값 X, 1·2차 구매·재구매율로 평가

## 두 진영이 충돌하는 지점 (있다면)

| 축 | 엠타트업 입장 | 환사장 입장 | 통합 |
|---|---|---|---|
| 채널 | 다채널 종합 (쿠팡·네이버·카페·지식인) | 자사몰 + 메타 단일 라인 | **사장님 사업 단계에 따라 선택** — 자사몰 빌딩기엔 환사장, 다채널 회전엔 엠타 |
| 상페 모델 | 오메클 (오프닝-메리트-클로징, 오프닝 90%) | 5단계 플로우 (인지-고려-탐색-평가-구매) | **둘 다 [[상페 구조]] 페이지에 공존**. 단일 상페 안에서 오메클 구조 + 5단계 동선 동시 적용 가능 |
| 사고 vs 실행 | 통찰·프레임 (사고법) | KPI·픽셀·소재 (수치) | **직렬** (사고 → 실행), 충돌 X |
| 소재 우선도 | "자각이 정확해야 가치더하기 작동" | "소재가 꽃, 머신러닝은 부차" | **완벽 합치** — 두 진영 모두 "통찰 → 소재"가 매출 본질 |
| 근육돌이 보완 | "광고에서 컨텐츠가 10배 중요" | (같은 결) | 3 진영이 다 같은 명제 |

→ **충돌 거의 없음.** 채널 범위만 사업 단계로 결정하면 됨.

## 사장님 운영 시나리오 — "자사몰 한 SKU 런칭" 예시

가설: 사장님이 새 제품 1 SKU로 자사몰 + 메타광고 라인 시작한다고 가정.

- **D-30 ~ D-15**: 자관인간 1-4단계로 R 통찰 ([[엠타트업]] 사고법). [[NRCS]]·[[욕구와 결핍]]·[[가치더하기]] 카피 후보 5-10개.
- **D-14 ~ D-7**: 자사몰 셋업 + 물 아래 ([[환사장]] 매뉴얼). 리뷰 500개·블로그 체험단·기사 깔기.
- **D-6 ~ D-0**: 광고 소재 4형식 제작. 통찰 결과 → 감성/B급/시나리오 변환. END ROAS 가설 (마진 + 광고 비중 + 안전 마진).
- **D+1 ~**: 메타 판매 캠페인 ABO 시작. 일주일 50개 = 머신러닝. 터진 소재 어드벤티지 + 리타겟 추가.
- **D+30 ~**: [[근육돌이]] 사이클 평가. ROAS 단순값 X, 1·2차 구매·재구매율로 평가. CRM 카카오톡 채널.

## 미해결 (사장님 결정 필요)

- 자사몰만 할지, 다채널 (쿠팡·네이버 스마트스토어 동시 운영)도 할지 — 둘 다 가능하지만 운영 부하 차이 큼.
- 환사장은 자사몰 단일이지만 [[근육돌이]] 168글에는 GFA·구글까지 다채널 ([[마케팅/근육돌이]] M1·M2 그룹).
- 사장님이 자본·인력 여건에 따라 선택. → [[사업 카테고리별 적합 진영]] synthesis 페이지로 분리 분석.
