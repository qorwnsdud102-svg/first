# 1-3주차 우선피드백 통합 — 광고주 5명 사례 + 자주 틀리는 패턴 7

## 한 줄 요지

강사가 수강생 광고주 캔버스·콘텐츠를 직접 피드백한 8 회차. **자주 틀리는 패턴 7개** 추출 + **광고주 5명 카테고리별 사례**. 사장님이 그로스짐 수강생이라면 본인 SKU 피드백도 일부 포함 가능.

## 자주 틀리는 패턴 7 (강사 반복 지적)

| # | 패턴 | 강사 지적 | 해결 |
|---|---|---|---|
| **1** | Persona 과도하게 추상적 | "30대 여성", "초보 엄마" 통계만 | 구체 직업·거주지·소득·생활 모습 채우기. "내 얘기다" 느낄 요소 |
| **2** | 문제-해결 연결고리 약함 | "여드름 있다 → 없어졌다" 표면만. "왜 그런 문제? 왜 이 제품?" 미흡 | 근본·인과 명시 (고객 의문·이해 막힘 제거) |
| **3** | 근거 조사가 키워드 중심 (판매자 관점) | 성병명·제모 키워드만. 커뮤니티 고민 토로 분석 부족 | "고객 관점"으로 → 공감 요소 |
| **4** | USP·경쟁력·권위 추상적 | "안심", "최초" 단어만. 어디서 인증? 누가 1등? 비워짐 | 구체 자료·숫자·인증 표시 |
| **5** | 욕구 1-2개 나열, 표면/원초 미분리 | 매슬로 미연결. "성욕"·"안전욕" 단어만, 후킹될 장면 없음 | 표면+원초적 분리 + 시각화(비유·이미지·상황) |
| **6** | 사회적 증거 1-2개뿐 | 권위·통념만, 근거·숫자 빈약 | 3-4개 채우기. 리뷰·임상·설문 데이터 |
| **7** | 포지셔닝 두 방향성 섞음 | (왁싱 + 여드름) (냄새 + 가려움) 이중 목표 | "캔버스 2개로 분리하라" — 한 페이지 = 한 메시지 |

## 광고주 5명 사례 (카테고리별 디테일)

| 광고주 | SKU/카테고리 | 주요 피드백 |
|---------|------------|-----------|
| **최용민** | 얼굴 제모 기기 (잔털+여드름) | 잔털 vs 여드름 명확화 요구. USP 분산 지적. 이미지 준비 잘 함. |
| **이현지** | 아기 옷 세탁용 세제 | 문제 장면 심화 필요 ("빨개진 피부" 강조). 후킹 약함 ("포근한 거실" → 자극적으로). |
| **정선경** | 여성 팬티 (Y존 냄새) | Persona 가장 약함. 근거 조사 키워드만. 성욕 vs 안전욕 미분리. |
| **최은미** | 여드름 개선 제품 (턱 여드름) | 콘셉트 혼합 (제모 + 여드름). 카피는 최고 수준이나 무의식 약함. |
| **준영** | 허리 견인기 (중년 허리 통증) | AI 사진만 의존. 진단서·수술 피한다 같은 "손해 강조" 시각화 부족. |

## 강사 명제 — 사장님 학습 즉시 활용

- **"충동 구매 = 계획형 구매 아님"** (Persona 검증 중요)
- **"우회 표현·비유"** (플랫폼별 심의 회피 기술)
- **"전후 비교 = 고객 입장의 손해/불편 먼저 심화 → 해결"** ([[매몰비용]]·고통지수 활용)
