# 환사장

## 한 줄 정의

**연 100억 매출** 자사몰 사업가. 2026-02-05부터 4주간 "아마겟돈" 자사몰 + 메타광고 유료 강의 진행. [[엠타트업]]과 별개 출처. 사장님 학습의 **3번째 1차 출처**.

## [[엠타트업]]과의 차별점

| 축 | 엠타트업 (M군) | 환사장 |
|---|---|---|
| 채널 범위 | 다채널 (쿠팡·네이버·카페·지식인) 종합 | **자사몰 + 메타 광고** 단일 라인 |
| 강의 톤 | 사고 프레임 (자관인간·프레임 조종) | 운영 매뉴얼 (예산 증감액·픽셀 세팅) |
| 사례 풍부도 | 다양한 카테고리 | 자사몰 100억대 사례 집중 |
| 깊이 | 통찰·사고법 | 실행·수치·KPI |

→ 두 출처는 **상호 보완**. 엠타트업으로 사고를 짜고, 환사장으로 자사몰 실행 디테일을 채운다.

## 강의 4주차 전체 구조

| 주차 | 핵심 내용 |
|---|---|
| 1주차 | 자사몰 기본 세팅 + 팔리는 제품 찾는 법 + 브랜드 기획 + 원가율 30% |
| 2주차 | 리뷰·블로그·기사·지식인 깔기 + 메타 광고 픽셀·캠페인 4종 세팅 |
| 3주차 | 광고 소재 제작·디벨롭 4형식 + 카카오톡 CRM |
| 4주차 | 댓글 작업·반응도 + 트렌드·밈 + 100억 브랜드 레퍼런스 + 돈 버는 마케팅 10명제 |

## 자사몰 핵심 — 5단계 판매 플로우

> 우리가 광고를 보고 제품을 사는 과정 = 자사몰 판매 플로우

1. **인지** — '끌리는' 광고로 사이트 직접 OR 자사몰 검색 유입
2. **고려** — 상페와 리뷰를 본다 (저관여 5만원 이하 → 충동구매 / 고관여 → 추가 검증)
3. **탐색** — 타사와 비교
4. **평가/확인** — 블로그·카페·유튜브 **내돈내산** 후기 검색
5. **구매·재구매** — CRM (카카오톡 채널)

### 물 아래 vs 물 위 작업

| 구분 | 작업 | 초반 중요도 |
|---|---|---|
| **물 아래** (깔아두기) | 리뷰 500-1000개 (상위 5개+포토·영상이 핵심) | **상** |
| 물 아래 | 블로그·카페 체험단, 기사 배포, 유튜브 체험단 | 하 |
| 물 아래 | 자사 인스타 피드 | 중하 |
| **물 위** (광고) | 메타 광고 (초반 집중) | **상** |
| 물 위 | 협력광고 (인플루언서 릴스) | 중 |
| 물 위 | 유튜버 의뢰 + 2차 활용 | 중 |

## 자사몰에서 터지는 제품 4공통점

1. **제품 자체 유니크함**(신비성) — 눈에 바로 보이는 특이함
   - 예: 기존 눈썹칼 → 눈썹 정리기계
2. **의외성** — OO인 줄 알았는데 OO인 제품
   - 예: 데일밴드인 줄 알았는데 관절패치
3. **트렌드 + 문제 보완 + 제품 연결**
   - 예: 공복 올리브 효능 유행 → 비림 → 알약제조
4. **기능(컴플렉스/B급) + 감성의 조화**
   - 예: 뽕브라 → 감성 연출

## 메타 광고 4 캠페인

| 캠페인 | 약자 | 목적 | 운영 |
|---|---|---|---|
| **판매** | ABO | 매출 (기본) | N개 세트 - 1개 소재. 논타겟 + 메타 AI |
| **어드벤티지** | CBO | "메타가 자동 분배" | 소재 10개+ 일괄, 일예산 20만+ |
| **트래픽** | - | 초반 모수·인지도 | 예산 비중 0.5-1% |
| **리타겟** | - | "살 때까지 광고할게" | 14일 방문자·30일 장바구니. 판매캠 50% 예산 |

### 핵심 KPI

- **CTR 1% 미만 → 후킹 부족**
- **END ROAS 미만 → 즉시 OFF**
- CPM 1-3만원이 정상
- 일주일 50개 판매 = 머신러닝 완료

> "머신러닝이고 나발이고…**광고 소재 자체가 가장 중요하다.**
> 머신러닝 깨졌어요 ㅠㅠ 할 시간에 알고리즘 터진 소재 하나 더 적용하기."

## 광고 소재 4 형식

1. **감성형** — 패션·뷰티 (단일 사진/영상, 여성 의류 ROAS 1000% 가능)
2. **감성 + B급** — 대부분 카테고리 (감성 베이스 + 카피 멘트)
3. **B급 + B급** — 어그로 + 직관적 소구점
4. **시나리오 형식** — 협력광고 + 2차 활용 + B급 (현재 가장 효율 높음)

## 돈 버는 마케팅 10명제 (4주차 핵심)

1. **대중은 똑똑하지 않음** — 사기·과대광고 회사도 월 수십억 판매
2. **그럴싸해보이면 구매** — 광고 3초 후킹·상페 연출·외국인 모델
3. **100마디 판매자 < 1마디 친구** — 리뷰 (자사몰 → 블로그 → 카페·유튜브)
4. **권위의 힘** — 올리브영·29CM·무신사 입점 / 연예인·인플루언서 후기
5. **조급함 유도가 기본** — "재고 예상 소진 OO일 / 재입고 2주" 카피
6. **충동 구매 타겟 = 40-50대** — 다이어트·주름·탈모·건강
7. **진짜 브랜드는 말이 없다** — 명품 (상페 텍스트 X)
8. **누끼컷 4장으로 판매** — 제품 설명 < 리뷰·사회적 증거
9. **니즈 → 욕망** — "필요하세요?" X, "**더 예뻐지고 싶은 욕망**" → 워너비 마케팅
10. **입소문(내돈내산·선물) = 브랜드 시작**

## 사장님 사업 직결 핵심 인사이트 5개

1. **원가율 30% 이하** = 메타광고 적정 원가 (판매가 3만원 → 9천원 이하 소싱)
2. **물 아래 먼저** — 리뷰 500-1000개 깔린 후에 광고 시작
3. **소재가 꽃, 광고 관리는 물 주기** — 머신러닝 X, 소재 기획에 집중
4. **터진 소재 복사 증액** — END ROAS 2배 이상이면 그대로 복제
5. **B급 페이지 개설** — 경쟁사 추적·신고 회피용 별도 페이지
