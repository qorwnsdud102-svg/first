# 그로스짐

## 한 줄 정의

[[마케팅모먼트]](= [[근육돌이]] 회사)의 8주 1:1 코칭형 이커머스 강의 프로그램. 2기 = 테스트 기수 다음의 첫 정식 기수. 사장님 학습의 **4번째 1차 출처**.

## 출처 관계 — [[근육돌이]]와 동일 회사

마케팅모먼트는 [[근육돌이]] 작가가 운영하는 마케팅 광고대행사. 그로스짐은 이 회사의 교육 프로덕트.

- 근육돌이 i-boss 168글 = **개인 글쟁이 톤** (회고·인사이트·시리즈)
- 그로스짐 39자료 = **8주 코칭 프로그램 톤** (실습·피드백·과제 중심)
- 근육돌이 글 [[100억짜리-자료-퍼포먼스-캔버스-활용법]]이 그로스짐의 핵심 메소드 = 같은 줄기.

→ 두 출처는 **상호 보완**. 근육돌이로 큰 회로를, 그로스짐으로 캔버스 작성·디테일 채움.

## 강사 정체

- 마케팅모먼트 광고대행사 대표 (이름 본문에서 확인 안 됨; **민상·민트** 같은 사내 호칭 등장).
- 14년차 이커머스. 스마트스토어/스토어팜 100개 컨설팅.
- **마리보**(식물) 국내 1등 (연 매출 70억) — 팀장 시절.
- 네이버 GFA **3천억** 집행 (근육돌이의 220억보다 더 큰 누계 — 회사 단위).
- 페이스북·구글·틱톡 등 DA광고 운영. 자체 뷰티 브랜드 하루 매출 **3억**, 월 매출 500억대 뷰티 브랜드 함께 키움.

## 강의의 메인 프레임

### 1. 퍼포먼스 캔버스 (Performance Canvas)

> 강사가 정립한 핵심 워크시트. 본질적 존재이유 → 문제·해결 → 매슬로 욕구 → USP → Pain Point → 페르소나 → 사회적 증거 4축까지 한 장에 정리.

근육돌이 i-boss 글 [[100억짜리-자료-퍼포먼스-캔버스-활용법]]이 같은 도구의 회고판.

### 2. 성공 콘텐츠 5요소 (광고 콘텐츠 흐름)

1. **후킹** — 콘텐츠를 볼 이유. 경쟁사+오가닉 콘텐츠 100개 캡처 → 공통 키워드 추출 → 제품에 녹이기.
2. **스토리텔링** — 고객 공감 ([[스토리텔링]]).
3. **설득** — 4가지 사회적 증거 ([[상징성과 당위성]] · [[상징재]]).
4. **프로모션 / 포모** — 광고가 아니라 **랜딩 페이지**에서 (네이티브 광고 보호) ([[매몰비용]]).
5. **행동요구 (CTA)** — 셀링 광고에선 강하게, 네이티브에선 약하게.

### 3. 4가지 사회적 증거 (강사의 가장 핵심 프레임)

> "결국에 모든 e-commerce 콘텐츠는 다 동일하다. 이거의 반복."

| 축 | 정의 | 예시 |
|---|---|---|
| **사회적 통념** | 보편적 상식·규범 | "초등학생도 이해하는 키워드", "약사·피부과 추천" |
| **사회적 근거** | 많은 사람이 쓰고 있다는 증거 | 몇 명 샀습니다 / 줄 서 있는 사진 / 별점·리뷰 개수 |
| **사회적 권위** | 명사·전문가·연예인 | 강남 유명 피부과 / 인플루언서 사진 컬렉션 / 박사·교수 |
| **숫자적 증거** | 객관성 강화 | "00% 개선 임상" / "네이버·쿠팡 N위" / "몇만 개 판매" |

[[상징성과 당위성]] (당위성 = 통념) + [[상징재]] (상징성 = 권위·숫자) 와 같은 결의 4분할.

### 4. 표면적 vs 원초적 욕구 (매슬로 + 인간 기본 욕구)

- 표면적 이유 (예: "예뻐지려고") → **원초적 이유 (식·성·수면·생존)**가 진짜 동력.
- 다이어트 광고 = 자아실현 표면 + **식욕** 원초적 (음식 녹이는 장면, 당뇨 트렌드 키워드).
- 명품 향수에 섹시 모델 쓰는 이유 = 향수의 **성적 욕구** 레이어.
- 단계 레이어를 깔수록 매출 차이 — 1축만 건드리는 vs 표면+원초+트렌드 키워드.

→ [[욕구와 결핍]] (NRCS의 N) 보강 사례.

### 5. 고객 정의 5요소 (페르소나 ≠ 인구통계만)

1. 인구통계 (나이·성별·거주지·소득)
2. **고객 행동** — 우리 제품 만나기까지의 고민·동선
3. **Pain Point + 고통지수** — 1~10. 충동구매가 일어나려면 ~10 (불치병형: 기미·주름·트러블)
4. 실존인물처럼 느껴지는 스토리
5. 정의는 정답이 아니라 테스트로 발전 (바디워시 → 향기 바디워시로 한끗 전환 → 월 매출 10억)

### 6. 컨셉·포지셔닝 (한끗 차이)

> "세상에 없는 제품은 못 만든다. 컨셉과 포지셔닝으로 승부."

- **말기주름** (말기 = 암 연상 → 무서운 키워드)
- **반달크림** (이마에 바르면) / 애기손 크림 / 깊은 계곡주름
- 약국 연고형 용기 + 약국풍 디자인 = 일반 화장품 → 약처럼 인식
- 마리오네트 주름·약산성 폼클렌징 = 누군가 만들어낸 컨셉 단어

→ [[프레임 분리]] · [[가치더하기]] 보강 사례.

### 7. 상세페이지 구조 (USP + Pain + 손실 3축)

- **요약 포인트 3**: USP + Pain Point + **고객의 손실·손해** 강조
- **화자 4종**: 고객(리뷰) / 전문가(임상·근거) / 판매자(셀링) / 제3자(연예인·인플루언서·외부 전문가)
- 순서는 가변. 광고 데이터로 매번 갱신 (연예인 위 vs 리뷰 위 등).
- **GIF는 USP 핵심 1-2곳에만** (남발하면 사이트 느려짐 → 3초 이탈).
- **3 flow**: 문제 제기 → 문제 해결 → 근거 제시 (순서 가변, 화자만 바꾸면 무한 변주).

→ [[상페 구조]] 보강 사례 (오메클·자사몰 5단계와 다른 3번째 모델).

### 8. 광고 콘텐츠 / 랜딩 / 바이럴 3축

콘텐츠를 광고만이 아니라 3종으로 본다:
1. **광고 콘텐츠** — SNS·DA. 후킹·스토리텔링·약한 CTA.
2. **랜딩 페이지 콘텐츠** — 강한 프로모션·포모·CTA OK.
3. **바이럴 콘텐츠** — 블로그·카페·지식인·유튜브·인스타 게시물 (환사장의 "물 아래 깔기"와 같은 결).

### 9. AI 활용 (Claude · Gemini · MCP)

- 강사 메인 = **Claude** (MCP로 데스크탑 파일·폴더 자동 정리, 크롬 자동화로 i-boss·유튜브·커뮤니티 분석).
- ChatGPT는 보조. Gemini도 활용.
- 시장조사·리뷰 만 개 분석·재미있는 리뷰 추출·유튜브 썸네일 30개+댓글 자동 분석 등.
- **딥리서치는 정형화된 보고서 → 소비자 카피용 스몰 데이터에는 부적합.**
