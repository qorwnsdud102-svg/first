# 그로스짐

## 한 줄 정의

[[마케팅모먼트]](= [[근육돌이]] 회사)의 8주 1:1 코칭형 이커머스 강의 프로그램. 2기 = 테스트 기수 다음의 첫 정식 기수. 사장님 학습의 **4번째 1차 출처**.

## 출처 관계 — [[근육돌이]]와 동일 회사

마케팅모먼트는 [[근육돌이]] 작가가 운영하는 마케팅 광고대행사. 그로스짐은 이 회사의 교육 프로덕트.

- 근육돌이 i-boss 168글 = **개인 글쟁이 톤** (회고·인사이트·시리즈)
- 그로스짐 39자료 = **8주 코칭 프로그램 톤** (실습·피드백·과제 중심)
- 근육돌이 글 [[100억짜리-자료-퍼포먼스-캔버스-활용법]]이 그로스짐의 핵심 메소드 = 같은 줄기.

→ 두 출처는 **상호 보완**. 근육돌이로 큰 회로를, 그로스짐으로 캔버스 작성·디테일 채움.

## 강사 정체

- 마케팅모먼트 광고대행사 대표 (이름 본문에서 확인 안 됨; **민상·민트** 같은 사내 호칭 등장).
- 14년차 이커머스. 스마트스토어/스토어팜 100개 컨설팅.
- **마리보**(식물) 국내 1등 (연 매출 70억) — 팀장 시절.
- 네이버 GFA **3천억** 집행 (근육돌이의 220억보다 더 큰 누계 — 회사 단위).
- 페이스북·구글·틱톡 등 DA광고 운영. 자체 뷰티 브랜드 하루 매출 **3억**, 월 매출 500억대 뷰티 브랜드 함께 키움.

## 강의의 메인 프레임

### 1. 퍼포먼스 캔버스 (Performance Canvas)

> 강사가 정립한 핵심 워크시트. 본질적 존재이유 → 문제·해결 → 매슬로 욕구 → USP → Pain Point → 페르소나 → 사회적 증거 4축까지 한 장에 정리.

근육돌이 i-boss 글 [[100억짜리-자료-퍼포먼스-캔버스-활용법]]이 같은 도구의 회고판.

### 2. 성공 콘텐츠 5요소 (광고 콘텐츠 흐름)

1. **후킹** — 콘텐츠를 볼 이유. 경쟁사+오가닉 콘텐츠 100개 캡처 → 공통 키워드 추출 → 제품에 녹이기.
2. **스토리텔링** — 고객 공감 ([[스토리텔링]]).
3. **설득** — 4가지 사회적 증거 ([[상징성과 당위성]] · [[상징재]]).
4. **프로모션 / 포모** — 광고가 아니라 **랜딩 페이지**에서 (네이티브 광고 보호) ([[매몰비용]]).
5. **행동요구 (CTA)** — 셀링 광고에선 강하게, 네이티브에선 약하게.

### 3. 4가지 사회적 증거 (강사의 가장 핵심 프레임)

> "결국에 모든 e-commerce 콘텐츠는 다 동일하다. 이거의 반복."

| 축 | 정의 | 예시 |
|---|---|---|
| **사회적 통념** | 보편적 상식·규범 | "초등학생도 이해하는 키워드", "약사·피부과 추천" |
| **사회적 근거** | 많은 사람이 쓰고 있다는 증거 | 몇 명 샀습니다 / 줄 서 있는 사진 / 별점·리뷰 개수 |
| **사회적 권위** | 명사·전문가·연예인 | 강남 유명 피부과 / 인플루언서 사진 컬렉션 / 박사·교수 |
| **숫자적 증거** | 객관성 강화 | "00% 개선 임상" / "네이버·쿠팡 N위" / "몇만 개 판매" |

[[상징성과 당위성]] (당위성 = 통념) + [[상징재]] (상징성 = 권위·숫자) 와 같은 결의 4분할.

### 4. 표면적 vs 원초적 욕구 (매슬로 + 인간 기본 욕구)

- 표면적 이유 (예: "예뻐지려고") → **원초적 이유 (식·성·수면·생존)**가 진짜 동력.
- 다이어트 광고 = 자아실현 표면 + **식욕** 원초적 (음식 녹이는 장면, 당뇨 트렌드 키워드).
- 명품 향수에 섹시 모델 쓰는 이유 = 향수의 **성적 욕구** 레이어.
- 단계 레이어를 깔수록 매출 차이 — 1축만 건드리는 vs 표면+원초+트렌드 키워드.

→ [[욕구와 결핍]] (NRCS의 N) 보강 사례.

### 5. 고객 정의 5요소 (페르소나 ≠ 인구통계만)

1. 인구통계 (나이·성별·거주지·소득)
2. **고객 행동** — 우리 제품 만나기까지의 고민·동선
3. **Pain Point + 고통지수** — 1~10. 충동구매가 일어나려면 ~10 (불치병형: 기미·주름·트러블)
4. 실존인물처럼 느껴지는 스토리
5. 정의는 정답이 아니라 테스트로 발전 (바디워시 → 향기 바디워시로 한끗 전환 → 월 매출 10억)

### 6. 컨셉·포지셔닝 (한끗 차이)

> "세상에 없는 제품은 못 만든다. 컨셉과 포지셔닝으로 승부."

- **말기주름** (말기 = 암 연상 → 무서운 키워드)
- **반달크림** (이마에 바르면) / 애기손 크림 / 깊은 계곡주름
- 약국 연고형 용기 + 약국풍 디자인 = 일반 화장품 → 약처럼 인식
- 마리오네트 주름·약산성 폼클렌징 = 누군가 만들어낸 컨셉 단어

→ [[프레임 분리]] · [[가치더하기]] 보강 사례.

### 7. 상세페이지 구조 (USP + Pain + 손실 3축)

- **요약 포인트 3**: USP + Pain Point + **고객의 손실·손해** 강조
- **화자 4종**: 고객(리뷰) / 전문가(임상·근거) / 판매자(셀링) / 제3자(연예인·인플루언서·외부 전문가)
- 순서는 가변. 광고 데이터로 매번 갱신 (연예인 위 vs 리뷰 위 등).
- **GIF는 USP 핵심 1-2곳에만** (남발하면 사이트 느려짐 → 3초 이탈).
- **3 flow**: 문제 제기 → 문제 해결 → 근거 제시 (순서 가변, 화자만 바꾸면 무한 변주).

→ [[상페 구조]] 보강 사례 (오메클·자사몰 5단계와 다른 3번째 모델).

### 8. 광고 콘텐츠 / 랜딩 / 바이럴 3축

콘텐츠를 광고만이 아니라 3종으로 본다:
1. **광고 콘텐츠** — SNS·DA. 후킹·스토리텔링·약한 CTA.
2. **랜딩 페이지 콘텐츠** — 강한 프로모션·포모·CTA OK.
3. **바이럴 콘텐츠** — 블로그·카페·지식인·유튜브·인스타 게시물 (환사장의 "물 아래 깔기"와 같은 결).

### 9. AI 활용 (Claude · Gemini · MCP)

- 강사 메인 = **Claude** (MCP로 데스크탑 파일·폴더 자동 정리, 크롬 자동화로 i-boss·유튜브·커뮤니티 분석).
- ChatGPT는 보조. Gemini도 활용.
- 시장조사·리뷰 만 개 분석·재미있는 리뷰 추출·유튜브 썸네일 30개+댓글 자동 분석 등.
- **딥리서치는 정형화된 보고서 → 소비자 카피용 스몰 데이터에는 부적합.**
- **Claude 200달러 Max 플랜** + **Claude for Excel** (RAG 역할 — 데이터 직접 접근으로 hallucination 회피) + **5~7창 동시**.
- **AI 워크플로우 7단계**: 새 채팅 분리 (context hallucination 회피) / 큐샷 예시 / 단계별 1개씩 수정 / 매번 직전 결과 재첨부.
- **Gemini Nano Banana** = 이미지 생성 메인. **한 장씩 새 채팅창 단계 빌드업** = 디자이너 외주 100만→30~50만 절감.
- **Higgsfield(힉스필드)** = 이미지 SaaS 보조.
- **Claude Code 터미널 스킬** (8주차 학생 특전) — 네이버 데이터랩 + 검색광고 API + 메타 라이브러리 자동 다운 + 자사몰 스크래핑 → 종합 보고서 자동.
- 강사 측 자체 **E-commerce AI 솔루션** (크롬 확장) — 스마트스토어·쿠팡 리뷰 1만개 자동 수집.

## 강사 정립 메소드 (사전 VOD 외 — 본강의에서 추가 발견)

### 10. 키워드 확장 마인드맵 3단계

효과 → 의학 → 시술/질병/암 → 검색량 1위 픽. 누룩밤 기미수면팩 (크림→수면팩 프레임), 달바 미스트세럼, 달링태그 앰플 (검색량 2천→2만 폭증·역산 월매출 10억대) 사례. **퍼포먼스 폭발 SKU 지속 = 3~6개월·연 100억 패턴.**

### 11. 회색지대 채우기 (숫자 만들기 노하우)

규제·임상 자료 부족 시 합법 숫자 만드는 패턴:
- 서베이 20명 만족도 95%
- 4년 = 1400일
- 3개 런칭 = 3개 완판
- 어머니 200ml 주름크림 (런칭 직후 월 3억 — 본인 사례)

### 12. 권위 분류 — 표면 vs 공감

| 종류 | 정의 | 사례 |
|---|---|---|
| 표면 권위 | 학력·자격 | 석사·박사·강남 피부과 |
| 공감 권위 | 같은 입장의 진짜 사람 | 진짜 엄마·트러블 전문 |

### 13. 니즈 점수 × 반복 문제 점수 매트릭스 (강사 핵심 사고 도구)

> 컨셉 잡을 때 가장 자주 쓰는 2축 매트릭스. 양축 만점 = 다이어트.

- 양축 만점 → 다이어트
- 고가 정책 → 발기부전·치매·탈모
- 노인 액취증·발톱 앰플 등 시뮬 사례 풍부

### 14. 이커머스 4가지 기초체력

1. 제품기획
2. 운영발주
3. 자금
4. AI

→ 이 4축이 안 되면 광고비 부어도 매출 안 남.

### 15. 무의식·인식·근거 3층

- **무의식** (선천적 vs 학습된) — "딱 봐도 의사상" 같은 학습된 통념
- **인식** — 표면 키워드
- **근거** — 숫자·임상

3층이 일관되게 쌓여야 매출.

### 16. 가격 = 시장조사 X, 명분으로 결정

> "**가격은 시장조사로 정하지 마라.** 명분으로 결정."

- 베리홉 비타민C 앰플: 원가 6%, 15ml에 4만 9800원, 월 40억
- 비싸게 런칭 후 할인 룰
- 시즌 발주 2부류 사장 분류

### 17. 가설→검증→결과 사이클 + 분석 임계점

- **분석은 매출 500억부터 의미.** 그 이하는 가설 검증으로 충분.
- CPM 비딩 카테고리별 시세: 화장품 3-8만 / 생활용품 1-2만 / 구두 7천-1만

### 18. KPI 5종 + 역산 매트릭스 (부록 2 — 메소드 운용 본체)

> 객단가 → CPA → CPC 역산. 통제 가능 우선순위 = 객단가 먼저.

| KPI | 정의 |
|---|---|
| 한계비용 | 매출 손익분기 광고비 |
| 목표비용 | 영업이익 목표 광고비 |
| CPA | 1구매당 광고비 |
| CPC | 1클릭당 광고비 |
| BEP ROAS | 손익분기 ROAS |

> 강사 매트릭스 예: 객단가 8만 + ROAS 200% → CPA 4만 → CPC 800.

### 19. CBO · ABO · DCO 운용 룰 + "의도 적어 보내"

- **ABO 초기 실험 → 고성과 소재 → CBO 재투입** → 비율 70:30 증액.
- 광고 세팅 시 "의도 적어 보내" (직원 가이드 — 모든 세팅에 의도가 명시돼야 디벨롭 가능).
- **메타 빈도 = 2 = 최적**. 별도 리타겟팅 거의 불필요 (머신러닝 자동 포함).
- 직접 전환만 봐도 상위 10%.

### 20. 매체별 머신러닝 매트릭스

| 매체 | 머신러닝 | 운용 핵심 |
|---|---|---|
| **메타** | 최강 | 자동 맡김 |
| **구글** | 가장 느림 | 맞춤타겟·미디어믹스 적극 |
| **GFA** | 반쪽 | CPC 우선·4050 핵심 |

### 21. BEP ROAS 룰 (자사몰 → 유통 낙수)

자사몰 BEP에 맞춰 운영하면 유통채널 낙수로 영업이익 무조건 발생 (광고비 20~100%). **자사몰 영업이익 0이지만 쿠팡·유통 포함 25~30%** (플리오 사례).

### 22. 시너지 상품 vs 리셀러 회피

- **GWP** (Gift With Purchase) — 앰플 2 사면 광고 X 크림 1.
- 1인당 구매 제한.
- 장기 사용 암시 세트 (리셀러·2+2 차단).
- **SKU 1개 운영의 한계 = 4개월 후 리셀러 침투** → 데이/나이트·앰플 추가로 방어.

### 23. 매출 임계 (SKU 수)

- SKU 1~2개 = 200억
- SKU 4개 = 300~400억
- 달바 천억~2천억 분기점에 브랜딩+유통 전환

### 24. 매체 확장 순서

**메타 → GFA → 유튜브.** DA 광고 유통기한 약 1년.

- **마이크로 인플루언서 ROAS 1263%** (시골 60대 브이로그 9400명 50만원 → 매출 630만원).
- **CPV 40~50원 이하면 GO**.
- 자폐아 육아맘·시골 브이로그 같은 니치 채널 중시.
- MCN 통하면 단가 2배.

### 25. 바이럴 메소드

#### 채널 우선순위

> **블로그 ≥ 카페 > 인플루언서 ≥ 유튜브 > 커뮤니티**

#### 네이버 섹션 순서 = 키워드별 고객 관심도

| 키워드 | 1순위 섹션 |
|---|---|
| 기미 | 의사답변 |
| 기미크림 | 쇼핑 |
| 얼굴하얘지는법 | 블로그 |

→ 키워드별로 어디에 자료를 깔지 결정 (광고 카피 성분키워드 반복으로 검색량 자연 증가).

#### 내돈내산 키워드 선점

경쟁사 침투 방어. 누룩밤 1등이 경쟁사 카페 작업글 → 상표권 주장 불가.

#### 내적 보상 4법칙 (바이럴 콘텐츠 = 선동)

1. **계기**
2. **실용성**
3. 대중성
4. 좋아하는 사람

→ 계기 × 실용성이 매출 1위. **칼마디 발기력 바이럴 → 하루 5천만원** (강사 본인 운영).

#### 채널별 성향

- 트위터 = 10·20대 (모공·여드름)
- 쓰레드 = 자영업자
- 인스타 = 50·60대 릴스
- 유튜브 = 4050+

#### 의료기기·건기식 viral 우회

3자 카드뉴스 2등 끼우기. 검색결과 채우기 = 경쟁사만큼만.

#### 5단 설득

### 32. 한 페이지 6요소 적층 (8주차)

### 33. 충동형(퍼포먼스) vs 계획형(나이키)

- 충동형: 다이어트·주름·트러블 — 즉구매. 광고 의존.
- 계획형: 스포츠·취미 — 검색·비교. CRM 별개 단계.
- → CRM은 충동형엔 어울리지 않음.

### 34. B급 연예인 = 가성비 모델

- 한가인·박한별 2~3천만원
