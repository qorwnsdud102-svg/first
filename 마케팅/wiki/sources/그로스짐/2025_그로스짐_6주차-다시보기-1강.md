# 6주차 다시보기 1강 — 유튜브 바이럴 + 마이크로 인플루언서 ROAS 1263%

## 출처

- 원본: `마케팅/raw/그로스짐/30. 6주차 다시보기 1강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 6주차 (유튜브 바이럴 본격)

## 한 줄 요지

유튜브 광고비 월 3억~5억이 효율 피크. 개인 시딩은 **메가급(3천만+) 비추, 마이크로(구독자 1만 미만) 추천**. 시골 60대 브이로그 9400명 채널 50만원 PPL → **매출 630만원·ROAS 1263%**.

## 핵심 골자

### 환절기·홍조 화장품 피드백 (이은 대표 과제 사례)

- 협찬 표시 없는 내돈내산 X → 5주차 강조 그대로.
- 비포·애프터 사진 **상단 배치**가 검색 클릭 후 이탈 방지 (검색자는 글 안 읽음).
- "환절기·갱년기 얼굴 붉은기" → **퍼포먼스 자체가 어려움**. 보습 좋은 로션만 잘 발라도 잡힘. **고통지수 낮음**.
- 답: 진정 → 피부결·요철·주름 등 **해결 불가능 + 나이 들수록 심해지는** 영역 (고통지수 8.9급)으로 비틀어라.

### 측정 가능한 매체 = 측정 가능한 마케팅 (새 디테일)

- TV 광고 = 누가 몇 초 봤는지 모름 → 측정 X.
- 유튜브 = 영상 100% / 80% / 60% 조회·이탈 지점까지 측정 O.
- 매체 확장 순서: **메타 → GFA → 유튜브** (예산 규모 따라).
- 유튜브 효율 피크 = **월 3억~5억** (3천만~1억도 ROAS 나옴).

### 유튜브 4가지 마케팅

1. **자체 채널** — 강사 회사 [[CCFM]]이 운영. 조회수 적어도 마케팅 문의 인입.
2. **PPL** — 영상 사이 1분 등장 가장 효율. 30~50만, 비싸도 100만.
3. **댓글 상위노출** — 크몽. 좋아요 1000개 = 10만원. 메인키워드 3·4개만 작업해도 효과.
4. **파워채널** — "유용한 제품 5가지" 식. 살림남·실험남 익명형. 인스타·틱톡 동일 패턴.

### 마이크로 인플루언서 = 강사 핵심 인사이트 (새 메소드)

- 본래 공식: 메가→중상위→마이크로 하향 흐름.
- 강사: **마이크로부터 → 중상위로 상향**. 메가 한 번도 안 함.
- 단가 50만원 9400명 시골 브이로그 60대 = **매출 630만원, ROAS 1263%**.
- 평균 조회 2~3만 = 구독자 대비 조회 비율로 판단.
- 중상위 500만원 1건 = 매출 50만원 (실패 사례).
- 측정지표 = **CPV (조회당 비용)**. CPV 40~50원 이하면 GO.
- 단, CPV 무조건 기준 X — 구독자 대비 조회수가 더 중요.

### 자폐아 육아맘·시골 브이로그 = 니치 카테고리

- 팬덤 강력. 고통지수 높은 제품에 매칭 시 폭발.
- 4050 여성 + 주름·기미 화장품 매칭 ↔ 60대 시골 아주머니 채널.

### 도구 — Claude Chrome + Gemini 딥리서치

- 유튜브 영상 30개 / 쇼츠 검색 → 좋아요·댓글 패턴 추출.
- 영상 사이 PPL 우리 제품으로 만들지 말고 **일상 영상 사이에 2~3초 컷**.
