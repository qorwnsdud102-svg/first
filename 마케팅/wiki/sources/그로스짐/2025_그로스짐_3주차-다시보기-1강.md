# 3주차 다시보기 1강 — 자사몰 리뷰의 중요성·구조 (사회적 증거 본격편)

## 출처

- 원본: `마케팅/raw/그로스짐/20. 3주차 다시보기 1강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 3주차 (리뷰 편 1강)

## 한 줄 요지

광고와 **자사몰 리뷰의 메시지 일치**가 전환율의 핵심. 리뷰는 사회적 증거 + 숫자적 근거 + 동기부여 + 밴드웨건의 4축으로 작동하며, 상세페이지보다 더 강하다.

## 핵심 골자

### 캔버스 과제 피드백 — "단어만 적지 말고 설명·맥락까지"

- 페르소나 무의식 정의에 "예쁜 여성" "섹시한 여성" 정도만 적는 건 부족.
- **포즈·얼굴 형태·스타일·직업까지** 디테일하게. 디자이너가 시각화 가능해야 함.
- 무의식 = 트리거 (생각 안 했는데 떠오르게 하는 단어). 인식 = 트리거가 만들어낸 욕구·니즈 ("우리 딸 여드름 없어지고 자신감 얻은 이야기").
- 근거 6축: **숫자적 근거 · 사회적 근거 · 사회적 통념 · 경쟁력 · 컨셉 · 기능적 특화 포지셔닝**. 판매자 단어 그대로 쓰지 말고 **고객 일상어로 치환** ("99.9% 항균" → "세균 싹 사라집니다").
- "골프 안 치는 사람한테 100돌이·18홀 단어 쓰지 마라" — 페르소나 전용 어휘만 쓰기.

### 리뷰 = 결정적 구매 트리거

- 사전 VOD의 4가지 사회적 증거(통념·근거·권위·숫자)를 **리뷰 단일 자산에서 동시 발생**시킴.
- 동기부여 = "나도/우리 딸도 저렇게 될 수 있지 않을까" + 매슬로 안전·가족 욕구.
- **밴드웨건 효과**: 줄 서 있는 맛집 효과의 온라인판. 최신 리뷰·최신 Q&A 알람 위젯이 줄서기 시각화 대체.

### 미디어 커머스 구매 패턴 (강사 정의 5단계)

광고 노출 → 랜딩 유입 → 리뷰 확인 → 구매 결정 → 완료.

- 자사몰 리뷰만 보고 당일 전환: **2~3%** (방문자 100명당 2~3명).
- 나머지 97~98%는 **하루~3일 이탈** (유튜브·카페·지인 검증 후 복귀). 230% 정도가 2~3일까지 늘어남.
- 메타 광고 단독으로 자사몰 전환율 7~10%까지 가능.

### 수치 증명 — "리뷰 한 끗에 전환율 4배"

- **잡화(변기 청소제) 사례**: 매출 하루 1천만 원 도달 → 신규 부정 리뷰가 상단으로 밀려옴 → 전환율 **10% → 2.5%로 4배 폭락**.
- 메타 광고 단독으로 자사몰 전환 7~10% 갈 수 있는 환경에서, 리뷰 상단 한 줄 차이로 전체가 무너짐.
- 사회적 통념상 좋은 리뷰만 있으면 의심받음 → **1~2점 리뷰도 일부러 작업** ("제품은 좋은데 배송이 늦었다" 식의 판매자 외 사유로).

### 매일 아침 리뷰 점검 루틴

- 출근 10시 → 10시 30분까지 자사몰 + 유통 채널(스마트 스토어·쿠팡) 리뷰 점검.
- 직원에게 "광고 메시지 방향성 맞춰 상단 리뷰 재설정" 명령. 어떤 수단·방법으로든 우리 의도 리뷰가 상단에 깔리게.
- 부정 리뷰는 패턴화 누적 → 1년 치 분석하면 **다음 시즌 제품·매출 2~3배 힌트**.

### 광고-리뷰 메시지 일치 원칙

> "통제할 수 있는 요소는 무조건 통제해야 한다."

- 광고 카피와 자사몰 상단 리뷰가 **같은 단어·같은 페르소나 경험**이어야 전환됨.
- 광고: "팔자주름 5번 발랐는데 단종이라 큰일" (호소문) → 리뷰: "팔자주름 인상 어두웠는데 5번 바르고 10년 젊어 보인다는 소리, 단종이라니 더 샀어야" (좋아요 108).

### 2천억대 대표들의 공통점

- 매일 부정 리뷰 따로 filtering → 보완점 추출.
- **누적 매출 1% 충성 고객 FGI** (전화 인터뷰): "어떤 요소로 우리 화장품을 선택했는지 → 숫자/사회적 근거 중 어떤 게 먹히는지" 추출 → 그대로 광고 소재로.

### 화자 4종 중 "제3자 추천"의 법적 우회 사례

- 초록입홍합(관절·당뇨) 같은 **법적 위반 가능 효능** = 상페에 직접 못 씀.
- TV 방송 "몸신" 같은 프로그램이 효능 언급 → 검색 → 상페는 그냥 성분 설명만 있어도 매출.
- 리뷰가 같은 역할 = 고객 화자의 간접 효능 추천.

### 데이터랩 → 블랙키위 경쟁사 발굴 워크플로우 (실시간 시연)

1. naver.com → 데이터랩 → 쇼핑 인사이트 → 카테고리(화장품/스킨케어/크림) → 최근 1개월 인기 검색어.
2. 유명 브랜드 제외, 처음 보는 브랜드 클릭 (예: **세인트 프랑 링클 크림** — 9월부터 검색 시작 → 12월 재개 → 2월 2.2만 회로 **3개월 10배 성장**).
3. 자사몰 리뷰 적은데 검색량 폭증 = 홈쇼핑·연예인 협찬 플레이 (퍼포먼스 X).
4. 퍼포먼스 사례를 찾아야 → 페이스북 픽셀 헬퍼로 픽셀 설치 확인.
