# 8주차 다시보기 2강 — 헤어리듬 루트너 케이스 분해 + 1:1 실습 피드백

## 출처

- 원본: `마케팅/raw/그로스짐/36. 8주차 다시보기 2강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 8주차 (실시간 분해 + 학생 4명 개별 피드백)

## 한 줄 요지

급상승 탈모 케어 [[헤어리듬 루트너]] 사이트를 **광고 → 모바일 랜딩 → 리뷰 → 상페**까지 풀스택 분해. 모든 요소가 **다 의도된 것** — 누끼 사진 X, 모델 비율 일치, 카피·이미지 동기, 리뷰 스토리·연령 매칭. 광고 카피와 상페 카피가 **단어 단위로 일치**.

## 핵심 골자

### 헤어리듬 루트너 케이스 (8주차 핵심 분해 사례)

급상승 경쟁사 발굴 결과. 두피 앰플 + 알사브록신 함유 두피 브러시. 광고 → 자사몰 모바일 분해.

**광고 소재 (메타) 카피 흐름**:
- "얼굴선이 왜 이렇게 처지지? 알고 보니 가르마 넓어지면 얼굴도 처져 보일 수 있대요"
- → 두피 말기 시작 → 루트너 두피 앰플 + 알사브록신 + 톡톡 뿌리고 브러시
- → "1분 루틴" / before-after / "얼굴 lifting 느낌" / "출산 후 머리숱 고민" 페르소나

**광고 → 상세페이지 동기 (단어·비율 일치)**:
- 광고 멘트 = 상페 인트로 카피 = **단어 단위 동일**.
- 광고 모델 = 상페 모델과 같은 **비율·구도**로 합성·연출.
- "두피 관리만 했을 뿐인데 얼굴선까지" → 상페 인트로 "빠지는 머리카락, 무너지는 얼굴 동시에 케어" — 같은 **하나의 프레임**.

**썸네일 = "절대 누끼만 두지 마라" 명제 (강사 핵심 피드백)**:
- 1등 브랜드가 누끼만 깐다고 따라하지 말 것. **"1등이 바보면 다 바보 된다"**.
- 헤어리듬은 누끼 사진 + **연출 컷** + 모델 합성 등 다채로운 USP 표현 컷 동시 배치.
- 한 학생 (대표) 피드백: "썸네일에 누끼만 있는데 왜 그러셨어요? 아기 제품이면 아기 합성 / USP 강조 / 하이라이트 디테일 등 의도된 컷이어야".

**모바일 랜딩 진입 직후 보이는 것 (의도된 시퀀스)**:
1. 할인율 + 수치 (한정 강조)
2. 리뷰 개수·점수 (4.6, 별점 일부러 세팅 가능성)
3. **펼쳐진 포토리뷰** (더보기 누르지 않게)
4. "지난달 고등학교 동창 모임에서 친구가 머리숱 빡빡해져서 비결 물어보니 이 제품 박스째…" — **광고 페르소나(출산 후 여성)** + **타겟 연령 (40대 남성·30대 여성 등)** 매칭된 리뷰 의도 배치.

**숫자 카운트 위젯 = 회색 지대**:
- "현재 N명 보고 있습니다" / 새로고침마다 +N — **세팅 스크립트**로 자동 증가 가능.
- 홈쇼핑은 팩트 의무 / 온라인은 규제 회색 지대 (네이버에 세팅하면 실제 표시).
- "사기 vs 포모" 관점 차이. 자사몰 가능, 숫자 크게 키워도 OK.

**상세페이지 본문 = 문제 제기 → 해결 → 근거 무한 반복**:
- "MFDS 기능 보고서 완료" — **소비자 모름** / 위에 "탈락수·혈행" — **소비자 알고 통념 적용** = 통념 키워드 + 미상 키워드 결합.
- "스위스 모낭 활력 성분 함유" — 스위스 = 사대주의(통념), 성분명 = 권위 위장.
- 임상·안정성·두피 탄력 개선·리프팅 증거 등 한 페이지에 **문제·해결·근거·통념·권위·숫자** 6요소 다중 적층.
- 사회적 권위 모델로 **티아라 지연 + 이시아** 동시 사용 — **콘텐츠 라인을 30대·50대 두 라인으로 분기** (USP는 동일하지만 페르소나 분기).

### "상세페이지 = 시작점이 아니라 광고 라이브러리 링크 = 시작점"

- 자사몰 검색 도착 페이지 ≠ 광고 도착 페이지. **메타 라이브러리 → "더 알아보기" 클릭으로 본 진입 페이지** 확인.
- 카페24는 링크 값으로 시작 위치 조정 가능. 강사 테스트: 다양한 시작 지점 → **상단(=썸네일+가격) 시작이 가장 좋음**.

### 제3 화자(권위) 섭외 = 수단·방법 가리지 마라

- 의사 직접 섭외 어려우면: **"의사가 만들었다"가 아니라 "박사·교수 자문"** + 회의록만 있어도 OK.
- 비뇨기과/학문외과 등 카테고리 비매칭 의사도 가능. 식약처 통과 후 신고 들어오면 그때 "박사·교수 자문"으로 표기 변경.
- "엄마·아빠·내 자신"도 권위. **마동석으로 합성한 "대표"** 사례 — 진짜 마동석 섭외 전까지 합성 활용 가능.
- 다이어트 식품 = "기초대사량 문제" 카피 = **"풍이"라는 옛 브랜드와 동일** — 컨셉만 살짝 변형하면 먹힘.

### Claude for Chrome 활용 (실시간 분해 시연)

- 클로드 포 크롬 켠 채 경쟁사 사이트 진입 → "위치, 멘트, 메인카피, 메인 이미지 느낌 + 전체 흐름" 자동 분석 명령.
- 예전: 사람이 직접 캡처·메모장 정리·회의 → 지금: AI가 다.
- "편해진 만큼 다른 판매자도 똑같이 편해짐 → 안 하면 뒤처짐" (강사 위협 메시지).

### 학생 1:1 실습 피드백 (4명)

| 학생 | 카테고리 | 강사 조언 |
|---|---|---|
| 관절 크림 (메디터 부스터 프리·아닌비체) | 0 → 8천 검색량 | 객단가 비싸게 가능. 두 개 정리 + 퍼포먼스 캔버스 매칭. 빈 부분 = 돈 써서라도 채움 |
| 파라다이스 그램·지중해 오일맘 클렌즈 (다이어트) | 0 → 7600 | 페르소나·근거 매칭. 식품 권위 = 비뇨기과·박사·자문 회의록 |
| 헤어 스칼프 | 두 카테고리 묶음 | 권위 어디서 가져올지 — 약사·약국이라도 / 활력 연예인 (PPL·B급) 작은 예산 활용 |
| 아기 옷 세제 (이연방지·비비드) | 시장 작음 | "이연 방지" 자체 카피 X. **인접 카테고리(아기 크림·아기 세제·30대 후반 화장품)**의 급상승 사례 참고 |

### 히든 페이지 운영

- 같은 상품에 상페 다르게 → 한 버전은 메이저 5점만 공개, 나머지 비공개로 **A/B 테스트**.
- 광고주 운영에서 일반화된 기법.

### 카피 침해 위험과 활용 (해외 제품 동업 사례)

- 동업자 = 옛날 성인용품 (애널 용품) 1등 카피 → 색상·모양 살짝 변형 → 고소 들어왔으나 **불성립**.
- PSD 원본·일러스트 파일 있으면 **창작물 인정**.
- "감당할 만한 리스크. 하라/말라 결정 안 해줌" = 강사의 책임 회피 + 실용주의.
