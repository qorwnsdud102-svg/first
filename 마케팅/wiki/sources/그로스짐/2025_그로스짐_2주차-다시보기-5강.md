## 한 줄 요지

광고 운영을 데이터 분석으로 풀려 하지 말고, **가설→검증→결과→재가설** 단순 사이클로 돌려라. 패턴 분석 = 빅데이터 영역인데 자사몰은 빅데이터 X. 매출 500억 이전엔 분석 무의미.

## 핵심 골자

### 인식 카피 = 무의식의 완성

- 사례: 고등학생 아들 + 세련된 엄마 사진 → 카피 "엄만데 누나라고 해요" or 더 자극: **"아들 졸업식 갔는데 담임 선생님이 번호 물어봐요"** + 서브카피 "딱 7일 발라보세요. 100% 환불"
- 인식 카피의 본질 = **고객이 꿈꾸는 비전과 가까울수록 강력**.
- 고객이 추천하듯 들리는 톤 (셀러가 외치는 X).
- 반전 카피 = 임팩트. (유머 게시판식 — 평범 그림 + 웃긴 카피)

### 오이 수딩젤 사례 (의도하지 않은 바이럴)

- 올리브영 출시 → 이미지 1장으로 여성 커뮤니티 바이럴 → 품절 → 단종.
- 제작자가 의도한 게 아니라 **"리셀해 달라" 댓글 폭주의 다른 의미**의 바이럴 (= 안 좋은 쪽이지만 결과적 매진)

### 100% 환불 카피

- 그냥 "100% 환불" X. **구체화** → "어디 팔자 주름이 며칠 이내에 변화 없을 시 환불" — 구체성이 광고 소재 자체가 됨.

### 가설→검증→결과 사이클 (강사의 핵심 운영 사이클)

```
가설 → 검증 → 결과 → 참/거짓 판단 → 가설 재수립
```

- 결과 = 데이터지만 자사몰 데이터는 통제 변수가 너무 많아 **패턴 분석 불가능**.
- 빅데이터 정의: 반복적 행위 + 패턴이 있는 구조 (예: 서울 버스 노선 늦은 시간대 분석 = 같은 지역민 반복 사용 → 진짜 빅데이터).
- 자사몰 = 들어오는 사람 매번 다름 + 재구매율 10% 미만 + 30대도 다 다른 생각.
- "**저는 이커머스 천억·2천억·3천억 한 분 중 패턴 분석으로 매출 낸 사람 한 명도 못 봤다**" — 강사 단언.

### 매출 500억 이전 = 분석 X, 가설 검증 빨리 돌려라

3가지 통제 가능 영역만 빠르게 갈아끼움:
1. **객단가** ↑ (셀링: 제품 새 개발·세트·옵션·할인율 — 매일 못 바꿈)
2. **방문자 수** ↑ (광고비 + 타깃 + 싸게)
3. **전환율** ↑ (통제 어려움)

→ **방문자 수 늘리기**가 매일 갈아끼울 수 있음.

### CPM 비딩 메커니즘 (광고 매체 본질)

- 모든 광고 매체 (메타·네이버·구글) = **경쟁 입찰**.
- 카테고리별 CPM:
  - 화장품: 3만 ~ 8만원
  - 생활용품: 1만 ~ 2만원
  - 구두: 7천 ~ 1만원
- CPM 비싼 이유 = 광고하려는 자 多 + 보여줄 유저 한정.
- **국가 바꾸면 CPM 2-3배 싸짐**: 일본(인구 1.2억) · 미국 (한국 5천만 대비).
- **품질점수**: 비딩 가격 외에 클릭률·체류시간 등으로 매체가 광고 품질 평가. CPM 낮춰도 됨.
- → **CTR을 2배로 높이는 것 = 콘텐츠 기획 = 퍼포먼스 캔버스의 본질**.

### 광고 매체 선택

- 메타 (페북·인스타) > 유튜브 (소액 광고 시).
- 유튜브 = 광고비 많이 들어가야 머신러닝 작동, 안정성 차이.
- 인스타 유저 = 보합·하락. 유튜브 = 70대도 트로트 보러 들어옴 (강사 아버지 사례).
- 본질: "어떤 매체가 좋냐"는 **무의미한 질문**. 어느 매체에서 내 제품 어필되는 집단이 많은지 + 머신러닝 친화도 보고 결정.

### 분석 매출 임계점 = 500억

- 강사가 광고대행한 단일 브랜드 국내 최대 = **500-600억** (화장품). 그 이상은 해외로.
- 500억 이전 사장은 분석 X — 가설 검증으로 1~5번 반복:
  1. 객단가 ↑
  2. 방문자 수 ↑
  3. 전환율 ↑
  4. **권위 찾기** (연예인 협찬·병원 입점)
  5. **바이럴 챙기기** (옹호·추천 증빙 쌓기)

### 권위 = B급 연예인이 답

- IU·핫한 연예인 X. **2~3천만원짜리 B급 (옛날 S급)** = 진짜 효율.
  - 한가인 (활동 안 했을 때) - 40~50대에겐 여전히 파스타. 그 제품 핵심 타깃 = 40~50대였음.
  - 박한별류 옛날 S급.
- "S급이었는데 지금 B급" = 가성비.

### 도마 10억 사례

- 한 광고주가 자기 엄마 친구를 "주방용품 전문가, 주부 27년"으로 등장시켜 도마 광고. **10억 매출**. (= 권위는 잘 만들면 가짜도 OK)

### 사이트 리뉴얼 = 브랜딩 전문가 X

- 사이트 리뉴얼은 브랜딩 전문가 X. **고객들이 반복적으로 좋아하는 거 (= 성과 좋은 배너의 카피·색감·톤무드)**를 패턴화해서 반영.
- 예: 폴리오 = 1년차부터 **보라색만** 일관 사용. 광고비 30~40억/월 = 노출 7~8억 회/월 → 보라색 = 폴리오로 인식 학습됨. (광고비 많이 쓰는 단계의 전략)

### 스몰 데이터 - 마틴 린스트롬 책

> 강사 공식 추천 도서: **"스몰 데이터" - 마틴 린스트롬**.
> 
> 빅데이터 X. 고객 관심사·일상 디테일을 보라는 것. 강사 본인이 착한구두 광고 맡고 성과 안 나서 매일 경쟁사 리뷰 1000개씩 읽어 키워드(통굽 45도·슬링백·스퀘어 뮬·뾰족 하이힐·통굽 cm) 다 학습한 경험에서 나온 책.

### 강사 시행착오 1순위 (2020년)

- 만들었던 광고를 또 만들어 실험 → **기록 안 남겨서 중복 작업**. 광고주가 "왜 지난번에 실패한 거 또 만드세요" 피드백.
- → 빨리 피봇 + 기록 남기기.
