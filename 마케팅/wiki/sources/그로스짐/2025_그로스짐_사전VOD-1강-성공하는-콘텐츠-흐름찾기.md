## 한 줄 요지

매출의 본질 = 제품이고 설득의 본질 = 콘텐츠. 성공 콘텐츠는 **후킹 → 스토리텔링 → 설득(4가지 사회적 증거) → 프로모션·포모 → 행동요구**의 5요소를 갖는다.

## 핵심 골자

### 성공 콘텐츠 5요소

| 요소 | 정의 |
|---|---|
| 후킹 | 콘텐츠를 볼 이유. **경쟁사 + 오가닉 콘텐츠 100개 캡처 → 공통 키워드 추출** → 제품에 녹이기 |
| 스토리텔링 | 고객 공감 (대상에 따라 다름) |
| 설득 | 4가지 사회적 증거 (통념·근거·권위·숫자) |
| 프로모션 / 포모 | 할인·임박감. 광고 콘텐츠 X, **랜딩**에서 (네이티브 보호) |
| 행동요구 (CTA) | 셀링 광고 = 강하게, 네이티브 광고 = 약하게 |

### 설득의 4가지 사회적 증거 (강사의 핵심 프레임)

- **사회적 통념** — 보편 상식 (피부과·약국 추천)
- **사회적 근거** — 많은 사람이 쓴다 (몇 명 샀다·줄 서 있다·리뷰 개수)
- **사회적 권위** — 명사·전문가 (강남 유명 피부과·인플루언서·박사)
- **숫자적 증거** — 객관 (00% 임상·N위·만 개 판매)

### 후킹 케이스 — 뷰티 디바이스 광고

- 메타 CPC 5천원 / CTR 1.5% 시작.
- 100개 캡처 분석 → 공통점: 여성·중안부 짧은 얼굴(영포티·장원영·아이유)·볼터치.
- AI 모델 (나노바나나·시드림) 30명 테스트.
- → CTR 4% (2.5~3배), CPC 절반 이하.
- **데이터는 가설 검증용**. 행동 이유 추론 도구가 아니라 가설 채택/탈락의 기준점.

### 콘텐츠 수집의 50% 룰

- "경쟁사만 보면 다 똑같이 만든다."
- 50% 이상은 **오가닉 콘텐츠** (조회수 많은 비광고 게시물). 사장님 직원 사례: 10·20·30·40·50대 계정 분리 학습 → 알고리즘별 콘텐츠 차이 관찰.

### 성공 콘텐츠 5 공통 흐름

1. 고통이 생긴 이유를 이야기 (공감)
2. 동화 같은 흐름 (초등학생도 이해 — 의학적 영어·복잡 메뉴 X)
3. **하나의 핵심 관통** (여드름 화장품에 비타민C·보습 다 넣지 말고 여드름만)
4. 명확한 키워드 (회춘 = 50·60대 OK, 20대에는 X — **연령별 다름**)
5. 마음속 최상급 키워드 (여드름 → 피부과·약국)

### 한끗 차이 = 전환율 0.5% 상승

- 약한 키워드로 스마트스토어 등록 → 리뷰 10개 → 1위 → "네이버 1위" 인용 → 신뢰.
- 카피 작은 한끗 = 전환율 0.5%씩 누적.
