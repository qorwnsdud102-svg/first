## 한 줄 요지

수강생 질문(폴리오 안마기 어떻게 들어갔나) → 폴리오 캠페인 회고로 이어지면서 강사가 **이커머스 4가지 기초체력**과 광고 콘텐츠의 **무의식·인식·근거 3층 구조**를 정립.

## 핵심 골자

### 폴리오 (Polio) 안마기 사례 — 강사 본인 광고

- 광고주 = 안마기 회사. 강사가 **2021년 매출 0일 때부터** 광고 시작 → 500억까지 함.
- 핵심 메시지: "다리 뭉친 거 쉽게 푼다" 
- 다리 뭉침 → 여자 키워드 파생: 무다리·종아리알·다리 붓기 발견 (남자는 다리 뭉침 고민 X)
- 후킹 = "예쁜 다리 만들어 드립니다" (성적 욕구 = 표면 = 무의식)
- 후속 메시지 = 다리 순환·건강·근육 풀림 (안전 욕구 = 내면)
- 광고비 누적: 폴리오 자체 매출 70억 → 500 → 700 → 1300 → 2000억 후 하락. 한 달 광고비 30~40억 (강사사로만 15~16억).
- 폴리오 대표 = 이 전에 다른 브랜드로 100억 매출 경험. **이커머스 두 번째 브랜드**라 자금·운영 다 준비된 상태에서 빵 터짐. (= 첫 브랜드 대표는 따라하면 안 되는 사례)

### 이커머스 4가지 기초체력 (새 프레임)

> 이거 다 못 갖추면 **연매출 50억·월 5억에서 멈춤**.

1. **제품 기획 기초체력** (= 퍼포먼스 캔버스. 대표가 직접)
2. **운영·발주 시스템 기초체력** (3PL·CS 외주화 가능)
3. **자금 기초체력** — 매출 펌핑 시 정산 묶임 대비
4. **AI 기초체력** (강의 본강의 = 1+4)

이 중 1·4는 대표 직접, 2·3은 외주·표준화 가능.

### 매출 30억 빵 터졌는데 그 달 마이너스 사례

- 광고주(화장품) 평소 월 10억 → 유튜브 협찬+퍼포먼스 동시 터져 월 30억.
- 결과: 운영·발주 사고 → **그 달 마이너스**. (= 펌핑이 곧 이익 아님)

### 자금 기초체력 디테일

- **고이드 (Goyd)** = 매출 증빙 기반 단기 대출 서비스. 분기 매출만큼 빌려줌. 연 이자 ~10% (월로 환산하면 1~2억/30억 차입).
- 보증보험 정산 묶임: 갑자기 매출 7~8억 → 38억 늘어나면 보증보험이 의심 → **6주 묶음**. → 한도 늘려달라 협상 필요.
- 쿠팡 정산 = 70% 즉시, 30% 3개월 뒤.
- 마이너스 통장·선대출 만들어두기 필수.

### 가전·공산품의 가격 통제

- 강사 사례: 가전기기 광고주 연 매출 1500억 / 영업이익 150억 / **원가율 60%** → 통장에 돈 없음. 분기 매출 400억 중 240억 원가, 광고비 40억 → 300억 회전 묶임 → **흑자도산** 위험.

### 리드타임·예약구매

- 매출 펌핑 → 재고 떨어짐 → 6주 리드타임 → 광고 끄기 vs 예약구매로 광고 유지.
- **예약구매 취소율**: 2주 이내 = 거의 0% / 2주 초과 = 급증 (10~20%).

### 광고 데이터의 매출 공식

> 매출 = 방문자 수 × 전환율 × 객단가

통제 우선순위:
1. **객단가 ↑** (제일 쉬움 — 돈 안 쓰고 가능. 세트·GWP·고가정책)
2. **방문자 수 ↑** (돈 쓰면 됨)
3. **전환율 ↑** (제일 어려움 — 인간 변수 통제 불가)

### 광고 콘텐츠 표현의 4요소

> 온라인 콘텐츠는 4가지로만 표현됨: **글 / 이미지 / 영상 / 소리(효과음)**

이 중 영상·이미지·글이 핵심. 영상 = 메타에서 압도적 성과.

### 무의식·인식·근거 3층 (광고 콘텐츠 기획의 새 프레임)

> 사전 VOD 4강의 **5요소(후킹·스토리텔링·설득·포모·CTA)**와 다른 축. 콘텐츠 1편 안에 다 들어가는 구조.

1. **무의식** — 후킹 (생각 없이 클릭)
   - **선천적 무의식** = 본능 (예쁜 여자·잘생긴 남자·아기·음식·동물)
   - **학습된 무의식** = 교육·환경 학습 (예: 제니라는 인물·드럭스토어·약국 풍경·서울대 로고·마켓컬리 보라)
2. **인식** — 무의식을 완성. 반전 카피·임팩트 있는 메시지. ("어? 뭐야?" 임팩트)
3. **근거** — 사회적 권위·통념·숫자 (4사회증거)

### 학습된 무의식 사례

- **짭제니**: 제니 닮은 인플루언서 모델 → 뷰티 디바이스 광고 ROAS 200% 이상. 하루 광고비 1천만원 소진.
- **AI 1979년생**: 어린 얼굴+머리 스타일 중년 → "1979년생인데요" 카피 = CTR 폭발 (단, 전환은 약함).

### 광고 후킹의 본능 패턴

- 후킹 = 표면 욕구 (성적 어필) → 클릭만 유도
- 본 메시지 = 내면 욕구 (건강·안전·자기실현) 해결
- 다리 광고: "성적 매력 만들어줄게"가 아니라 "다리 부음으로 너 건강해져" — 부수적으로 예쁜 다리

### 운영비/원가 30% 룰

- 가전 마진: 생원가 20% + 판관비 10% (수수료·물류·세금·패킹) = **30% 안에 들어오면 OK**.
