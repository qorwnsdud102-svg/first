# CCFM 마케팅사업부 — 퍼포먼스 캔버스 워크시트

## 한 줄 요지

강사 제공 **퍼포먼스 캔버스 실습 템플릿** (xlsx). 사장님이 본인 SKU로 직접 채우는 워크시트. **변환 X — 사장님이 직접 엑셀로 열어 사용.**

## 활용 방식

1. 엑셀 열기
2. 사장님 SKU 1개 선택
3. 빈칸 채움: 본질적 존재이유 → 문제·해결 → 매슬로 욕구 → USP → Pain Point → 페르소나 5요소 → 사회적 증거 4축 (통념·근거·권위·숫자)
4. [[그로스짐]] entity §1 "퍼포먼스 캔버스" 참조하며 작성
5. 작성 후 광고 콘텐츠 5요소·메타 4캠페인·자사몰 5단계로 실행
