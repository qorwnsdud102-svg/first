# 8주차 다시보기 1강 — Claude Code 스킬 자동화·급상승 경쟁사 발굴·SKU 전략

## 한 줄 요지

마지막 주차 특전 = **Claude Code 터미널 스킬 (네이버 데이터랩 + 검색광고 API + 메타 라이브러리 자동 다운로드 + 자사몰 스크래핑까지 자동 보고서)**. 그리고 SKU 1개로는 4개월 후 **리셀러 침투 → 가격 무너짐**, 데이·나이트·앰플 등 SKU 확장으로 방어.

## 핵심 골자

### Claude Code 터미널 스킬 (강사 제작, 8주차 출석자 특전)

- 자동화 흐름: 네이버 데이터랩 급상승 키워드 수집 → 브랜드 키워드 검색량 파악 → 메타 광고 라이브러리 검색 → **경쟁사 광고 소재(영상 포함) 다운로드** → 클로드가 영상도 캡처해서 장면별 분석 → **종합 보고서**.
- 필요 API 키:
  1. 네이버 검색광고 센터 → SA API 사용관리 → 커스터머 ID + 액세스 라이선스 + 비밀키
  2. 네이버 디벨로퍼 센터 → 앱 등록 → 데이터랩 API 키
- 24시간 백그라운드 실행 가능 → 매주 시장조사 자동화.
- 클로드 코드 강의 별도 신설 예정 (선착순 결제, **100달러짜리 Claude 요금제 필수**).

### CLI 스킬·plugin 진화 (강사 본인의 6개월 성장 메모)

- 교안 작성 시점에는 Claude for Chrome 권장 → 8주차 시점 강사는 API + 웹스크래핑 + **Playwright** 사용 단계.
- 직접 사이트 접근하여 데이터 긁어 프로그램화. 비개발자도 가능.

### 급상승 경쟁사 = "0에서 폭등한 브랜드"만 의미 있음

- 단순히 1~2등 = 이미 잘 팔리는 브랜드 (참고 X). **"3개월 전 검색량 0 → 갑자기 수직 상승"**한 브랜드만 진짜 인사이트.
- 실습 카테고리 사례: 스킨케어/아이케어 3개월치 → 유세라블(강사 자체 브랜드, AHC와 어깨), 이노프팅 크림, 멜라논 크림, 파라다이스 그레인(다이어트 11만 검색량 폭등) 등.
- 카테고리에 급상승 없으면 **6개월·1년치**로 확대, 또는 **기전 비슷한 인접 카테고리** 확장 (크림 → 화장품 스킨케어 전체).

### 카피 침해 현실 (실전)

- "완전 똑같이"가 아니면 **법적 처벌 거의 불가**. 지적재산권 등록 안 되어 있으면 도덕적 기준만 작동.
- 강사 본인 카피 당한 경험: 키워드(예: "안심") + 공장·색상·스펙까지 따라온 사례. 하지만 본인 키워드가 강사가 만든 신조어 아니므로 처벌 X.
- → "**앞에서 잘 따라 해서 돈 많이 버는 사람이 사실상 승자**" (한국 사회 1순위 = 돈). 독창은 혁신가지만 돈 못 벌 수 있음.
- "월 매출 5억에서 끊어 두기" 전략: 폭등하면 카피·신고 타겟 → 의도적으로 5억에서 멈추는 운영.
- 강사 본인 사례: 신고 폭주 → 자극 표현 다 빼서 노멀해짐 → 누적 광고 노출·콘텐츠로 매출 유지 → 원가 절감 → **유통 협상력 (홈쇼핑·올리브영) = 명함**.

### 노출 효과 관점 — 올리브영의 의미

- 올리브영·홈쇼핑·카카오 톡딜 = "매출+이익이 아니라 **노출 확장**" 채널.
- 올리브영 매출은 안 남는데 계속 입점하는 이유 = **외국인·인플루언서가 "올리브영에 있는 화장품 = 좋은 화장품"으로 인식** → 그쪽으로 매출 확장.
- 폐쇄 보호몰 = 노출 X → 수수료 줄이고 이익 확보로 가야.

### 퍼포먼스 캔버스 = 나침반 (마지막 주 복습)

- 캔버스 = 광고 콘텐츠 + 바이럴 + **상세페이지** 모두의 일관 메시지 원천.
- 광고 "주름 펴짐" → 바이럴 "촉촉" 같은 mismatch가 전환율 추락의 가장 흔한 원인. 강사 직원도 반복 교육해도 놓침.
- "삼위일체 같은 말 = 같은 결" 원칙.

### 가격·구성 = SKU 전략 (실제 운영 회고)

- **GWP** (Gift With Purchase) 추가 사은품 — 따로 만드는 경우 / 본품 추가 증정 케이스.
- **1+1 (본품 추가) 지양 이유 (2가지)**:
  1. 리셀러 침투
  2. 가격 무너짐
- 강사 자체 사례: SKU 1개로 3개월 → 4개월째 **리셀러 침투** → 추가 SKU 만들어 떼어냄. 화장품에서 **앰플 추가 증정** + **데이·나이트** 두 버전 출시 = SKU 풍성 + 가격 메리트.
- 초반 SKU 부족 시: "3개월치 사면 최대 할인" 식의 **수량 묶음**. 2+1·1+1 회피.

### USP·차별점 = "최초·최고·유일" 표현

- 경쟁사 직접 언급 = 법적 문제. → "**세계 최초**" / "**유일하게 가능**" 형태로 우회.
- 신뢰 = 정량화. 사람들이 의심 단계에서 검증하는 게 정량 수치.

### 포모 vs 신뢰 — 학생 질문에 대한 답

- 학생: "광고 자주 뜨는 브랜드는 신뢰 무너져 안 사게 됨".
- 강사: 그건 **계획형 구매(나이키류)** 관점. **퍼포먼스 광고는 충동형·신규 획득**이 전부. 재구매 가정 X. 만족 디벨롭은 CRM 단계에서 분리.
- 충동 구매 → 만족 → 그때부터 **고객 경험** 시작. CRM = Customer Relationship Management로 분리 운영.
- 기능성 화장품 만족 기준 = **"부작용 없음"** (실제로 기미 사라지는 화장품 = 의약품 분류). 카테고리별 다름.
- 플리오 사례: V1 → V2 → V3로 1년 6개월간 진화. **"종아리 마사지 = 시원함"이라는 본질**에 집중 → 압 강도·온열·릴렉스 강화 (냉찜·온찜 통념 활용).
