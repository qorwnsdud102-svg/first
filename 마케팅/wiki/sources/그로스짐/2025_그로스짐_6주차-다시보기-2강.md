# 6주차 다시보기 2강 — 유관기 + 마이크로 인플루언서 + 커뮤니티 바이럴 기획서

## 출처

- 원본: `마케팅/raw/그로스짐/31. 6주차 다시보기 2강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 6주차 (유튜브 + 커뮤니티 바이럴 마무리)

## 한 줄 요지

유관기 = 유튜브 시청자 연령·성별·CPV 분석 SaaS. **MCN 거치면 단가 2배** (300→150·100만원). 커뮤니티 바이럴 = 클로드에 베스트글 학습시켜 말투 자동 추출. 바이럴 기획서 = **판매자 관점 X, 소비자 공감 관점**.

## 핵심 골자

### 유관기 (강사 회사 사용 도구 — 새 디테일)

- 유튜브 크리에이터 시청자 성별·연령 / 광고 영상 모아보기 / 협업 비용 예측 / AI 크리에이터 리스트업.
- 구글 투명성 센터 = 일부만·영상 재생 X / **유관기 = 영상 소재 30·40개 다 노출**.
- 광고비 1억 쓰는 유저 즉시 식별 사례 (강사 지인 5천만원 쓰자마자 유관기 1등 떴음).
- **베이직만 결제해도 광고 레퍼런스 수집 OK**.

### 인플루언서 단가 매트릭스 (사전 VOD에 없는 새 수치)

| 등급 | 비용 | 효과 |
|---|---|---|
| 메가급 (1억 채널) | 1억~수억 / 화장품 1억 부르기도 | 모델·PPL용. 1회성 |
| 중상위 (500만 1건) | 500만 → 매출 50만 (실패 사례) | 효율 X |
| **마이크로** | 10·20만에 매출 100·150·200만 (강사 베스트) | 한 달 링크 살아 매출 누적 |

- 직접 컨택 vs MCN: **300만 → 150만·100만**. MCN 통하면 2배.
- 메가급 인사이트 ("메가 쓰면 중상위·마이크로가 자기도 하고싶다고 연락 옴") 비싸서 비추.

### 시청자 매칭 (유관기로 정량화)

- 채널의 시청자 연령 (예: 25-34, 35-44) 확인 → 우리 페르소나 매칭.
- 강사 사례: 40-64 / 35-44 / 25-44 시청자 채널만 골라 1년간 주 1회 = 50개 협찬 진행.
- ROAS는 **3~4건 중 1건**이 폭발. 평균은 4건 중 2건.

### 커뮤니티 바이럴 = 선택사항 (5주차 3강의 우선순위 마지막)

- **여성형** 죽방·여시·파우더룸·맘솔릭 (2030 → 4050 순) / **남성형** 보배드림·디시·디젤매니아.
- 디젤매니아 = 바이럴 효과 베스트.
- **역풍 리스크**: 휴대폰 케이스 업체 → 부정 바이럴 + 회원 중 기자 → 부정 인터넷 기사 발행 사례.
- 네이버 카페가 그나마 안전 (강퇴·벤만 됨).
- 정회원·오랜 활동 계정 없으면 그냥 옵션. 어둠의 경로 계정 구매 = 정지 빈번.

### 클로드 폴 크롬 + 스킬 = 커뮤니티 말투 자동 학습 (새 메소드)

- 죽방 베스트글 분석 → "글 작성에 쓸 수 있는 패턴 추출" → 말투·언어 패턴 자동 도출.
- 도출된 결과를 **클로드 스킬로 저장** → 슬래시 단축키로 언제든 호출.
- 매크로 + 복잡 업무 정의 결합. e-commerce 반복업무에 응용 가능.

### 바이럴 기획서 — 결과물 표준화·이해도·USP (새 핵심)

- 3대 요소: **결과물 표준화 / 제품 이해 / USP 전달**.
- 가장 흔한 실수: 판매자 관점 ("이거 꼭 써야 됩니다"). 답: **소비자 관점** ("이런 부분이 가려운데 너무 늦게 알아 후회").
- 경쟁사 리뷰 긁어 분석 → 고객이 우리 제품을 이해하는 방식 추출 → USP를 그 목소리로 표현.
- 기획서에 들어갈 메시지: 효과(%·메커니즘) X → **공감 + 감정**. 기미가 없어졌을 때의 감정.

### 5분 Q&A (수치 위주)

- 부유방 키워드 = 7월 피크. 묶여놓지 말고 시즌 전에 EC 기초체력 + 영상소스 확보 후 소액 태핑.
- DA 광고 유통기한 = **약 1년**. 계절성 SS·FW = 6개월. **3·4개월마다 신제품**이 정석.
- 5년 광고 운영 사례: 연 매출 500억 → 200억 + 해외 시너지.
- MOQ: 화장품 5천~1만개. 임상 받는 SKU = 5천~1만, 임상 없는 = 1천.
- 홈쇼핑 = 거의 안 남고 노출 후 낙수 떨어짐 + 리셀러 다수 → 비추 / 퍼포먼스 띄운 후 번들로 진입이 낫다.
