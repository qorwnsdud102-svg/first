# 1주차 우선피드백 2 — 이현지님 (아기 옷 세탁 / 이염방지 시트)

## 한 줄 요지

광고주 캔버스 1:1 피드백. **표면적 권위 vs 공감 권위 구분** + **맘카페 작업글도 조회수 높으면 봐야 함** + **문제정의 더 단순화** ("아기 옷 세탁" → 그냥 "세탁").

## 핵심 골자

### 광고주 제품·캔버스 요지

- 제품: 아기 옷 이염방지 시트 (세탁 보조).
- Persona: "아기를 잘 키우고 싶은 엄마, 초보 엄마."
- 인구통계: 30대 여성.
- 권위: "와이드 의류 환경학과 석사 출신 대표."

### 강사 피드백 1 — Persona 체크리스트화

- 추상적 "잘 키우고 싶음" X. **두려움/체크리스트 1~10번**으로 적기.
  - 예: "엄마로서 처음인데 빨래는 어떻게 해야 애기에게 문제 없을까?"
- → 콘텐츠 만들 때 카피 직접 변환 가능.

### 강사 피드백 2 — 인구통계 좁히기

- "30대 여성"만 = 너무 넓음.
- 필요 항목: **도시 / 삶의 형태 / 소득 수준 / 주거 형태**까지.
- Persona와 인구통계가 연결되어야 뾰족함 발생.

### 강사 피드백 3 — 욕구 칸 = 시각화 (1강 우선피드백1과 동일)

- "안전·사랑" 단어만 X. **무의식적으로 와닿을 수 있는 장면**까지.
- 표면 / 내면 두 칸 분리.

### 강사 피드백 4 — 문제정의 단순화 (제품 카테고리도 빼라)

- "아기 옷 세탁이 어렵다 → 쉽다" → **"세탁이 어렵다 → 쉬워진다"**로 더 단순.
- 어차피 제품은 아기 옷용이지만 직관성 우선. 1강 메소드의 극단 적용.

### 강사 피드백 5 — 권위 = 표면 권위 vs 공감 권위 (이 강의 새 메소드)

- 광고주 적은 권위 = "**와이드 의류 환경학과 석사**" → 엄마들 공감 어려움 = **표면적 권위**.
- 강사 추천: "**석사 + 본인이 직접 엄마**, 아이를 몇 살까지 키운 노하우 몇 년"으로 → **공감 권위**로 전환.
- 즉, 권위 안에서도 **학력·자격 = 표면 / 같은 처지에서의 경험 = 공감**으로 2분.

### 강사 피드백 6 — 컨셉 "안심" = 구체성 부족

- "안심할 수 있는 최고의 제품" → 어디서 인증한 안심? 어떤 키워드로 1등? 제조사 장점? 등 **구체 증빙** 필수.

### 강사 피드백 7 — **맘카페 작업글도 조회수 보면 봐야 함**

- 광고주 자료조사: "맘카페·블로그·이염방지 시트 글은 대부분 작업글이라 댓글 반응 없음" 결론.
- 강사 반박: **작업글도 조회수 높으면 = 실제 유입·노출 많았던 것** → 그 자체로 가치 있는 데이터.
- 강사가 직접 시연: 네이버 맘솔릿 카페에서 "이염방지시트" 검색 → 조회수 많은 글 ("다이소 아기 이염방지 사용해도 되나요?" 등) → 광고주가 놓친 니즈 발견.
