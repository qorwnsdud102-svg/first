# 1주차 우선피드백 1 — 최용민님 (제모 디바이스 + 여드름)

## 출처

- 원본: `마케팅/raw/그로스짐/9.1주차 우선피드백1.txt` (자동 트랜스크립트, 강사 1:1 음성 피드백)
- 강의: [[그로스짐]] 2기 1주차 우선피드백 1번 광고주

## 한 줄 요지

광고주가 작성한 캔버스에 강사가 1:1 피드백. **컨셉이 2개 섞여 있으면 캔버스 2개로 분리하라** + **페르소나 검증 도구 (썸트렌드)** 두 가지가 핵심.

## 핵심 골자

### 광고주 제품·캔버스 요지

- 제품: 제모 디바이스 (미세진동으로 절삭). 여드름 + 제모 두 효과 주장.
- Persona: 47세 고등학생 자녀 키우는 여성, 노화로 호르몬 변화 → 피지 조절 X → 트러블 폭발.
- 컨셉: "여드름이 많은 피부의 각질·잔털 제거 → 여드름 없어짐."

### 강사 피드백 1 — 문제정의 단순화

- "여드름 많은 피부의 각질·잔털 제거해 여드름 없어졌다" = 너무 김. 사람들이 "왜?" 의문 발생.
- → "여드름 많은데 여드름 없어진다"로 단순화 (1강 메소드 적용).

### 강사 피드백 2 — Persona 검증 도구 = **썸트렌드**

- **썸트렌드** = 키워드 연관어·연관 워딩 분석 사이트 (구글 로그인). 12개월 단위.
- "여드름 + 중년" 결합 분석: 피부 환경·건강 변화·증상·염증·트러블·**나이** 연관어 도출 → 실제 중년 여드름 호소 글 존재 검증.
- 그러나 검색량 데이터 자체에서 "**40대보다 청소년·젊은 층 여드름이 더 많음**" 발견 → 광고주 Persona 가정 (47세 갱년기 호르몬 여드름) 시장이 작을 수 있음.
- **40대 여드름 충동구매자 진짜 페르소나** (강사 경험): 자녀에게 사주거나, **자기 관리 많이 하는 결혼 안 한 40대 여성** (피지·화장 트러블).

### 강사 피드백 3 — **컨셉 2개 = 캔버스 2개로 분리**

- 한 캔버스에 "여드름" + "제모"가 공존 → 고객이 두 컨셉 동시 이해 불가.
- 솔루션: **캔버스 2개로 split**.
  - **캔버스 A**: 여드름 전용 — "여드름 나는 이유 = 이 털 때문" 식 포지셔닝.
  - **캔버스 B**: 제모 전용 — "왁싱샵 비용 vs 집에서 하루 100원 = 안전·깔끔 제모".
- **퍼포먼스 = 충동구매**. 직관적이어야 함. 두 컨셉 섞이면 매칭 안 됨.

### 강사 피드백 4 — 카테고리 포지셔닝 단계

- 제품 일치 → 니즈 반영 → 세분화 확장 (사전 VOD에서도 등장. 본 강의에서 광고주 사례에 적용).
- 광고주 케이스 (분리 후 캔버스 A): 일치 = 제모 디바이스 / 니즈 = 트러블·여드름 안 생기는 제모 디바이스 / 세분화 = 갱년기 여성 전용 트러블 없는 제모 디바이스.

### 강사 피드백 5 — 욕구 칸 표현법

- "인정욕" 적었으면 표면적 인정이 어떤 상황에서 이뤄지는지 적기.
- "성욕" = 후킹할 때 어떤 이미지 쓸지 적기.
- "안정욕" = 어떤 건강 안정인지 적기.
- → 욕구는 **시각화 가능한 장면**으로 적어야 광고소재로 변환됨.

### 광고주가 잘한 부분

- 사회적 근거: **에르메스급 향**, **방송 협찬** (사전 VOD 2강의 강사 섬유유연제 에르메스 사례 학습 결과로 추정).
