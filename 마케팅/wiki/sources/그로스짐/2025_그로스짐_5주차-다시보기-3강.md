# 5주차 다시보기 3강 — 페이드·오가닉 바이럴 우선순위 + 검색결과 채우기

## 출처

- 원본: `마케팅/raw/그로스짐/29. 5주차 다시보기 3강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 5주차 (바이럴 3편 = 정리·우선순위)

## 한 줄 요지

**의료기기·건기식 자극광고는 형사처벌. 답은 viral.** 검색결과(네이버·구글) 두 곳에 경쟁사 수만큼은 깔아라. 우선순위 = **블로그 → 카페 → 인플루언서 → 유튜브 → 커뮤니티**. 카페 챌린지·서브 인스타 계정으로 오가닉도 같이.

## 핵심 골자

### 의료기기·건기식 viral 우회 (새 디테일)

- DA 스폰서 광고로 자극적 의료기기 = **무조건 형사처벌**.
- 답: 광고 X. **카드뉴스 "통증 있는 사람이 꼭 쓰면 낫는 제품 1234"의 2등에 우리 제품** 노출 / 카페 글 "이거 쓰고 좋아졌다" → **처벌 불가능**.
- 효능효과 처벌 > 배합 기술 처벌. "특수 배합 컨셉어"는 처음엔 그냥 넣어놓고 식약처 연락 오면 "몰랐다" → 보통 처음엔 봐줌.
- 건기식 개별 인증 원료 = 대기업만 가능 (몇백~몇천 종 한정). 일반 제조사 X. 컨셉 원료 + 스토리(마운자로 깠던 식)로 후광효과 노려라.

### 검색결과 채우기 — 경쟁사 매칭 룰 (새 메소드)

- 우리 제품 키워드 검색 → 경쟁사만큼은 채워라 (메인키워드는 상위노출 작업필수, 세부키워드는 발행형 가능).
- **세부 키워드 100~200개 점유 시 매출 억 단위** (발행형 10만원 × 세부키워드).
- 누룩밤 사례: 한달후기·세달후기·연관글 → "한 달 쓰니까 끝났다" 메시지 광고에 반복 사용 추정.
- 아누아 사례: 매트버트글로우 내돈내산 / 한다루기 / 제품별 내돈내산 키워드 모두 작업.

### 구글 + 유튜브 영상 검색 (사전 VOD 4강 보강)

- 구글 = SEO (meta·tag·description) + 트래픽 점수 → 검색광고로 트래픽 의도 쏘는 작업도 가능 (불필요·콘텐츠 발행이 더 효율).
- 4050 여성도 갤럭시 크롬 검색 多. 네이버만 챙기면 안 됨.
- 네이버는 유튜브 영상 노출 막음 (자체 플랫폼 보호). 티스토리는 구글 SEO 매우 강함.

### 오가닉 바이럴 — 카페 챌린지 + 서브 계정

- 르무통·톰프로그램(화장품)·라셀턴(발톱밴프) — **공식 카페 인증 챌린지** 사례.
- 7일차 사진 인증 → 쿠폰·적립금. 재구매율·브랜드 각인 효과.
- 네이버 카페 글 노출 막힘 추세. 그래도 브랜딩으론 의미.
- VV정육점 인스타 = 인물 캐릭터 콘텐츠 / 베리쉬 = 공식 + "직원임" 리뷰 계정 분리.
- **대표님 캐릭터 서브 계정 → 페르소나 꿀팁 콘텐츠 + 브랜드 공식 페이지 태그**가 최선.

### 파워페이지 콘텐츠 확산 (새 디테일)

- 자체 채널 X, 파워페이지 활용. 월 200만 = 100~200만 조회. **월 1천만 조회 이상 = 효과**. 최소예산 천만원.
- 연예인 PPL 2차 활용 못 받아도 → A사·B사 + 우리 제품 섞어 만든 후광효과 콘텐츠 → 파워페이지 뿌림 → 우회 가능.

### 우선순위 (강사 결론 — 새 핵심)

> 블로그 ≥ 카페 > 인플루언서 ≥ 유튜브 > 커뮤니티

- **블로그·카페는 무조건**. 30대 후반~50대 초반 (퍼포먼스 핵심 연령) 모두 사용.
- 10·20대 제품은 퍼포먼스 효율 약함 (객단가 낮음).

### 과제 가이드 (구체)

- 블로그 체험단 가이드 + 내돈내산 2~3건 링크 + 퍼포먼스 캔버스 + "광고에서 어떻게 플레이할 건데 바이럴 의도 깔았습니다" 메모.
- 광고 켜놓은 건 잠깐 끄셔도 됨 (이거 안 하고 돌리면 차이 큼).
