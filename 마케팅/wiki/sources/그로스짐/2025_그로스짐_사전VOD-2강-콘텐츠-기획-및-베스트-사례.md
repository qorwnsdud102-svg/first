# 사전 VOD 2강 — 콘텐츠 기획 및 베스트 사례

## 출처

- 강의: [[그로스짐]] 2기 사전 VOD

## 한 줄 요지

콘텐츠 기획의 입력은 **AI로 정제된 리뷰·Q&A·충성 고객 FGI**. 페르소나는 인구통계 4요소만으로는 부족하고 **고객 행동·고통지수·실존 스토리·키워드** 5요소가 필요.

## 핵심 골자

### AI로 리뷰·Q&A 자동 분석

- 경쟁사 리뷰 만 개·2만 개 다운로드 → Claude/ChatGPT 입력.
- "위트 있는 커뮤니티 리뷰 원고"를 미리 학습 → 비슷한 결의 리뷰만 추려달라고 요청.
- 사례: **섬유유연제 리뷰** — "세탁기에 코 박고 3분 동안 냄새 맡고 있습니다" → 각색 없이 그대로 광고 소재로 사용.
- Q&A 반복 질문 → 상세페이지에 미리 표기 → CS 감소 (시간 = 돈).

### 충성 고객 FGI

- 화장품 월 50억 고객사 사례: 마케터가 **상위 1~100등 고객에게 1시간씩 전화**. 5명 만에 키워드 발굴 → 1년 효율 좋은 메시지.
- "고객은 이미 정답을 가지고 있다."

### 페르소나 5요소

1. 인구통계 (나이·성별·거주지·소득)
2. **고객 행동** — 우리 제품 만나기까지의 동선·스토리
3. **Pain Point + 고통지수** — 1~10. 충동구매·반복구매 = ~10 (불치병형: 기미·주름·트러블)
4. 실존인물처럼 느껴지는 스토리
5. 정답으로 박지 말 것 — **테스트로 발전** (성인여드름→청소년여드름 전환 / 보습 바디워시→향기 바디워시: 일 30~40만 광고비, 월 매출 100~300→10억)

### 컨셉·포지셔닝 (한끗 차이)

- 주름 위치별 분리: 눈가·팔자·이마·손 → **각각 다른 컨셉**
  - 이마 주름 = "반달 필러 안 맞아도 돼" → **반달크림**
  - 손 주름 = "애기손 크림"
  - 깊은 주름 = "**말기주름**" (말기 = 암 연상) / 깊은 계곡주름
- 피지 광고에 **벌레 기어다니는 장면** → 모낭충 연상. 치료 카피 안 써도 머릿속 공포 활성화.
- 리프팅·보톡스·필러·피부과·성형외과·동안·회춘 = 주름 카테고리 연상 키워드 풀.
- 5 Why 기법 = 경쟁사 빈틈 발견 (눈 밑 지방 재배치 수술 → 왜 나왔지? → 공감되는 신규 컨셉).

### 컨셉 = 새 제품 인식

- 같은 성분, 같은 용기, 뚜껑 색만 변경 → **4종 주름 세트** (이마·팔자·눈가·손) 판매. 됨.
- 약국 연고 짜는 용기 + 약국풍 디자인 = 일반 화장품 → 약처럼 인식.
- 마리오네트 주름·약산성/알칼리 폼클렌징 = **누군가 만든 컨셉 단어** (의학 단어 아님).

### 부작용 없는 화장품·건기식이 롱런

- "효과"의 본질 = 일시적 광택·보습·심리적 만족 (비타민C 화이트닝 = 씻으면 사라짐).
- 다이어트 건기식의 가르시니아·브로멜라인 = 실제 빠지지 않아도 "기분 좋음"이 매출 동력.
- 환불 = 부작용. **부작용 없는 라인이 롱런.**

### 사회적 근거의 구체

- "많은 사람이 쓴다" 표현법: 몇 명 샀다 / 줄서있는 사진 / 별점·리뷰 개수 / 다수 추천 사진 묶음.
- 안전한 선택 욕구 = 실패 회피 = 리뷰 의존.
