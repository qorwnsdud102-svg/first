# 4주차 다시보기 2강 — 쿠팡 리뷰·판매자 로켓·아이템 위너·경쟁 구도 리뷰

## 한 줄 요지

쿠팡은 **판매자 로켓 딱지 = 전환율 3배** + **아이템 위너 = 가격 매칭** + **사용자 클릭 기반 동적 롤링** 구조. 리뷰 + 썸네일 + 가격이 핵심 변수.

## 핵심 골자

### 쿠팡 핵심 메커니즘 — 강사 정의

**아이템 위너**: 같은 상품 등록 다수 판매자 중 **가장 저렴한 판매자**가 1등으로 노출.

**사용자 클릭 기반 동적 롤링**: 검색 결과가 **유저별로 다름** + 검색 후 클릭한 상품에 따라 다음 검색 시 노출 순위 바뀜. (실시간 시연: "기미크림" 검색 → 도미나스 클릭 → 새로고침 시 도미나스 8등으로 떠오름.)

### 판매자 로켓 vs 일반 로켓 vs 윙 — 전환율·수수료

| 유형 | 전환율 | 수수료 |
|---|---|---|
| **윙(일반)** | 3% | 가장 저렴 |
| 일반 로켓 | (3배 효과) | **45~50%** (절대 금지) |
| **판매자 로켓** | **9~10%** (윙 대비 3배) | 15~20% |

> **강사 격언**: "절대로 그냥 로켓은 하지 말라."

- 판매자 로켓 = 로켓 크로스 신청 + 3PL 합의로 입점 가능.
- 브랜드 키워드 검색 시 판매자 로켓이면 **전환율 70%까지** 봤음 → 하루 100건 검색 시 70건 추가 전환.

### 매출 vs 영업이익 구조 (강사 운영 사례)

- 자사몰 광고 100만원 → 자사몰 ROAS 150% (영업이익 마이너스 거의 90%).
- **광고 노출의 낙수효과** = 사람들이 쿠팡·스마트 스토어에서 브랜드 검색 → 추가 매출.
- 판매자 로켓이면 낙수효과 전환율 더 올라감 → **전체 ROAS 200% 가까이** 가능.
- **플리오 사례**: 자사몰만 보면 영업이익 0 → 쿠팡·유통 포함 시 전체 영업이익 **25~30%** 맞춤.

### "단감 바디워시" 케이스 — 허브 키워드 침입

- **메디톨드**라는 업체가 "단감 바디워시" 허브 키워드 광고로 미는 중.
- 그 옆에 같은 단감 성분의 **밀롭** 단감 바디워시가 가격 50% 싸게 일반 배송으로 가만히 등록 → 광고 안 해도 일부 가져감.
- 허브 키워드 단점 재강조: 상표권 X → 누구나 옆에 붙어 팔 수 있음.

### 경쟁 구도 리뷰 — "1등 약점 → 우리가 해결" 메시지

- 쿠팡 검색 시 사람은 1등만 사는 게 아니라 **2~5등도 비교** 후 결정.
- → 1등의 1점/3점 리뷰에서 반복되는 **불만 패턴** 발굴 → 우리 리뷰에 "우리는 그게 해결됐다" 명시.
- 사례: "1만 원대 같은 성분 단감 바디워시 썼더니 트러블 났는데, 이 2만 5천원짜리는 트러블 없고 각질 제거 잘됨" → 별점 1점·3점·5점 적절 분포.
- "사람 심리상 1등 별로다 → 2등 해결됐다 보면 2등 살 가능성 생긴다" — 강사 멘트.

### 쿠팡 리뷰 4점 vs 1점 작업 비율의 매출 임팩트 산수

- 전환율 1% → 0.9%로 떨어지면 100명 중 10명 손실.
- 객단가 5만 원 → 하루 50만 원 → 30일 1500만 원 → 두 달 3000만 원.
- 영업이익 10% 가정 → **300만 원 이익 날아감**.
- 강사 자체 브랜드 = 월 10만 건 택배. 5% 차이 = 5천 건 × 5만 원 = **2.5억 매출 날아감**.

### "그냥 로켓"의 50% 수수료 함정

- 입점 거절될 수도 있고, 매출 올라도 판매자 로켓 유지 추천.
- 일반 로켓 = "판매자가 직접 운영하지 않음" → 쿠팡이 다 떼감.

### 자사몰 가격 정책

- **자사몰 = 최저가** (정답). 쿠팡은 가격 매칭 강제 → 자사몰만 쿠폰으로 추가 할인 가능.
- 쿠팡·네이버 = DB 주체가 플랫폼 → CRM 자산 안 됨.
- 자사몰 회원수 = **올리브영·홈쇼핑 입점 협상 카드** ("우리 회원 20만 한테 메시지 쏠게").

### 자사몰 = 카페24 권장

- 메이크샵 < 카페24. 아임웹은 제한적.
- 자사몰 제작 비용: 100만 원 ~ 최대 500만 원 (강사 지인 업체 연결).

### CS·CRM·기타 Q&A

- 카페 모니터링: 카페차트 + **클로드 코드로 네이버 카페 크롤링 자동화** (불법 회색지대 → AI가 답변 안 줄 수도) / 외주 크몽 몇십만 원.
- 여성 뷰티 참조 커뮤니티: **쭉방 · 여시 · 인스타그램** (별도 여성 페르소나 계정 만들어 알고리즘 학습 시킨 후 시청).
- 화장품 같은 패키지·다른 성분 등록: 가격 방어용. 쿠팡은 최저가 매칭 강제 → 패키지 차이로 회피. 이지두 사례 = 용량별 행사 (15ml/30ml/100ml) 회전.
- 쿠팡 트래픽 광고: 화장품·건기식·뷰티 디바이스 카테고리는 **굳이 안 써도 됨** (브랜드 키워드 매출 카테고리이므로).
