## 한 줄 요지

본강의 진입 직전 Claude 데스크탑·MCP·코워크 환경 셋업 가이드 (2주차 1강은 거의 전부 도구 설치 실습).

## 핵심 골자

- AI 활용 (Claude·MCP)은 사전 VOD 4강과 동일. 추가 디테일만:
- **설치 필수 4종**: Claude 데스크탑 앱 / Node.js (3.12 추천) / VS Code / Git. + Python 보조.
  - 이유: Claude가 PC 제어하려면 OS 확장 프로그램이 필요. VS Code가 코드 작성 매개체.
- **코워크(Co-work) 기능**: 일주일 전부터 Mac 외 **Windows에서도 활성화** (강의 시점 = 발표 직후). 기존 커맨드 MCP 대체 가능. 폴더 지정 → Claude가 폴더 안 파일 작업·구조화·폴더 신설.
- 사례: 광고 어드민 데이터 매일 다운로드 → 폴더에 날짜별로 쌓아두기만 하면 Claude가 보고서 엑셀로 자동 추출.
- **코워크 vs 커맨드 MCP**: 채팅에서 새 대화로 폴더 접근하려면 여전히 커맨드 MCP 필요. 코워크는 폴더 지정 모드.
- **터미널 vs CMD 구분**: CMD = 도스 시절 명령 프롬프트. 터미널은 호환성 더 높은 별개 도구. Windows는 앱스토어에서 설치.
- **요금제**: 코워크는 원래 100달러($)짜리만 열려있었음. 강사 본인 = 200달러 플랜. 3만원 플랜 열렸는지 강사도 확인 안 됨.
