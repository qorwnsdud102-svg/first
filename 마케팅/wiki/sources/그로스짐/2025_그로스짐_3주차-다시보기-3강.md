## 한 줄 요지

리뷰 솔루션·위젯 종류와 강사가 실제 쓰는 **알파리뷰 + 펼쳐진 리스트형 위젯** 선택. 목요일 미션 = **리뷰 100개 + B&A 이미지 50~100개**.

## 핵심 골자

### AI 이미지 생성 단계 정식화 (3주차 2강 보강)

1. 원본 얻기.
2. **큰 범위부터** 1개씩 수정 (배경 → 인물 옷 → 얼굴 형태 → 소품).
3. 매 수정 시 **직전 결과 이미지 다시 첨부** 필수. 첨부 없이 텍스트 명령만 주면 첫 원본 기준으로 수정됨.
4. 한 번에 여러 개 변경 = 결과물 망가짐.

### 강사 운영 툴 스택

- **원고**: 클로드 (가장 잘 함, 분석·데이터 추출).
- **이미지**: 제미나이 나노 바나나 + 힉스필드 (튜닝된 SaaS).
- **외국 모델 이미지**: 미드저니 (보조).
- GPT는 원하시면 보조로 OK, 강사는 클로드 메인.
- 사장님 방침과 일치 = 클로드 + MCP 메인.

### 리뷰 솔루션 (실명·가격대)

- **카페24** 자사몰이 전제. 카페24 앱스토어 → "리뷰" 검색 → 솔루션 다수.
- 강사 사용: **알파리뷰** (가격 싸고 가성비, 업데이트 잘됨, 제휴 X).
- 다른 솔루션: **크리마 · 스냅리뷰 · 카페24 자체 리뷰**.
- 모든 솔루션이 거의 비슷한 구조라서 **가격 보고 선택**.

### 위젯 형태 선택 — 강사는 "펼쳐진 리스트형"

- 위젯 종류: 포토 위젯 / 게시판형 / 리스트형 / 플로팅형 / 펼쳐진 리뷰.
- 강사 = **펼쳐진 리스트형** (포토 + 글씨 같이 강제 노출).
- 이유: "고객이 클릭해서 보는 경우가 많이 없다 → 귀찮음 제거 → 강제로 읽게."
- 1등 따라 하는 게 최적화 X. **AB 테스트로 직접 확인**.

### 다른 마케팅 솔루션 추천

- **데이터 분석**: 카페24 통계 + 카페24 애널리틱스(곧 출시) + 구글 애널리틱스. 그 외 불필요.
- **CRM 자동화**: 비용 적으면 **데이터라이즈** (자동화 잘됨, 입문용). 비용 많이 쓰면 **스냅푸시** (담당자 1명 붙어 푸시 도와줌, 커스텀 더 다양).
- "물건이 팔려야 CRM이 의미 있다" — 초기엔 몰입 X.

### 1점·3점·5점 리뷰 비율 작업

- 50개 리뷰 만들 때 **재미있는/위트 있는 리뷰**도 1점·3점·5점에 2~3개씩.
- 1점 리뷰 작성법: "다 좋은데 [별 거 아닌 거 하나] 빼고 다 좋았다 → 그래서 별점 1점" → 오히려 띄워줌.
- 배달의민족 리뷰 패턴 (긴 줄거리·웃긴 리뷰)을 LLM에 예시로 주면 우리 제품 리뷰화 잘함 (20초).
- "웃긴 리뷰" 검색 → 이미지 결과물에서 패턴 다수.

### 미션 (목요일까지)

- 리뷰 텍스트 100개 (엑셀 파일).
- B&A 이미지 50~100개 (포토리뷰용).
- → 강사가 LLM·툴로 피드백 후 수정 요청.

### 영상 콘텐츠 + 블로그 랜딩 DA Q&A

- **블로그 랜딩 DA + 퍼포먼스 병행 사례**: 광고주 월 1~2억 광고비, 파워 콘텐츠만으로는 매출 4억 한계 → 강사 방식(퍼포먼스) 위에 얹으니 **월 매출 10억으로 성장**.
- 블로그 랜딩 단점: 픽셀에 모수 잘 안 잡힘. 픽셀 모수 쌓이는 퍼포먼스와 병행해야 선순환.
- 영상 콘텐츠 (기예프·스토리보드)는 7주차에 다루지 않음. 강사가 별도 교육 만들 예정.
- **CS 자동화**: 월 50만 원으로 챗봇 + 정형화 답변 자동화 가능 (직접 개발 X — 솔루션이 더 쌈).
