# 5주차 다시보기 1강 — 바이럴 필요성 + 키워드 섹션 분석

## 출처

- 강의: [[그로스짐]] 2기 5주차 (마케팅 기초환경 세팅 — 바이럴 1편)

## 한 줄 요지

광고 켜기 전에 **불쏘시개(바이럴)부터 까라.** 키워드별 검색 결과의 "섹션 순서"가 곧 고객 관심도. 쇼핑이 위면 쇼핑 공략, 블로그가 위면 블로그 공략. **내돈내산 키워드는 무조건 선점**.

## 핵심 골자

### 바이럴 = 선동 (강사 정의)

- "viral = 우리가 의도한 방향으로 바람 불기. 나쁜 말로 선동."
- 광고 전 **불쏘시개**. 누구도 모르는 제품의 활성화 장치.
- 실패 회피 심리 → 1만원·100만원 가격 오를수록 검증 욕구 폭증.

### 고관여/저관여 이분법 X — 입체 매트릭스

> 사전 VOD 3강의 페르소나 5요소 + 고통지수 1~10 보강.

- **고관여 + 고통지수** / 고관여 + 저고통 / 저관여 + 고통지수 / 저관여 + 저고통 → 4사분면
- 반복구매 vs 1회구매 축까지 더하면 8분면.
- 고통지수 8.9 = 당뇨 1단계. 6~7 = 당뇨 고위험군. **충동구매 임계 ~10**.

### 네이버 섹션 순서 = 키워드별 고객 관심도 (새 메소드)

- "기미" = 의사답변·증상·시술 위주 → **쇼핑이 맨 마지막** = 화장품 구매의도 거의 없음.
- "기미 크림" = 쇼핑이 맨 위 → **무조건 선점**.
- "얼굴 하얘지는 법" = 블로그·카페 인기글 위 → 자연어 검색 = **블로그 공략**.
- 키워드 분류 3축: ① 자연어 검색 ② 카테고리 메인·준키워드 ③ 세부 키워드 (제품 직결).

### 세부 키워드 효율 (사전 VOD에 없는 새 디테일)

- 검색량 500~1000 이상 세부키워드 = 발행형 블로그(건당 10만원)로 점유 가능.
- 1·2·3등 잡으면 유지기간 김 (몇 개월).
- 메인 키워드 = 보장형 상위노출 비싸서(몇백만원) **건당 발행 30개를 30개 세부키워드에 분산**이 더 합리.
- 광고 카피에 "PDRN 100" 같은 성분키워드 반복 → 검색량 자연 증가 → 그 키워드에 우리 글 깔려 있어야 전환율 0.1~0.2%p 상승.

### 내돈내산 키워드 (강사 신신당부)

- 누룩밤 자동완성 = 내돈내산·성분·효과·부작용·찐구매·실제후기.
- 모두 "진짜 후기 보고 싶은" 검색.
- **경쟁사가 우리 브랜드 키워드 침투 가능**: 누룩밤 검색 1등이 경쟁사 카페작업글 (조회수 폭발 → 판매 이탈). 상표권 주장 불가 (경쟁사 본인이 글 쓴 거 증명 못함).
- 광고 늘리기 시작하면 검색량 오름 → **광고 시작 전에 내돈내산 글 선점** 필수.
- 루비RN 제조특허 사례: 작년 9월 검색량 9680 피크.

### 사례·수치

- 마리오네트 주름·약산성 폼클렌징 = 누군가 만든 컨셉어 (사전 VOD 3강 보강).
- 모그람 팬티 (디젤매니아 카페 1글 → 5천만원 매출) — 부록2가 아닌 여기서 첫 등장.
- 화장품 임상 2달, 품목신고 포함 최소 3개월.
