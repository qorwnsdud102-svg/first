# 부록 1 — 상세페이지 특성 및 설계

## 출처

- 원본: `마케팅/raw/그로스짐/(부록) 상세페이지 특성 및 설계.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 부록 1 (상페 디자인 디테일)

## 한 줄 요지

상페 = **생명체** (계절·시즌·키워드·CTR 데이터로 계속 업데이트). 화자 4종 × 3 flow 무한 변주. 공포 → 배경 → 해결 → 불안 → CTA 5단 설득. **광고/상페/외부페이지 메시지 삼위일체** 깨지면 운으로만 팔린다.

## 핵심 골자

### 상페 구성 디테일 (사전 VOD 1강 + [[상페 구조]] 보강)

- 컬러·톤 **2~3개 제한**, 문장 짧고 쉬운 단어.
- GIF 남발 → 사이트 느려짐 → **3초 이탈** (사전 VOD 1강과 동일).
- GIF는 **USP 핵심 1~2곳에만** — 아이크림 = 눈 밑 차오르는 3D / 다이어트 = 내장지방 녹는 모션 / 화장품 = 흡수 그래픽.

### 11구성 요소 — 순서 가변 (새 디테일)

이미지·카피·인트로·상세·회원혜택·프로모션·가격·고객리뷰·근거·기능적 특징·페르소나 연출.

- 정답 순서 X. 광고 데이터로 매번 재배치.
- 메이커 스토리가 중요한 제품 → 페르소나·메이커 위로.
- 고객 리뷰가 광고에서 잘 먹힘 → 인트로보다 위.
- 연예인 사진 = 인트로 위에 넣었다가 전환율 떨어지면 고객리뷰 밑으로.

### 5단 설득 흐름 (새 메소드)

> 공포 → 배경 → 해결 → 불안·걱정·근심 → 행동요구

1. **공포** = critical pain 극단 표현. "이거 안 하면 죽어" 수준.
2. **배경** = 코로나·미세먼지·내장지방 수명 단축 등 사회 배경 + 숫자 자료. 제품 효능 X (식약처 처벌 회피).
3. **해결** = 우리 제품이 그 공포 벗어나게 해줌.
4. **불안** = "지금 안 쓰면 고통 극대화. 목숨·삶 지장."
5. **CTA** — 광고·랜딩·바이럴 모두 동일.

→ 이는 [[상페 구조]] 3 flow (문제 제기→해결→근거)의 5단 확장.

### 비유·연상 우회 카피 (새 사례)

- "동창회 갔는데 동기들이 성형수술했냐고 물어본다" / "요즘에 시술받아? 피부과 다녀?"
- 효능효과 직접 표기 X, 묘사·연상 OK → 식약처 회피.

### 광고 효율 좋았던 카피 = 상페 최상단 (새 운용룰)

- 패션 쇼핑몰 = 광고에서 CTR 좋았던 이미지를 썸네일로 재활용.
- 직접 피드백 (CS·리뷰·문의)을 상페에 즉시 반영.

### USP 방향성 1·2·3 = 같은 본질 (새 룰)

- 올바른 방향성: USP 본질 1개 + 표현 방식 3종.
- 틀린 방향성: USP·부수기능1·부수기능2를 동시에 테스트 (=실패 99%).
- 콘텐츠 200·300개 만들지 마라. 만 원씩만 써도 9만 원. CPC 천 원대에 의미 없음.

### 삼위일체 (새 핵심 프레임)

| 축 | 메시지 통일 |
|---|---|
| 광고 콘텐츠 | "향기가 좋아요" |
| 랜딩(상페) | "안전·무자극·애기도 OK" |
| 외부 페이지(블로그·리뷰) | "향기 좋은데 자극적이지 않은 향기" |

→ 셋 다 다르면 운으로만 팔림. **본질 같음·화자만 변주**.

### 히든 페이지 (카페24 상품 복사 + 진열 안 함) — 새 도구

- 같은 상품을 두 가지 USP 메시지로 만들어 링크로만 접근 → A/B 비교.
- 바디워시 보습 vs 향기 = 향기가 이김 → 사전 VOD 2강 사례의 운용 메커니즘.

### 광고 카피와 상페 카피 동일 → 상단 활용

- 광고 카피 효율 좋으면 상페 최상단으로.
- 패션 = 광고 CTR 좋은 이미지 → 썸네일 적용.

### 상페 = 생명체 (5축 업데이트)

- 계절·시즌(추석·설·새 학기) / 광고 카피 / 광고 이미지 / CS 피드백 / 외부 리뷰.
