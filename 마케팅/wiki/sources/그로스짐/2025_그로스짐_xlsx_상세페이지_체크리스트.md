# 상세페이지 체크리스트 (xlsx)

## 한 줄 요지

상세페이지 기획 → 제작 → 검수 전 단계 **체크리스트 워크시트** (xlsx). **변환 X — 사장님이 직접 엑셀로 열어 사용.**

## 활용 방식

1. 엑셀 열기
2. 사장님 자사몰·쿠팡·스마트스토어 상세페이지 1개 선택
3. 단계별 체크:
   - **기획**: 페르소나·pain·경쟁사 벤치·USP
   - **제작**: 인트로·메인 카피 10-15자·B&A
   - **검수**: 가독성·설득력·모바일 테스트
4. 누락 항목 보완
