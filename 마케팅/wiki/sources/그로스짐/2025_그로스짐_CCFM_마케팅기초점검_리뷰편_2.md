# CCFM 마케팅기초점검 리뷰편 2

## 한 줄 요지

리뷰 솔루션 선택 + 체험단 업체 사기 방지법 (171줄). 리뷰편 1의 후속 — 운영 디테일.

## 핵심 발견

### 스마트스토어 vs 쿠팡 속도·수량 규칙

- **초기**: 주간 2-3개 (자연스러운 증가 패턴 필수, 한꺼번에 수백 X = 스토어 정지)
- **성장기**: 주간 7개+
- 솔리샵·디지터 = 검증된 체험단 (에이베러 7년 운영)

### 리뷰 위젯 종류 (자사몰)

- **포토 위젯** (가장 권장) / 게시판형 / 리스트형
- 펼쳐진 리스트형 = 귀찮음 제거 강제 노출

### 업체 사기 방지 체크리스트

피해야 할 업체 패턴:
- "무조건 가능" 단언
- 선금만 받고 진행
- 계약 미확인 (구두만)
- 환불 거부 약관
- 리뷰 내용 검수 X

### 1-2점 리뷰도 작업

> 제품 만족 + 배송 불만 같은 혼합형. 순수해 보이며 신뢰도↑ + 경쟁사 약점 보완 내용 삽입 → 전환율 10% 상승 가능.
