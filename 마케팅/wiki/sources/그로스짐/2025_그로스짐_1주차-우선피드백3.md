## 한 줄 요지

광고주 캔버스 1:1 피드백. **숫자적 키워드 만으로는 부족 → 사람들이 왁자지껄 떠드는 워딩 데이터** 필요 + **유사 제품 많을 땐 경쟁력 비우는 게 나음**.

## 핵심 골자

### 광고주 제품·캔버스 요지

- 제품: Y존 향균 기능성 팬티 (특허 보유).
- 문제정의: "Y존 냄새가 나고 안 난다" (단순·잘 적음).
- Persona: "Y존 냄새가 나는 여성."
- 자료 조사: Y존 질염·칸디다·트리코모나스·가드넬라균 키워드.
- 권위: 산부인과 의사 추천.
- 통념: "25년 이노웨어 브랜드 평판 1위" (브랜드 대상으로 추정).

### 강사 피드백 1 — Persona에 "공감되는 상황" 필수

- 단순 "냄새 나는 여성" X. → **냄새 났을 때 불쾌했던 경험·공감되는 상황** 같이.
- 1강·우선피드백1·2와 동일한 메소드의 변주.

### 강사 피드백 2 — **키워드 데이터 vs 워딩 데이터 구분 (이 피드백의 새 메소드)**

- 광고주의 자료조사 키워드 (질염·칸디다 등) = **판매자 관점의 의학 키워드**.
- 진짜 필요한 건 = **사람들이 어떻게 왁자지껄 / 조잘거리며 얘기하는지의 워딩**.
- 키워드만 보면 "사세요 사세요 팝니다 팝니다"식 카피. 워딩 데이터 보면 "어? 이거 내 일인데" 공감.
- → 사전 VOD의 "스몰 데이터" 개념을 **키워드 vs 워딩**으로 명확화.

### 강사 피드백 3 — 욕구 표면/내면 분리 (다른 광고주와 동일)

- 광고주 적은 욕구 = 성욕만.
- 강사 추가 진단: **성욕(표면)** = 남친·남편 있을 때 냄새 고민 / **안전욕(내면)** = 향균 잘 돼서 건강한 성기 관리.
- 가려움 / 향균(안전) = 별도 욕구 줄기.

### 강사 피드백 4 — 사회적 근거 = **3~4개 필수**

- 광고주 1개만 적음 → 부족. 같은 제품군에서 3~4개 장점 있는 제품을 결국 선택받음.

### 강사 피드백 5 — 권위 좁히기 (강사식 일관 메소드)

- "산부인과 의사 추천" → **어떤 산부인과·어떤 사람·어디 출신**까지 구체.
- 추가 권위: **여성 연예인·인플루언서**. "여성은 선망하는 대상에 대해 팔로업 강함."

### 강사 피드백 6 — **경쟁력 = 없으면 비우는 게 나음** (이 피드백의 새 메소드)

- 광고주 적은 경쟁력 = "기능성 향균 팬티·향균력".
- 강사: 향균 원단은 시장에 이미 많음 → 경쟁력 X.
- 경쟁력 = **경쟁사 없거나 경쟁사보다 월등히 좋아야**. 둘 다 X면 **빈칸 두는 게 나음**.
- 억지로 채우면 광고소재가 희석됨 = 4강 "니즈만 부여" 원칙의 변형.

### 강사 피드백 7 — 기능적 특화 포지셔닝 = 키워드 좁히기

- "특허받은 향균" → 어떤 특허·어떻게 작용·최초최고유일 증빙 필요.
- "**질염**" 키워드는 법적으로 못 쓸 가능성 높음 (법무 확인 후).
- 가능한 우회: "Y존 3종균에 대한 케어 / 향균 속옷" 식. 강사 자신의 과거 광고 사례 = "**겨드랑이 액취증**" 키워드 (좁힘 = 기능적 포지셔닝).

### 광고주가 놓친 패턴

- 강사가 보기에 캔버스 완성도 = **약 50%**. 자료조사가 키워드 위주에 멈춤 → 워딩 단계까지 더 가야.
