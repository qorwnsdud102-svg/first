# 4주차 다시보기 1강 — 스마트 스토어 리뷰·키워드 SEO·중간 허브 키워드

- 강의: [[그로스짐]] 2기 4주차 (리뷰 편 2 — 스마트 스토어)

## 한 줄 요지

스마트 스토어는 **카테고리 매칭된 키워드만 노출**되며, 일반 명사 키워드 매출이 나는 카테고리(남자 드로즈·아이폰 케이스)와 브랜드 키워드 매출이 나는 카테고리(화장품)를 구분해서 키워드 SEO 전략을 짜야 함.

## 핵심 골자

### 중간 허브 키워드 (Hub Keyword) — 강사 정의

> 광고와 사이트 사이에 두는 **일반명사형 합성어**.

- 직접 브랜드명 검색 유도하면 광고 티남 → 중간에 일반 명사 비슷한 키워드를 둠.
- 사례: **약알칼리성 폼클렌징** ("누가 만든 키워드인데 일반 명사처럼 됨" — 사전 VOD 약국 컨셉과 동일 결).
- 효능 효과 제품(건기식·화장품)일수록 신빙성 부여 효과.

**허브 키워드의 단점**:
- 상표권·특허 어려움 → 경쟁사가 가로챔.
- 너무 상업적이면 "업체가 만든 키워드네" 인식.

### 리뷰어 풀 검증 + 가구매 정지 리스크

- 옛날엔 작업 아이디 돌려도 안전 → 최근엔 **반복 구매 패턴 → 스토어 줄줄이 정지**.
- 진성 유저(다른 IP·지역)가 실제 구매·배송까지 가는 솔루션이 안전.
- **신생 스토어가 한 방에 많은 양 작업 = 정지** (스토어 자체가 막힘).
- 너무 싼 작업 의심.

### 강사 추천 솔루션 — 셀패스 / 솔리샵 (에이베러 운영)

- **에이베러** 회사 (강사 지인). 7년 된 회사.
- 솔리샵 = 체험단 가입 사이트, **셀패스** = 판매자 어드민.
- 추천인 코드 **CCFM** → 20% 할인.
- 강사 자체 뷰티 브랜드도 여기서 작업, 한 번도 사고 X.
- 단점: **속도 느림** (실제 진성 유저 → 일주일 단위 작업).
- 보완: 레뷰 / 루커 / 투커 (속도 빠른 솔루션) 병행 → 일반 텍스트 리뷰는 빠른 데서, 포토는 솔리샵.

### 적정 리뷰 작업 속도

- 포토 리뷰 기준 **50개 정도**가 초기 적정.
- 주간 단위: 초기 2~3개씩 → 성장기 7개 이상.
- 한 방에 몇백 개 = 무조건 정지.

### 일반 명사 키워드 vs 브랜드 키워드 — 카테고리별

**일반명사 키워드로 매출 나는 카테고리** (상위 노출 자체가 매출):
- 남자 드로즈 / 아이폰 투명 케이스 / 맥세이프 케이스 / 사생활 보호 필름 / 지문 방지 필름 / 토너 패드.
- 기호성이 **명사 키워드**에 있음.

**브랜드 키워드로 매출 나는 카테고리** (상위 노출돼도 매출 안 늚):
- 기능성 화장품 / 색조 화장품 / 건기식.
- 기호성이 **브랜드**에 있음 → 노출량으로 브랜드 인지 쌓고 → 브랜드 검색하게.
- 예: 기미 앰플 검색량 9천만 회 정점이지만 연관어 다 브랜드 키워드 → 일반 키워드 상위 노출 무의미.

**예외 — 시장이 만들어진 케이스**:
- 베리업 등 거대 커머스가 "기미 앰플" 키워드 시장을 만들어 일반 명사 키워드로도 일부 팔림.
- 외국 트렌드 영향: 화장품도 **성분 키워드** 검색 증가 (히알루론산 종류·PDRN).

### 카테고리-키워드 매칭 구조 (실시간 시연)

- "기미 세럼" "기미 앰플" → 둘 다 **에센스 카테고리**로 묶임.
- "병풀 추출물" → 스킨 단독 카테고리. **앰플로 등록하면 병풀 추출물 키워드로 검색 안 잡힘**.
- → 상품명·태그·해시태그에 어떤 단어를 넣을지 카테고리 매칭 검증 필수.
- 카테고리 매칭 안 되는 키워드는 상품명에 **낭비할 필요 없음**.
- 판단 도구: **아이템스카우트 · 판다링크 · 키위(블랙키위)**.

### 네이버 SEO 가이드 활용법

- 스마트 스토어 → 상품 검색 SEO 가이드 PDF 다운로드.
- AI에 PDF 통째로 먹임 + 내 상품 정보 + 카테고리 → "어떻게 등록하면 좋은지" 가이드 자동 생성.
- "상위 노출 전문가" 외주 불필요.
- 옛날엔 크림으로 등록한 상품 에센스로도 추가 등록 = 가능 → **요즘은 abusing으로 정지**.

### 자사몰 → 스마트 스토어 검증 동선

- 자사몰 들어와도 바로 안 사고 "스마트 스토어에서 한 번 더 검색"하는 사람 다수.
- → **두 채널의 리뷰가 같은 USP 메시지로 깔려있어야** 의도한 동선 완성.
- 동선상 의도 세팅된 곳 vs 안 된 곳의 매출 차이 큼.

### 정육점 비유 — 접객 = 리뷰 답글

- 강사 = 정육점도 운영. 소고기 특별 X, 그러나 큰 인사·기다림·말 한마디로 단골.
- "쇼핑몰도 접객 가능. 리뷰 답글 = 접객. 개선되고 있다는 메시지가 critical."
- 답글 없는 리뷰는 아쉬움 표시.

### 1점 리뷰 작업의 정밀 활용

- 경쟁사가 1점 리뷰에서 반복하는 불만 패턴 발견 → 그 불만이 "우리 제품에선 해결됐다"는 1점/3점 리뷰 작업.
- 1점 리뷰 사유 = **제품 외 (배송·사은품 등)**, 본문은 제품 만족.
