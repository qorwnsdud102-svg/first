# 7주차 다시보기 1강 — 상세페이지 = 랜딩 페이지 전체 (재정의)

## 출처

- 원본: `마케팅/raw/그로스짐/32. 7주차 다시보기 1강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 7주차

## 한 줄 요지

상세페이지는 긴 이미지가 아니라 **광고 클릭 → 랜딩 직후 보이는 첫 화면 전체**(썸네일·가격·옵션·리뷰·포토리뷰 + 그 뒤 이미지). 도입부 70~80% 이탈하므로 도입부에 모든 에너지를 몰빵.

## 핵심 골자

### 상세페이지 재정의 (사전 VOD에서 처음 명시)

- 통념: "긴 스크롤 이미지 = 상세페이지" → **틀림**.
- 강사 정의: 광고 → 떨어지는 모바일 페이지 **상단 전체** = 상세페이지.
  - 썸네일 → 가격 → 옵션 → 리뷰(별점·개수) → 포토리뷰 → 그 다음에 우리가 흔히 말하는 이미지 상세페이지
- "상세페이지만 잘 만들면 매출 오른다"는 컨설팅·상세페이지 판매업자 말 믿지 말 것 — **리뷰·포토리뷰만 잘 바꿔도 전환율 2배 오른 사례**.

### 모바일·PC 상세페이지 분리 (꿀팁)

- 자사몰은 PC·모바일 상세페이지 **별도 세팅 가능** (스마트스토어는 X).
- 신고 단속 공무원은 PC로만 보는 경우 多 → **PC는 무난, 모바일은 USP를 자극적으로**.
- 단, 절대적이지 않음. 모바일 단속 시작 가능성도 있어 신고당했을 때 분리 확인.

### 이커머스 이탈률 데이터 (수치)

- 사이트 평균 체류시간 < 2분.
- 잘 나오는 브랜드 이탈률 50~70% / 안 팔리는 브랜드 95%.
- "도입부에서 광고비 절반이 낭비" → 첫 3초 = 후킹.

### 도입부 우선순위 = 포토리뷰 (테스트 결과)

- 강사 직접 테스트: ① 상세페이지 상단 먼저 vs ② 포토리뷰 상단 먼저 → **②가 압승**.
- 화장품·건기식 자사몰 모바일 = 거의 다 **썸네일 직후 포토리뷰 펼쳐진 상태**로 노출.
- 이유: Performance 광고 = **충동형 구매**. 광고에서 이미 설득됨 → 들어왔을 때는 **증명**(=다른 사람이 이미 효과 봤음)을 보여주는 게 효율 ↑.
- "리뷰 클릭해서 보겠지"는 판매자 착각. 클릭 안 함. **펼쳐서 다 보이게**.

### 상세페이지 6종 형식

1. 기본형 (이미지·카피)
2. **블로그형** — 텍스트·이미지 반복. 50~60대·농작물·판매자 셀링에 강함. 진정성·네이티브 톤.
3. **만화/웹툰형** — 4컷·6컷. 음식·밀키트·유명인 셰프류에 잘 어울림.
4. **자신의 스토리형** — 대표 손편지·진정성. 육아용품·아기 크림 (아토피 사연).
5. **질문 답변형** — 모델 인터뷰 컷. 기능·착용감 설명.
6. (텍스트 광고형) — 신문 시대 텍스트 광고 = 60대 어머님께 강력. 미국 사례: DB 모집 텍스트형 → 회원 500만 모음.

→ "ABCD 공식 없음. 타겟에게 매력적인 이야기가 정답."

### 사이트 업데이트·계절감 (오프라인 비유)

- 오프라인 백화점·맛집은 시즌마다 꽃단장.
- 온라인 = 홈페이지가 매장 → **계절·시즌 키워드(벚꽃·새 학기·SS 시즌·자외선)**를 도입부에 반영.
- "잘 되는 사이트는 끊임없이 업데이트". 노포 맛집 테이블이 깨끗한 이유와 동일.

### 이미지 설득의 2축

- **이익 표현** = before-after 긍정 시각화 (퍼포먼스 캔버스의 문제·해결).
- **손실 강조** = "안 사면 못 지운다" 부정 결과. 페르소나 방치 → 악화 모습.
  - 사례: "레이저 받아도 안 없어진 기미가 이 크림으로 없어졌다" / "레이저 1달 100만원 vs 크림 10만원" 가격 대비.

### GIF·동영상은 "굳이"

- "GIF 좋다 좋다" 통설에 강사는 **반대**.
- 단점: 호스팅 트래픽 ↑·로딩 3초 늦으면 이탈.
- Vimeo 자동재생용 = 유료 비쌈.
- **꼭 써야 할 때만**: 전후비교 신뢰도 / 메커니즘 설명 / 흡수·통과 장면 / 수치 변화 등 **2~3개 섹션**으로 제한.
- 사전 VOD와 동일 (GIF는 USP 핵심에만) — 보강 사례.
