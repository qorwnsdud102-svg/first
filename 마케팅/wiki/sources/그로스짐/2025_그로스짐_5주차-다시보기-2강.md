# 5주차 다시보기 2강 — 바이럴 채널별 특성 + 내적보상 6법칙

- 강의: [[그로스짐]] 2기 5주차 (바이럴 2편)

## 한 줄 요지

플랫폼별로 살 사람의 모양이 다르다. **외적 보상 = 사세요·사세요 / 내적 보상 = 친구가 추천**. 내적보상 6법칙(소셜화폐·계기·감성·대중성·실용성·이야기성) 중 **계기+실용성** 결합이 매출 가장 잘 남.

## 핵심 골자

### 채널별 유저 성향 매트릭스 (새 디테일)

- **트위터** — 10·20대. 모공·여드름·트러블은 잘 됨. 기미·주름은 안 됨. 50·60대 거의 없음.
- **쓰레드** — 자영업자·사업가·마케터 중심. 제품 판매 단발적 (연속성 떨어짐).
- **인스타그램** — 50·60대 릴스 소비. 카드뉴스 X, 릴스/영상 OK. 퍼포먼스 광고에 50·60대 구매 다수.
- **유튜브** — 4050이 오히려 많음. 30·40·50대·60대 분포. **애국보수 채널 50대 이상 → 객단가 100만원 건기식 판매 사례**.
- 카페: **여시·죽방·파우더룸 (여성)** / **디젤매니아·보배드림·맘솔릭 (남성·연령별)**.

### 핵심 사례 — "친구가 추천" 메시지 (새 사례)

- 옷가게 점원 "이거 입어보세요" = 사세요. 역효과 / 친구 "이거 써봐 진짜 좋다" = 즉구매.
- **블로그 체험단 표시 = "사세요 사세요"로 인식**됨 (요즘 소비자 똑똑해짐).
- 답: ① 협찬 표시 없는 내돈내산 블로거 5명 섭외 ② 지인·가족 명의 블로그 5개 개설 ③ 같은 컴퓨터에서 작성해도 노출 OK, 걱정되면 VPN으로 IP 분산.

### 내적 보상 6 → 4법칙 (강사 핵심 메소드 — 새 프레임)

| 법칙 | e-commerce 매출 효과 |
|---|---|
| **소셜 화폐** | 약함 (맛집·팝업스토어용. 이커머스에 도움 X) |
| **계기** | 강함 — "주부한테 유용한 5가지" "남자가 50대에 꼭 먹어야 할 5가지" |
| **감성** | 약함 |
| **대중성** | 약함 (사회 이슈·불합리) |
| **실용성** | 강함 — 꿀팁·치트키. 보관·재공유 동기 |
| **이야기성** | 약~중 |

> 4가지 정리: 계기 + 실용성 + 대중성 + 좋아하는 사람과 함께. **계기 × 실용성 결합이 매출 1위**.

### 충격 사례 — 칼마디 + 발기력 바이럴 (새 사례, 강사 본인 운영)

- 건기식 판매 대표가 **"칼슘+마그네슘+비타민D = 남성 발기력 향상"** 컨셉 콘텐츠 제작.
- 댓글에 자기 제품 패키지·제품명 노출.
- **하루 매출 5천만원**. 광고가 아니라 바이럴.
- 메커니즘 = 사전 VOD 3강의 성욕 레이어 + 카더라식 짜집기 (마그네슘 부족 → 발기력 떨어짐 TV 자료).
- 메타 픽셀 리타겟팅 자동 작동 → 살 때까지 광고 추격.

### 유튜버 섭외 (새 디테일)

- 메가급(3천만~1억) X. **마이크로(만 단위 구독자, 평균 조회 2~3만)가 효율 1위**.
- 비용: 일반 30·50만원 / 자기링크형 (수익공유) / **MCN 통하면 2배 비싸짐** (50→100만원·1500→3천만원).
- AI 자동화 섭외 (Claude Code + DM 매크로) — 비개발자도 가능.
- **인스타 파트너십 광고 (CC 광고)** = 인플루언서 프로필로 광고계정 연동 → 직접 만든 퍼포먼스 영상과 같이 돌리면 효율 상승.

### 시장조사 — 노트북LM + Gemini 딥리서치 콤보 (사전 VOD 4강과 동일하지만 새 도구)

- Gemini 3만원 결제 → 딥리서치 → 구글 독스 내보내기 → **노트북LM에 업로드** → 질문 답변.
- 보고서 다 안 읽고 요약 빠르게.
- 출처: 클리앙 · 뽐뿌 · MLB파크 · 보배드림 (50대 핫딜 정보).
