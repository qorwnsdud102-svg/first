# 사전 VOD 4강 — 고객의 마음 찾아보는 방법 + 퍼포먼스 캔버스 기획

- 강의: [[그로스짐]] 2기 사전 VOD (본강의 진입 직전 마지막 사전 VOD)

## 한 줄 요지

**퍼포먼스 캔버스**는 강사가 정립한 한 장짜리 워크시트. 본질적 존재이유 → 문제·해결 → 욕구·페르소나·USP·페인포인트·근거(4사회증거)까지 한 장에 정리. 작성 도구로 **Claude + MCP**.

## 핵심 골자

### 퍼포먼스 캔버스의 위치

- 8주 강의의 1주차에 **양식 작성 → 매주 디벨롭** → 6~7주차 상세페이지 → 광고 소재.
- "이 전체가 준비 안 되면 광고해도 유의미한 결과 없음. 두 달 광고비만 쓰고 나간 고객사 → 강사가 교육 만들기로 결심한 계기."

### 캔버스 1열 — 본질적 존재 이유

- "이 제품이 왜 세상에 있는가" 문제·해결.
- 가장 헷갈려 하는 영역 (1기 테스트 기수 피드백). 단순한 슬로건 X, **고객 고통 → 우리 해결**의 한 줄 정의.

### Claude + MCP로 시장조사 자동화

> 강사 메인 도구 = **Claude** (ChatGPT 비중 줄어듦, Gemini 보조).

- **MCP** (Model Context Protocol) = 데스크탑 파일·폴더 자동 정리. 엑셀 여러 개 통합·폴더 분류·파일명 변경.
- **Claude for Chrome** = 사람처럼 사이트 탐색.
  - "i-boss [[근육돌이]] 최근 글 찾아줘" → 자동 수집.
  - "유튜브 기미 키워드 30개 썸네일 분석 + 상위 댓글 5개씩" → 자동 보고서.
  - "쇼츠 제외해줘" → 필터 적용.
- 네이버 차단 우회 스크래핑도 Claude 코딩 (강사 비개발자).
- **딥리서치 vs MCP 차이**:
  - 딥리서치 = 정형화 보고서 (소비자 카피용 부적합)
  - MCP·크롬 자동화 = **스몰 데이터** (실제 트렌드·소비자 워딩) 가능

### "임팩트 있는 카피의 원천 = 스몰 데이터"

- 큰 데이터·통계 X.
- 최근 트렌드한 콘텐츠 + 사람들이 댓글에서 주고받는 워딩.
- 500억 뷰티 브랜드 광고 카피와 비슷한 결의 메시지를 Claude가 자동 추출 가능.

### 8주 코칭 흐름

| 주차 | 핵심 |
|---|---|
| 1 | 퍼포먼스 캔버스 양식 작성 |
| 2~5 | 캔버스 디벨롭 (시장조사·페르소나·USP·근거) |
| 6~7 | 상세페이지 초안 (디자인 완성도 X, 기획+이미지) |
| 8 | 광고 소재 기획 + 마무리 |
