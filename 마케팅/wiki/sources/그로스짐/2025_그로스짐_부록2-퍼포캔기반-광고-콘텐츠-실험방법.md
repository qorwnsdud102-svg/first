## 한 줄 요지

퍼포먼스 = **가설 → 실험 → 결과**의 무한 반복. 콘텐츠 1개에 한계비용·목표 CPA·CPC 정해라. **CBO·ABO·DCO 의도 없이 쓰면 똥**. 머신러닝이 리타겟팅 자동 → 별도 리타겟팅 = 비용 낭비. **메타 평균 빈도수 2 = 최적**.

## 핵심 골자 — 강의 핵심 메소드 운용 (새 디테일 다수)

### USP 방향성 — 부록 1과 동일하지만 매체 운용 결합

- 우선순위: 객관적 근거·사회 통념과 가장 밀접한 메시지부터.
- 5·6년 전 강사 자신이 콘텐츠 200·300개 만든 실패 회고.

### 광고 삼위일체 (부록 1 보강)

광고 카피 = 상페 카피 = 외부 리뷰. 본질 같음·화자만 변주.

### KPI 5종 — 콘텐츠 실험 시 정의 (새 메소드)

1. **한계 비용** — 콘텐츠 1개에 매몰비용 한도. CPC·CPM·평균 CPA로 역산.
2. **목표 비용** — 안 나오면 끔, 나오면 공격적 증액.
3. **목표 CPA** — 객단가 × (1/목표 ROAS).
4. **목표 CPC** — CPA × 평균 전환율 (커머스 평균 2%).
5. **BEP ROAS** — 광고비·매출 손익분기. **BEP 맞춰 자사몰 운영 시 유통채널(쿠팡·스마트스토어)에서 무조건 영업이익 발생** (강사 단언, 광고비의 20~100% 낙수).

### 실험 KPI 사례 계산

- 객단가 8만 + ROAS 200% 목표 → CPA 4만 → 전환율 2% → CPC 800.
- CPC 800 불가능 시 **객단가부터 건드려라** (통제 가능 우선순위).

### 직접 전환 vs 간접 전환 (새 디테일)

- ADMIN 기준만 보는 광고주 = **90% 이상**. 직접 전환만 봐도 상위 10%.
- 클릭 1일/7일 / 30일 전환 다 보기. GA·카페24 접속통계가 직접 전환 기준.
- 체류시간·이탈률 = 전환 0일 때 가설 잡는 보조 (전환 발생 후엔 매출만).

### 시너지 상품 조합 = ROAS 최대화 (새 메소드)

- 단일 상품 복수구매 (3+2·1+2+1) → **리셀러 폭주 → 자사몰 가격 무너짐** (사전 VOD에 없던 인사이트).
- 답: ① 시너지 상품 묶음 (크림+앰플+토너) ② **1인당 구매 제한** ③ GWP (본품+사용도구).
- 장기 사용 암시 ("3개월 마스터 세트"·"여드름 완벽 케어 세트") → 복수 구매 자연 유도.

### CBO·ABO·DCO 운용 (새 핵심 메소드)

| 옵션 | 정의 | 강사 룰 |
|---|---|---|
| **CBO** (캠페인 예산 최적화) | 캠페인 안 그룹 자동 분배 | 예산 10만원이면 그룹 1개·소재 2~3개. 의도 없이 그룹 늘리지 마라 |
| **ABO** (그룹 예산 최적화) | 그룹별 예산 고정 | 5만:5만 동등 가설 검증할 때만. CPC 좋은 그룹에 몰리는 머신러닝 회피 용도 |
| **DCO** (Dynamic Creative) | 이미지·영상·문구 자동 조합 | 메타 = 재활용 어려움 / **구글 = 무조건 사용** / GFA = 선택 |

- 전략: ABO에서 시작 → 고성과 소재 → CBO에 재투입.
- 광고팀 룰: **소재·타겟팅에 항상 "왜?" 적어라. 의도 없으면 똥**.
- 예산 그룹은 **CPA × 그룹수**가 통계적 의미 가질 만큼 있어야.

### 매체별 특성 (새 디테일)

| 매체 | 머신러닝 | 운용 룰 |
|---|---|---|
| **메타** | 최강 | 자동화 맡김. 지면·타겟 제한 거의 X |
| **구글** | 가장 느림 | **맞춤타겟·리타겟팅 적극** → 머신러닝 가속. GDN+검색+유튜브+디스커버리 미디어믹스 필수 |
| **GFA** | 반쪽짜리 | CPC 우선 → 직접 전환 좋은 콘텐츠만 증액. 4050이 핵심 |

### 광고 증감액 룰 (새 메소드)

- 직접 전환 70~80% / 간접 전환 20~30% 비율 증액.
- 둘 다 좋고 ROAS 목표 초과 → **직접 전환만** 증액.
- 둘 다 애매 → 동시 증액으로 시너지 노림.

### 메타 빈도수 = 2 (새 수치 — 사전 VOD에 없음)

- 강사 회사 분석 결과: 평균 **빈도 2**가 효율 피크.
- 빈도 3·4 = 간접 전환 비중 폭증 (이미 본 사람·산 사람 또 봄).
- 5·6·7 의도적 운용은 200만원 객단가 / 1회만 먹는 100만원 건기식 같은 특수 케이스만.
- → **별도 리타겟팅 광고는 거의 불필요**. 메타·구글·GFA 본 광고에 머신러닝이 자동 리타겟팅 포함.

### 고객 정의 입체화 (사전 VOD 3강 보강)

- 골프 관심사 + 4만원대 기미크림 매칭 = 실패. 러닝하는 분 = 햇빛 多 = 기미 多 추정 X (실제 안 씀).
- 핫한 카테고리 키워드 무조건 붙이는 광고주 多 → 시장조사·커뮤니티 확인 필수.

### 한계비용 = 광고대행사가 정해주지 않음

- 대표가 직접 정하라. 카테고리 평균 CPC·CPM·CPA 기반.
- 강사 회사: 카테고리 평균 데이터 제공 가능 (대행 시).
