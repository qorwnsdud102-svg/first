# 포토리뷰 + 이미지 만들기 — 실행 템플릿

## 한 줄 요지

리뷰 이미지 50-100장 만드는 **실행 가이드** (txt 메모). AI 단계별 수정법·구성 비율 명시. **사장님 직접 사용 가능 템플릿.**

## 리뷰 이미지 구성 비율 (AI 생성 50-100장 분포)

| 비율 | 종류 |
|---|---|
| **20%** | DNA (제품 자체) |
| **20%** | 바르는 장면 |
| **10%** | 애프터 좋은 피부 |
| 나머지 | 포토리뷰 블로그 변형 + 위트있는 1/3/5점 |

## AI 이미지 생성 단계별 수정법 (한 번에 1개씩)

순서대로 수정:
1. **배경**
2. **인물 특징·옷**
3. **얼굴**
4. **소품**

> 한 번에 여러 요소 수정 X → context hallucination. 새 채팅창에서 단계별로.

## 미션 정리 (사장님 직접 실행)

- 리뷰 이미지 **50-100개**
- 리뷰 내용 **100개**
- 매칭 + 엑셀 정리
