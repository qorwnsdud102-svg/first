# 사전 VOD 3강 — 내 브랜드 제품 포지셔닝

- 강의: [[그로스짐]] 2기 사전 VOD

## 한 줄 요지

고객은 **표면적 이유**가 아니라 **수면 아래 원초적 욕구**(매슬로 1단계 + 인간 기본 욕구)에 의해 산다. 같은 제품도 욕구 레이어를 어떻게 쌓느냐에 따라 매출이 달라진다.

## 핵심 골자

### 제품을 사는 3단계 이유

1. 표면적 이유 (예: "예뻐지려고", "건강해지려고") — 100명 중 90명이 여기에 집착
2. 심층적 이유 (안전·소속·존경·자아실현)
3. **원초적 이유** (수면 아래 — 식·성·수면·생존)

> "스토리텔링은 거들 뿐. 실제 매출 동력은 원초적 욕구를 건드릴 때."

### 매슬로 + 인간 기본 욕구 = 같은 결

| 매슬로 | 기본 욕구 |
|---|---|
| 1. 생리적 (식·성·수면) | 식욕·성욕·수면욕·생존욕 |
| 2. 안전 | |
| 3. 소속·사회 | |
| 4. 존경 | |
| 5. 자아실현 | |

### 다이어트 케이스 — 식욕 레이어

- 표면 = 자아실현 (날씬·예쁨)
- 원초 = **식욕** (생리적 욕구)
- 잘 팔리는 다이어트 광고:
  - 음식 녹는 장면 (식욕 자극)
  - 트렌드 키워드 = 당뇨·혈당 조절 (요즘 다큐멘터리에서 뜬 기전)
  - 위고비·마혼자로 (뉴스 신뢰 키워드 → 50대도 신뢰)
- vs 단순 Before/After만 = 매출 차이 큼.

### 남자 팬티 케이스 — 레이어 쌓기

- 표면 = 시원함·편함
- 레이어:
  - 페라리·정장·식스팩 → **자아실현·존경 욕구**
  - 술 좋아하는 영포티 → **사회적 동질감**
  - 시원·편함 → **남성성 강화 → 여성 어필** (성적 욕구)
- 단일 욕구 광고 vs 레이어 광고 = 효과 차이 큼.

### 명품 향수 = 모델 = 섹시성

- 향수의 본질 = 냄새 = **성적 욕구 환기**.
- 명품 브랜드도 다 함. "신생 브랜드가 이 욕구를 무시 = 명품도 하는 걸 무시하는 것."

### 퍼포먼스 광고 = 자극적이어야 하는 이유

- "SNS는 뇌 빼고 본다."
- 의사결정을 끌어내려면 자극·과대 같은 느낌. **퍼포먼스 광고 안 하고 100억 가능한 시장 거의 없음.**

### 컨셉·포지셔닝 = 단순 + 시대 트렌드

- 복잡할수록 좋다 X. 단순 + **시대·문화 트렌드 반영**이 더 좋음.
- 트렌드 = 그 시대 사람이 학습받은 키워드 = 신뢰 키워드.
