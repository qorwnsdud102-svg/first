## 한 줄 요지

**Gemini Nano Banana로 상세페이지 1장씩 단계 빌드업 = 디자이너 비용 절감 (100만 → 30~50만)**, 그리고 8주 메소드의 1줄 종합: **"퍼포먼스 캔버스 → 권위·증거 → 페르소나 입혀 광고 → 끝".**

## 핵심 골자

### 강사 종합 명제 (8주의 결론, 그대로 인용)

> "**퍼포먼스 캔버스 만들고 권위 붙이고 증거 붙이고 흐름 만들어서 그거 그대로 페르소나 입혀가지고 광고 소재 만들고 — 이게 다.**"

> "노래 잘 부르는 사람한테 '어떻게 잘 불러' 물으면 '그냥 부르니까 잘 되는데'처럼 — 체득되어 직관 설계 가능 수준까지 연습하면 됨."

### 화장품 임상 = "쓸 수 있는 멘트"의 사전 셋업

- "기미 제거된다"는 **기미 임상**이 있어야 광고에 쓸 수 있음. "가르마 이상" 멘트는 가르마 임상.
- 강사 노하우: **"광고 많이 받는 브랜드는 임상을 미리 다 받아둔 것"** — 신고 들어와도 그 멘트 못 빼앗기게.
- 무자극·형광물질 미첨가 = **서브 카피용**. 퍼포먼스 영향 X (받지 않아도 무방).
- 사전 셋업 절차: 경쟁사 광고 분석 → 자주 쓰는 멘트 추출 → 제조사·임상센터에 "이 멘트 쓰려면 어떤 임상 필요?" 역질문.

### 연예인 모델 활용의 디테일

- **A급 → B급 떨어진 연예인** = 2~3천만원에 **본인 촬영팀 10명 데리고 컷 6개** 찍는 게 핵심 ROI 모델.
- 컷당 500만원으로 환산 → 효율 매우 ↑.
- "A급 한가인 컷"보다 **"40대 다 아는 옛 A급 + 본인이 탈모 있던" 매칭** 효율 ↑ (디테일 차이).
- 촬영 스튜디오 직접 가서 멘트 디렉팅 → 진심 발화 강제. 지인 통한 단독 촬영 = "퍼포먼스 필름" 본인 페이로 6개월 계약.
- **운빨 사례 — IU 폭망사녹수다**: 1년 5억 계약 + 1년 연장 → 4월달 폭로 사건 → IU 검색량 30만 → **380만 폭증** → 브랜드 매출 **5배 ↑**.
- SBS 자회사 미디어 랩사 = 6개월~1년 전 드라마 캐스팅 정보 보유 → 신인 미리 발탁 셀링도 가능 (운빨 시드).

### 권위·증거 ≠ 연예인만이 아니라 "한끗"

- "엄마·아빠·내 자신·내 손등" — 강사 본인 = "GFA 3000억·메타 1000억·구글 합쳐 — 그리고 회사가 몇백억 규모" 자체가 권위.
- 강사 본인 사례 카피: **"이 제품을 10년 동안 고민 끝에 개발 / 2만 6700시간 투자"** — 시간 시각화로 스토리텔링.
- 기존 사례: 화장품 통째로 짜서 마시고 비비는 대표 → 권위 됨 (극단형).

### Gemini Nano Banana로 상세페이지 자동 생성 (실시간 시연)

**핵심 원칙: 한 장씩, 새 채팅창마다 빌드업 (포토샵 레이어 비유)**

- 전체 상세페이지를 한 번에 만들지 X. **인트로 1장 → 그 다음 1장 → …** 분리 생성.
- 1장 누끼 합성 → **그 결과 이미지를 새 채팅창에 첨부** → 카피 추가 → 다시 새 채팅창 → 색상 변경 → 다시…
- 같은 채팅창에서 길게 대화하면 **컨텍스트 길어져 픽셀 뭉그러짐 + 바보**. **새 채팅창마다 복사 첨부**.
- "장면 1번 인물 그대로 + 뒤 배경만 변경" 식 일관성 유지법.
- **빨간색 표시로 "여기에 카피 넣어줘" 지시 가능** — Gemini가 위치 인식.
- 폰트 종류·크기·컬러·스트로크·레이어 효과까지 텍스트로 디테일 명시 가능.
- 업스케일링 AI로 화질 보강 → 오타 부분만 다시 Gemini로 수정.

**비용 절감 효과**:
- 디자이너 외주 100만원 → 70~80% AI 완성 후 위임 시 **30~50만원**.
- 리터치 불필요 시 그대로 판매 가능.
- 30달러 Gemini 결제 = 하루 **광고 소재 100장**.

**클로드 + 피그마**: 강사 본인은 안 씀. 6개월 전 평가 = 한계 있음. 디자이너 아닌데 데드존·픽셀 간격·자간·행간·폰트까지 입력해야 → 비효율. AI 디자인 티 나는 이유 = 마이크로 매니징 항목 누락.

### 광고 트래킹 트릭 — 자체 블로그 페이지 복제

- 네이버 블로그 = 픽셀 못 심음.
- 똑똑한 운영자: **네이버 블로그 페이지 그대로 복제**해서 자사몰에 히든 만듦 → 픽셀 다 심김 + 고객은 네이버 블로그로 인식 → **광고 트래킹 + 신뢰 동시**.
- "1등이 이렇게 하니까 따라야지"의 고정관념 깨는 사람이 빨리 성공.

### 의사 모델 시각 연출 = 가운 X

- 의사라고 명시하지 말 것. **흰색 가운도 단속 대상**.
- 물리치료사 풍 / "딱 봐도 의사상 = 무의식 통념 활용" — 강사 친구 사례: "일주일 안 씻어 보이는데 하루 3번 샤워하는 친구 = 누가 봐도 의사상". 이런 통념 얼굴로 권위 매김.
- → **무의식·기본 욕구(성·식·안전)와 연결될수록 강력**.

### 8주 강의 마무리 (강사 마무리 멘트)

- 클로드 후속 강의 별도 결제 (선착순). **100달러짜리 Claude Code 요금제 필수**. 30달러는 토큰 부족.
- 강사·서브 강사 1명 더 합류 — 온보딩 별도.
- "축하드립니다. 8주간의 과정이 끝났습니다."
