## 한 줄 요지

캔버스 하단 **경쟁력·컨셉·카테고리 포지셔닝** 칸 채우기 + 퍼포먼스 광고 운용 부속 지식 (가격 정책·유통 믹스·리셀러 차단·플랫폼 선택).

## 핵심 골자

### 경쟁력 5대 요소 — "제품 제조 전에 결정해야"

- **가격**: 압도적이어야 의미 (예: **착한 구두** = 3만 원 시장에 9900원·만원 → 1년 매출 200억). 화장품·건기식은 가격 내리면 오히려 효과·품질 의심됨 = 비싸게 팔아야 함. 강사 사례: **만 원 시장 화장품을 4만 원에 팔았던** 케이스 (성분 동일·용량 더 작음). **정관장·참단** 홍삼 함량 논란에도 "비싸면 좋다" 심리적 허들.
- **가성비**: 두 가지 기능 한 디바이스 (뷰티 미용기기) 식.
- **기능**: 세상에 없는 기능 어렵지만 — **욕실화 사례**: 몰드 살짝 변형해 들리는 형태 → 물때 안 낌. 1년 수십만 족 판매 (강사 형님).
- **품질**: 화장품·건기식은 함량 너무 올리면 **의약외품·의약품으로 분류** = 못 팜.
- **명성**: 70년·50년 전통 → 가져올 수 있으면 가져옴.

### 니즈 vs 원츠 — 경쟁력 부여시 핵심

- 니즈 = **실제 원하는 것**. 원츠 = **필요에 의해 말하는 것**. 경쟁력은 **니즈만** 부여해야 함.
- **남자 팬티 사례 (광고주)** — 피드백: "팬티가 얇아서 흡수 안 됨" → 흡수 패드 덧댐 → **판매량 떨어짐**. 진짜 니즈 = "얇고 시원함" (덧대니 답답). 다음 시즌: 흡수 기술 개선하면서 더 얇게 = 회복.

### 컨셉 = 최초·최고·유일

- 카테고리는 **만들 수 있음**. 최고·유일도 어떤 관점·어떤 대상으로 보면 달라짐.
- **누룩밤 수면팩 사례** (1강과 동일) = 크림 → "수면팩"으로 프레임 전환.
- **바르는 보르피린**: 보톡스 성분이라는 키워드를 더 임팩트 있게 (미국 보톡스 / 슈퍼보톡스 등 파생).

### 마인드맵 키워드 확장 시 — "연관성 없으면 버림"

- 보르피린 ↔ 필러 = 연관성 X (그냥 시술 키워드라 가져왔어도 못 씀).
- 보톡스 ↔ 슈퍼보톡스 ↔ 미국 보톡스 = 연관성 O (권위 메시지 "미국 보톡스 성분 가져왔다"로 활용).

### 퍼포먼스 광고 매출 임계 (강사 경험)

- **국내 퍼포먼스로만** = SKU 1~2개 잘 터지면 200억 / SKU 4개 성공 시 300~400억.
- **달바**: 천억까지는 GFA·메타 매일매일 = 퍼포먼스로 큼. **천억~2천억 시점부터 브랜딩 광고 + 유통 채널 분배로 전환**. 이미 숙주가 너무 많고, 숙주 죽는 주기가 있어 계속 새 숙주 내야 함.

### 유통 채널 믹스 = 자사몰 광고의 낙수효과

- 자사몰 퍼포먼스 광고 = 쿠팡·스마트스토어 자연 매출 같이 발생 (광고 안 해도).
- 자사몰만 고집하면 한 50억에서 자빠짐. 200~300억 가려면 직원 60명 + 자체 유통MD.
- **도매·친구·사돈팔촌 절대 금지** = 가격 망가지고 계약서 잘못 쓰면 끝.
- **할인 정책**: 가격 지키면서 입점, 행사 칠 때만 할인.
- **2+2 절대 금지** = 리셀러 붙음. 대안 = **GWP 전략** = 앰플 2개 사면 크림(광고 X 제품) 1개.

### 광고 첫 플랫폼 추천

- 광고비 풍족 X (일 10~30만 원) → **메타 (인스타) 원툴**. 안정성 좋음.
- 이익 구간 나오면 그것만 재투자. 까먹지 말기.

### GFA·메타 광고 소재 분석 방법

- **GFA = 라이브러리 없음**. 계속 보면서 리타게팅 detecting 받는 수밖에.
- **메타** = 좋아요·댓글·공유수 / **유튜브·구글** = 조회수 보면 됨.
- **좋아요·댓글 작업 = 의미 없음** ("바보 같은 짓"). 안 쳐도 매출 나올 소재는 다 나옴.
- **잘 팔리는 광고 = 오래 돌아간 광고가 아니라 최근 조회수 폭증한 광고**.
