## 한 줄 요지

상세페이지 카피·인트로·설득 박자의 실전 룰. 메인 카피 10~15자·설명 카피 15~20자, 인트로 4역할(pain-고통지수-공감-기대), 설득 4단계(문제 제기 → 심화 → 해결 → 근거)가 무한 반복.

## 핵심 골자

### 블로그 랜딩의 장단점 (50대 타겟)

- 50대도 일반 자사몰 랜딩으로 충분. 블로그 랜딩 **필수 X**.
- 장점: 실제 후기 느낌 → 콘텐츠 부실해도 매출 보완.
- 단점: **광고 트래킹 거의 불가** (메타·구글). 네이버 GFA + 네이버 페이 결제 시 일부 트래킹.
- 네이버 블로그 = 50~60대 신뢰 ↑ / 티스토리·워드프레스 = 픽셀 추적 가능하나 신뢰도 ↓.

### 카피 자간·글자 수 (모바일 모달리티)

- 메인 카피: **10~15자** (1등 브랜드 패턴 = 8~15자)
- 설명 카피: **15~20자** + 줄 수 추가로 가독성.
- 1등 브랜드(AHC·닥터멜락신·누룩밤) 자간·횡간·폰트 크기는 AI에게 캡처 넣어 패턴 분석 → 그대로 적용.

### 후킹 메시지·이미지 일치 (sync)

- "의사 추천 기미 크림" 카피 → **의사 모델 + 병원 배경**과 합성.
- 사람 = 이미지 인식 후 텍스트 → 둘이 일치하면 인식 속도 ↑.

### 컬러·무드 컨셉 (플리오 사례)

- 강사가 플리오 광고하며 배운 핵심: **브랜드 컨셉 컬러 반복 노출** → 무의식 각인.
- 단, 광고비 월 10~20억 단위에서 효과. 미리 정의 vs 나중에 바꾸기 차이 큼.
- 마켓컬리·이마트·무신사 = 플랫폼 단위 컬러 반복.
- 머신러닝 광고 = 광고 자체가 반복 노출 → 컬러 일관성과 시너지.
- **무드 한끗 차이**: 같은 더마 화장품도 "일본 약국풍" / "미국 첨단 병원풍" / "따뜻한 일상풍"으로 분기. 사대주의 활용 — 미국·독일·유럽 색감·패턴 가져와 어색함 없는 사회적 통념 활용.

### 인트로 = 상세페이지의 얼굴 (4역할)

> Pain Point 건드림 → 고통지수 표현 → 공감 형성 → 기대감 유발

- 단, 정답 X. **인트로에 무조건 페인포인트 스토리 넣을 필요 없음**. 일주일 단위로 인트로 바꿔가며 테스트.
- 누룩밤 사례: 공식 판매처 강조 → 연예인 협찬 → intro pain 스토리 순서.

### 의심 4종 (선제 해결)

광고 보고 들어온 고객이 망설이는 것:
1. 정말 효과 있나?
2. 합리적 가격인가?
3. 나한테 맞을까?
4. 다른 사람도 만족했나?

→ 리뷰 비교 + **포모(한정수량)**가 의심 4종을 동시 봉합.

### 설득 4단계 (반복 박자)

문제 제기 → 문제 심화 → 해결 제시 → 근거 제시

- 4단계 순서 고정 X. 1·4 / 4·1 / 1·4·1·4 / 1·2·3·4·1·4 모두 가능. 화자만 바꾸면 무한 변주.
- 한 섹션 안에 4단계 다 들어가기도, 페이지 1장에 다 들어가기도.

### 문제 제기 = 공포 활용

- 간 건강기능식품 = "간이 썩고 있다".
- 보안·유아용품 = 과거 사고·통계 (40대 남성 음주 비율 → 간암 사망 비율).
- 단, 과대 주의 → 역효과.

### 숫자적 증거 = "기간 + 변화율"이 가장 강력

- "7일 이내 기미가 없어진 고객 N명" >> "기미로 많은 효과 봤습니다"
- 인트로 썸네일·옵션·임상 자료·제품명·옵션명까지 숫자 박기.

### 설득 강화 4종 (사회적 증거 변형)

- 실제 리뷰 + **리뷰 개수**도 강조 (스마트스토어 6개월 점수·리뷰 카운트)
- **누적 판매 N개** (제목·상단 노출까지)
- 미디어 노출·연예인
- 인증·수상 (브랜드 대상 100~200만원짜리도 한 섹션 추가 → 효과 ↑. "사람 이력서에 자격증 추가"와 동일)

### 프레이밍 (긍정 뒤 부정)

- **스피큘 제품·열감 화장품 사례**: "따끔거리는 = 효과의 증거"로 프레임 → 부작용 클레임 → 효과 인식으로 전환. 강사 자체 브랜드의 열감 크림 = 광고 소재에 "열감 = 주름 펴주는 신호" 프레임 → 고객이 직접 따뜻함을 느끼며 "주름이 펴지는 것 같다" 반응.

### 회원가입 강제 시 전환율 (실험 데이터)

- 네이버 페이 클릭 직전 "추가 할인 못 받음" 경고 띄움 → 회원가입률 ↑, **최종 구매전환 大幅 ↓**.
- → 회원가입은 "쿠폰 추가" 같은 베네핏형 유도만. 강제 X.

### AI 비유 활용법 (개념 지름길)

- 어려운 성분·기술 = 자석·구름·비서 같은 **고객 머릿속 폴더**로 빗대기.
- 비유 카피 + 서브 카피로 정확한 기술 설명 병기.
