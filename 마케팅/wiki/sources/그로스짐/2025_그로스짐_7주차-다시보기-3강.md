## 한 줄 요지

확신 4종(무료배송·환불보장·한정수량·고객 후기)에서 **한정수량 단독으로 전환율 +0.23%p (이커머스 평균 2% 기준 +10%)**. 메시지 7유형은 단독 X — 3~4종을 한 섹션에 조합. 모바일 98%, PC 무시.

## 핵심 골자

### 확신 제공 4요소 (실전 효과)

| 요소 | 효과 |
|---|---|
| 무료 배송 | 일부 도움. 원가율 안 좋으면 능사 X |
| 환불 보장 | "냅다 환불"은 효과 ↓ (다 하니까). 문제 해결 근거 + 환불 보장 묶을 때 임팩트 ↑ |
| **한정 수량** | 강사 모 디바이스 사례 = 폭발 매출. **+0.23%p (=+10% 상대치)**. 실제 한정수량 아니어도 OK |
| 고객 후기 | 70~80% 차지. 광고 소재마다 **연결되는 후기를 매칭**해서 최상단 고정 |

- 누룩밤 한정수량 표현: "지금 N명이 함께 보고 있습니다" / "현재 옵션 N개 남음" → 옵션·배송비 옆에 배치.
- 후기 정렬: 최상단 고정은 판매자 의도. 광고 CTR·CPC 좋은 소재의 설득 내용에 매칭되는 후기로 매주 갱신.

### 구매 심리 자극 메시지 7유형 (조합 룰)

7유형(비교·숫자·권위·증거·스토리·감정·CTA 등)을 한 섹션에 **단독 X, 3~4종 조합**.

- "효과가 좋은 크림" < "피부과 전문의 93% 추천 + 어떤 피부과·이유·위치·원장 경력"
- 광고 소재에서 셋업한 메시지 = 상세페이지 + 바이럴(블로거·인플루언서·유튜버)에 **삼위일체** 반복.

### 메시지 위치별 배치 (4구간)

도입부 / 중단 / 중하단 / 하단 — 단, 무조건 분리 X. 한 페이지에 다 들어갈 수도, 한 섹션에 설득·문제 제기 동시 들어갈 수도. **퍼포먼스 캔버스 기반 유연 조합**.

### 컬러·폰트 일관성 룰

- 강조 폰트 + 세부 폰트 = **2종**.
- 컬러 = **2~3종 max**. 같은 톤에서 연·진 변화. 브랜드 컬러 + 배경 컬러가 **같은 톤 안에서** 움직임.
- 디자이너 지시용 룰 — 대표가 알고만 있어도 디자인 디렉팅 가능.

### 손실의 피·포모 (인트로 직격탄)

- 광고 소재에서 이미 설득한 내용을 상세페이지에서 반복하기보다, **상단 = 손실의 피 + 이익 강조**.
- "지금 사야 봄 특가" / "지금 기미 시즌 특가" — **명분만 다르게** 반복. 한정 수량·세트 구성을 시즌별로 살짝 변형.
- "다 하고 있으니까 안 통해" 라는 판매자 착각 → 본질적으로 통상 메시지가 다 통함. 차이는 **임팩트 키워드·상황**.

### 모바일 우선 룰 (수치)

- **98% 모바일 구매**. PC 신경 거의 X.
- 폰트 크기: 모바일 **메인 30~40 포인트** / 서브 **14 포인트 ↑**
- 이미지 압축: 500KB 이하 권장 (텍스트 깨짐·로딩 ↓)
- 이커머스 기초체력 부족 시 흔한 실수: PC 디자인 / 잦은 텍스트 이미지 혼란 / **가로 스크롤** 레이아웃 / 작은 버튼 / LTE 환경 무시한 고해상도 이미지.

### 트래킹 지표 우선순위

1. **상단 변경에 따른 전환율** (주간 단위)
2. (전환 0일 때 보조) 스크롤 심도·체류시간. 30초 → 40초 미세 변화도 전환율 변화 신호.

### 실습 숙제 (다음 주 = 8주차)

- 엑셀(또는 피그마) 키고 모바일 사이즈로 **랜딩 페이지 1장 기획서** 작성.
- 경쟁사 캡처 + 우리 메인 카피·서브 카피·가격·옵션·리뷰·포토리뷰 캡처 모음 → 화살표·박스로 배치.
- 퍼포먼스 캔버스 시트에 추가. **50% 완성 상태로 8주차 입실** → 실시간 피드백.
- 본질만 지키면 OK: 구매 버튼 명확 / 모바일 가독 / 인트로 의도 / 리뷰·증거 명시. 인트로 위치·순서 자유.

### 광고 소재 = 상세페이지 = 바이럴 = 같은 메시지

- 광고 소재 만드는 방법과 거의 동일 (기획·설계·제작·검수·개선) — 단, **분기 단위가 아니라 1~2주 단위 빠른 루프**.
