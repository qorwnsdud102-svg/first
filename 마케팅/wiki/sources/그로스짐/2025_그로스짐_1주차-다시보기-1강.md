# 1주차 다시보기 1강 — 캔버스 시작·문제정의·키워드 확장 마인드맵

## 한 줄 요지

캔버스 작성의 첫 칸 = "문제·문제해결" 한 줄. **3초 안에 누구나 이해 가능한 한 문장**으로 좁히는 것이 광고 효과의 시작점.

## 핵심 골자

### 캔버스 1행 — 문제·해결의 문장 패턴

- "X가 있는데 X가 없어진다" 형태로 명사+상태만. 예: 기미가 있는데 기미가 없어진다 / 발 냄새 / 머리 땜빵.
- 판매자 관점("어떤 색소가 어떻게 변화") = 금지. **카테고리의 문제**로 정의 (내 제품으로 좁히면 효능 효과 설명에 빠짐).
- SNS 광고 = 정보·소통 사이에 끼는 광고 → **3초 이탈 룰**. 빵빵 하면서 before/after 보여주거나 "내 이야기인데" 공감 안 되면 끝.

### 키워드 확장 마인드맵 (이 강의의 새 메소드)

> 사전 VOD 4강의 "키워드 확장"을 더 구체화한 단계 절차.

1. 메인 카테고리 키워드 (예: 세럼) 검색 → 검색량 / 경쟁사 확인.
2. 경쟁사 컨셉만 보고 끝나면 절대 못 이김. → **확장 1**: 효과 키워드 (속건조·물광·스킨부스터·리주란·물광주사).
3. **확장 2**: 의학·시술 키워드. **확장 3**: 점점 세지는 방향 (기대효과 → 약/연고 → 질병/암).
4. 마인드맵으로 펼친 뒤 **검색량 가장 높은 것 1개**를 문제·해결 문장에 꽂음. → 위/아래는 채워넣기만.
5. 도구: 블랙키위 · 네이버 검색광고 도구 · 네이버 데이터랩 쇼핑인사이트.

### 새 사례

- **누룩밤 "기미수면팩"** — 크림인데 "수면팩"으로 포지셔닝. 잘 때 바르고 자면 기미가 옅어진다 = 단순 한 줄. 강사도 모르는 신생 브랜드인데 스킨토너 인기검색어 7위까지 올라옴.
- **달바 미스트세럼** — 보합세 그래프 = "이미 성숙". 비타(여름·미백) / 안티에이징(겨울·주름) 시즌 분리 운영.
- **달링태그 앰플** — 검색량 2천 → 2만 폭발 사례. 역산: 브랜드 검색 2만 = 월 광고비 약 4억(일 1300만), ROAS 200% 가정 시 월 매출 10억대.
- **주다코 앰플** — 달링태그 연관 폭증 키워드.
- **퍼포먼스 광고 100억 브랜드 패턴**: 한 SKU 폭발 = 3~5개월(짧으면 3, 길면 6) 지속. 3번 정도 터뜨리면 연 100억.

### "두 달 광고비만 쓰고 나간 광고주" 사례

- 기본기 없이 광고만 돌려서 성과 X → 강사가 교육 만든 직접적 계기.
