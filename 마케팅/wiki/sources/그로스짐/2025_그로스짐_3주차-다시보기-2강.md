# 3주차 다시보기 2강 — 누룩밤 분해 + AI 리뷰·이미지 생성 실습

## 출처

- 원본: `마케팅/raw/그로스짐/21. 3주차 다시보기 2강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 3주차 (리뷰 편 2강 — 실습)

## 한 줄 요지

**누룩밤** 기미 크림 케이스 스터디로 광고-리뷰 메시지 일치를 분해하고, AI(클로드/제미나이/힉스필드)로 **포토 리뷰 50개 + B&A 이미지 50개** 자동 생성 워크플로우 실습.

## 핵심 골자

### 누룩밤 케이스 — 광고와 리뷰 메시지 일치의 모범

- 광고 라이브러리: "**14일 만에 기미 없어진다**" + 어머니가 사용 + 사용 전/후 이미지 + 누룩·트라넥삼산·나이아신아미드 성분 + 겉기미·잠재기미 구조.
- 자사몰 베스트 리뷰: "**2주 만에 기미 옅어졌다**" + 몇 년 고민·피부과 레이저 몇백만 원 효과 잠깐.
- → **광고 14일 = 리뷰 2주**, 페르소나 경험 일치, 화자만 다름. 우연이 아니라 **설계**.

### 네이버 페이 구매평을 베스트 리뷰 위에 일부러 배치

- 네이버 페이 리뷰 = 작업으로 못 만듦 (실제 구매 + 배송 완료 후 작성, 공수 큼).
- 그래서 **신뢰도 가장 높은 리뷰**로 인식됨 → 일부러 위 배치.
- 누룩밤은 **히든 링크** 기능까지 활용 (특정 URL 통해서만 보이는 리뷰).

### 모바일 뷰 검증 — F12 → 기기 선택

- 99% 모바일 구매. 크롬 F12 → 디바이스 토글 → 아이폰 14 Pro Max로 보면서 검증.
- PC뷰 기준으로 리뷰 디자인 맞추는 건 의미 없음.

### AI 워크플로우 (실시간 시연 — 1시간 분량)

> 강사 메인 = 클로드 (이날 클로드 서버 다운 — "미국·이란 전쟁으로 AWS 공격받음"이라는 가설). 백업 = 제미나이.

**리뷰 50개 생성 7단계**:

1. 이커머스 AI 솔루션(크롬 확장)으로 경쟁사 자사몰 리뷰 110개 엑셀 다운로드.
2. 베스트 리뷰 2개 캡처 → "기미크림 리뷰 중 글자 수 길고 캡처 리뷰처럼 핵심 요소 잘 들어가 설득되는 리뷰 50개 추려줘" + 엑셀 첨부 + 캡처 2개.
3. 스프레드시트로 출력 → 다운로드.
4. **새 채팅창** 켜기 (AI context 누적 시 hallucination 폭증).
5. 우리 제품 정보(퍼포먼스 캔버스) + 추출 리뷰 → "각색 리뷰 20개를 스프레드시트로" + 사고 모드.
6. 결과 검토 → 제품명 hallucination 시 (예: "보나주르 멜라노 크림" 같은 가짜 이름) 다시 명시.
7. 클로드가 제미나이보다 분석·추출이 압도적으로 좋음 (이날은 클로드 다운으로 부득이).

**핵심 메소드 = 큐샷(Few-shot) 예시 강조**:

> "예시가 결국에는 얘가 판단해야 될 근거가 된다. 그냥 '재미있고 위트 있는 리뷰' 같은 광범위한 단어 쓰면 결과물이 안 좋다."

**B&A 이미지 50장 생성 (제미나이 나노 바나나)**:

1. 블로그 체험단 이미지(사용 가능한 인물 사진) 모으기. **30대 얼굴도 50대 타겟에 OK** (여성은 더 젊어지고 싶어함).
2. 새 채팅 → "20개 이미지의 포토리뷰 장면 추출 prompt 만들어줘" (스프레드시트 리뷰 순서대로).
3. 제미나이 이미지 만들기 → "예시 이미지 바탕으로 prompt 적용" + 사고 모드.
4. **단계별 수정 원칙** (한 번에 여러 개 X):
   - 큰 범위부터: 배경 → 옷 → 얼굴 형태 → 머리 → 소품
   - 매 수정마다 직전 결과를 **다시 첨부**해야 함 (안 그러면 첫 원본 기준으로 수정)
5. **불완전한 요소**를 prompt에 명시 (인간은 비대칭 — AI는 자꾸 대칭 만들려 함).
6. 카메라 prompt 핵심: 렌즈·기종·셔터 스피드·ISO·조명·시간대·장소.

**힉스필드(Higgsfield) AI** 추천:

- 제미나이 나노 바나나를 튜닝한 SaaS — 렌즈·카메라·아이폰 셀카 감도 등 미세조정 이미 입력됨.
- 비용 더 쓰지만 prompt 디테일 작성 귀찮은 사람한테 추천.
- 영어 UI지만 "유튜브에 사용법 너무 많으니 강의 사지 말 것."

### 비포 애프터 + 케이스 + 손 사진 비율 (50개 기준)

- B&A: 20%
- 손/부위 바르는 장면: 20%
- 애프터만 (피부 좋은 여성, 발음 좋은 여성): 10%
- 나머지: 케이스 + 블로그 체험단 변형

### 블로그 체험단 2차 활용 동의 X 우회법

- 체험단 원본 이미지 → 나노 바나나로 **특정 가능 요소 다 변경** (배경·식탁·손 모양 → 한 단계씩).
- 원본 질감·카메라 감도 유지되면서 다른 사람 됨.
- "초상권 동의 못 받았다 하더라도 구도는 그분 특허가 아님" — 강사 멘트.
- 단, 동의받은 이미지가 가장 안전.

### "리뷰는 한 번 깔고 끝" 아님

- 매주 신규 리뷰 등록되면 기존이 뒤로 밀림.
- 1점짜리 리뷰는 작업되어 있으면서 동시에 "제품은 좋은데 배송 늦었다" 식의 판매자 책임이 아닌 사유로 구성.
