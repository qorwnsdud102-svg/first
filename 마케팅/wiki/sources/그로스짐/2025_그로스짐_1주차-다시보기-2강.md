# 1주차 다시보기 2강 — 리뷰분석 툴·인구통계·페르소나·욕구 레이어

- 원본: `마케팅/raw/그로스짐/6.1주차 다시보기 2강.txt` (자동 트랜스크립트)
- 강의: [[그로스짐]] 2기 1주차 2번째 강

## 한 줄 요지

캔버스 1행을 채운 다음 단계. **리뷰 1만건 자동 추출 → 인구통계 → 페르소나 → 욕구 레이어**로 위쪽 칸을 채운다. 자체 개발한 크롬 확장툴로 자동화.

## 핵심 골자

### 리뷰 자동 다운로드 툴 (강사 측 개발)

- **E-commerce AI 솔루션** (크롬 확장 프로그램, 강사 측이 특정 개발사와 함께 만듦).
- **스마트스토어 + 쿠팡 리뷰 자동 다운로드 (최대 1만개)** + 상세페이지 이미지 일괄 다운로드.
- 네이버 키워드 도구 API 호출 → 검색창 옆에 검색량·연관키워드·월간 추이 그래프 자동 표시.
- 카테고리 1~5등 리뷰 모두 받아서 Claude에 입력 → "리뷰 글자수 100자 이상의 공통 단어 10개 추출 + 그래프"식 지시.

### Claude for Excel — 강사가 강조한 hallucination 회피 도구

- 100달러 플랜부터 사용 가능.
- 엑셀 10시트 1만 8천 건 + Vlookup 함수 자동 정리.
- 행·열 지정해서 "C열 100자 이상" 식으로 지시하면 직접 데이터 접근 → **hallucination 거의 0**.
- 일반 Claude는 데이터 추출 후 요약 (오차 있음) / 클로드 폴 엑셀은 데이터에 직접 접근 = RAG 역할.
- 강사 = **Claude 200달러 플랜**. 5~7개 창 동시 운용. "사람이 하지 말아야 할 일이 많아지고 있음."

### 인구통계학 = 키워드 도구의 연령·성별 그래프에서 도출

- 감으로 정하지 말고 **누적 데이터 신뢰도**. 예: "기미 크림" 압도적 40~50대 여성. 남자 21% → 버리고 뾰족하게.

### 페르소나 작성 절차 (강사식)

1. 1행 문제정의 → 2. 효과 키워드 확장 → 3. 의학 키워드 확장 → 4. 시술 키워드 확장 → 5. 경쟁사 키워드 확장 → 키워드 전체 리스트업.
2. 그 키워드를 **함축해 "이 인구통계 사람이 우리 제품을 쓸 수밖에 없었던 스토리"**로 묶음.
3. 광고 기획의 중심 캐릭터 = 이 사람.

### 표면적 vs 내면적 욕구 레이어 (강의에서 더 디테일하게)

> 사전 VOD 3강의 표면/원초 욕구 = 본강에서 **표면(후킹) / 내면(설득)** 짝으로 명시.

- 다이어트 건기식 = 식욕(내면) + 건강욕(레이어1) + 성욕(레이어2).
- **표면적 욕구 = 후킹용** (살 빠진 모습, 복근 보이는 잘생긴 남자, 몸매 좋은 여자가 "오빠 이거 한번 먹어봐").
- **내면적 욕구 = 설득용** ("너 떡볶이 치킨 술 다 먹으면서 뺄 수 있어?").
- 매슬로 욕구는 **존경·자아실현 위는 광고에서 안 먹힘** (복잡·어려움). 화장품·건기식·뷰티디바이스는 안전 욕구 이하만 활용.

### "주변에 숨 쉬듯이 페르소나 노출시키기" 메소드

- 강사 본인 사례: **대만 마케팅** 진행 중 → 별도 SNS 계정 파서 대만 콘텐츠만 학습시켜 매일 봄. 홍콩 동일.
- 50대 여성 건기식 = 50대 여성 SNS·커뮤니티에 그들이 보는 콘텐츠 학습시켜 노출.
- 예시 광고주: **주부 연구소** 만들어서 주부 인플루언서랑만 협업, 그 인플루언서 이름으로 제품도 만듦.

### 광고주 정의 vs 표면적 욕구 활용 사례 (남자 활력제)

- 본질 한마디 = "밤새도록 힘을 낼 수 있습니다." 식약처 우회 메시지.
- **광고소재**: 앞치마 입은 예쁜 여성이 주방에서 누군가 쳐다보는 사진 + 카피 "남편이 새벽 6시에 일어나서 설거지도 도와주네요. ○○ 마카라고 하니까."
- 욕구 분해: 표면(성욕·예쁨) + 내면(와이프 사랑받고 싶음) + 페르소나(50~60대 남편 + 그 부인) + 인구통계(50~60대).
