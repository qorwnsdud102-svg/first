# 2주차 다시보기 2강 — 캔버스 디테일 피드백 + 니즈×반복문제 매트릭스

- 강의: [[그로스짐]] 2기 2주차

## 한 줄 요지

수강생들의 1주차 캔버스 작성을 보고 받은 강사의 공통 피드백 + 캔버스의 컨셉/포지셔닝을 잡는 강사의 **니즈 점수 × 반복 문제 점수 매트릭스**.

## 핵심 골자

### 캔버스 작성 공통 실수 (광고주 피드백 패턴)

- **인구통계**: "50대 여성"만 적지 말 것. 직업군·생활패턴·주거환경까지. (캔버스는 **혼자 보는 게 아니라 협업 도구** — 디자이너·기획자가 같이 봄, 그래서 구체성 필수)
- **욕구·욕망**: "성욕·식욕"만 적지 말 것. 그 욕구가 **이 제품과 어떻게 연결되는지** 한 줄 메모 필수.
- **페르소나**: "이 사람이 이 제품 산다"만 적으면 X. 발톱 무좀 사례 — 무좀 발생 과정 → 자각(간지러움) → 동반 증상(냄새) → 사회적 부끄러움(친구가 발 냄새 누가 나냐 쿠사리)까지 **스토리화**.
- **문제·해결**: 표면 해결을 끌고 들어오지 말 것. "목마르니 야채 먹는다" 같은 우회 X — 직관적이어야 함.
- **근거 (4사회증거 중) 경쟁력**: "피부 무자극" 같은 보편 인증 X. 경쟁사가 못 따라오는 요소만.
- **컨셉 (최초·최고·유일)**: 단순 카테고리 세분화 X (예: "세로 모공 앰플"). 누구·왜·어떤 문제를 위한 X 카테고리까지 명사화.

### 강사의 컨셉 매트릭스 (새 메소드)

> **2축**: ① 많은 사람이 공감하는 고민의 점수 (니즈 폭) × ② 반복적으로 해결 못 하는 점수 (해결 불가능성)
> 
> **퍼포먼스 광고 최적 카테고리 = 양 축 모두 고점**.
> 
> 둘 다 높음 → 영원한 숙제, 사람들이 이거저거 다 시도 (반복 구매)
> 니즈만 높고 해결 가능 → **하지 마라** (가장 망하는 카테고리)
> 니즈 낮고 해결 불가 → **고가 정책 가능** (예: 발기부전·치매·유방암 예방 류, 노인 액취증)

매트릭스 적용 사례:
- **다이어트** = 양축 모두 만점 (퍼포먼스 1티어)
- **탈모** = 니즈 5명 중 1명 + 해결 불가 → 둘 다 적당히 높음
- **발톱 앰플** = 출시 시점엔 니즈 3-4점, 반복 점수 10점 초과. 광고로 시장 자체를 키워서 양축 모두 끌어올림 (= 광고 붐업으로 시장 확장)
- **유방암 예방 제품** = 니즈 낮음 but 시장 변동 가능 (시그널 = 검색 키워드·커뮤니티·방송 노출)

### 겨드랑이(액취증) 바디워시 캔버스 도출 사례 (강사 자작 시뮬)

- 카테고리: 바디워시 (몸 씻고 냄새 제거)
- 니즈 반영: "몸 냄새" 마인드맵 → 입냄새 X (바디워시는 입에 안 닿음) → 액취증 키워드 발견
- 세분화 (누가): 노총각 → "노" = 늙음 → **노인 전용 겨드랑이 바디워시**
- 충성 리뷰 워딩 추출: 바디워시 후 "은은한 살냄새"
- 광고 카피: "노인들의 심각한 지워지지 않는 악취를 지우고 은은한 살냄새로 만들어드립니다"
- 마케팅 핵심 요약 3줄: 미션·근거·비전 (강사가 입사자에게 보여줘도 다 이해되게 하는 한 페이지 요약)

### 다이어트 카테고리 = 같은 제품 다른 포지셔닝

- 다이어트 건기식은 가르시니아류. 잘 안 팔리니 **기타가공식 + 메인 성분만 갈아끼움**: 레몬즙·올리브유·혈당 다이어트 등 (모두 같은 맥락, 다른 포지셔닝). 
- 부작용: 과대광고 리스크. 그러나 강사 고객사 중 이 방식만으로 **연 매출 3천억** 한 분 있음.

### 사장 의사결정 = 100억 기로

- 100억까지: "수단 방법 가리지 말고 팔아라" 모드
- 100억 이후: 두 갈래 — ① 봉이 김선달식 복사붙여넣기로 3천억·5천억 (부작용 따름) ② 제품 개선·인플루언서·연예인 권위 플레이로 전환
- 화장품은 치료제 아니므로 권위 포장(인플루언서·연예인·디렉터) 어쩔 수 없음. **아누아** 사례.

### 시장조사 도구

- **썸트렌드** (분석툴): 9.9만원/월. 키워드 언급량·긍부정·연관 키워드. 액취증 검색 → 스트레스·땀·치료·다한증·수술·시술 키워드 추출 → 광고 카피로 활용.
- "고객 표현 워딩 = 패턴" — 사람 일상이 패턴이듯 단어/문장도 패턴화. 그래서 추출이 가능.
