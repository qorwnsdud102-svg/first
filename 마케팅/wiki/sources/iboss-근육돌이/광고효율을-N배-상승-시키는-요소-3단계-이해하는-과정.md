# 광고효율을 N배 상승 시키는 요소 3단계 이해하는 과정

- 효율 좌우 요소 10순위: 제품>고객>시장>USP>포지셔닝>랜딩>소재>매체>머신러닝
- 1단계(제품·고객·시장·USP) 비중이 90%, 매체는 2~3단계
- 타겟 구체화: '미백앰플' 대신 '40대 10년 기미 잡티 여성'식 뾰족화
- 비포에프터 명확한 시각화 가능 제품이 퍼포먼스에 유리
