# MCP

## 한 줄 정의

AI에 외부 도구·데이터를 USB 허브처럼 연결해주는 중간 다리 프로토콜. [[근육돌이]] 표현으로 "USB 포트를 연결해주는 역할".

## 핵심 주장 / 속성

- **Claude가 중심에 있다** — 근육돌이가 명시한 클라이언트는 클로드. (raw §5 "클로드가 가장 중심에 있습니다")
- **외부 데이터·툴 연동의 표준화** — 검색·DB·외부 API 등을 AI에 붙일 수 있다. 예시로 네이버 데이터랩 연결 → 시장조사 자동화.
- **마케터 관점 정확도** — 복잡하지 않은 일에는 정확도 매우 높음. 일부 할루시네이션은 있음.
- **설치·설정 난이도 낮음** — 근육돌이 표현 "유튜브 보시면 설정하는 방법이 너무 쉽게 잘 나와있습니다".

## 다른 엔티티와의 관계

- (후속 신설 예정) [[Claude]] — MCP 클라이언트로서.
