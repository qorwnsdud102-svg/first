3. CS ( 별도의 내부 팀이 구성 되어 고객 목소리에 빠른 피드백 )

4. 물류 ( 직접 물류를 운영해 타사는 3~4일 배송 해당 본사는 1~2일 이내 배송 )

5. 광고 개선 ( 데이터를 통한 광고 소재 상세페이지 개선 )

6. 실험 ( 끝 없는 광고 실험 진행 데이터 확보 히스토리 3년치 이미지 및 데이터 )
